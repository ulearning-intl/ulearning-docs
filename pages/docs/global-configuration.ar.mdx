# التكوين العالمي رقم 

 يمكن أن يوفر السياق `SWRConfig` تكوينات عالمية ([الخيارات](/docs/api)) لجميع خطافات SWR.

```jsx
<SWRConfig value={options}>
  <Component/>
</SWRConfig>
```

في هذا المثال، ستستخدم جميع خطافات SWR نفس الجلال المقدم لتحميل بيانات JSON، وتحديثها كل 3 ثوان افتراضيا:

```jsx
import useSWR, { SWRConfig } from 'swr'

function Dashboard () {
  const { data: events } = useSWR('/api/events')
  const { data: projects } = useSWR('/api/projects')
  const { data: user } = useSWR('/api/user', { refreshInterval: 0 }) // override

  // ...
}

function App () {
  return (
    <SWRConfig 
      value={{
        refreshInterval: 3000,
        fetcher: (resource, init) => fetch(resource, init).then(res => res.json())
      }}
    >
      <Dashboard />
    </SWRConfig>
  )
}
```

## تكوينات التعشيش [#تكوينات التعشيش] رقم 
 
يدمج `SWRConfig` التكوين من السياق الأصلي. يمكن أن يتلقى إما كائنا أو تكوينا وظيفيا. يتلقى الشخص الوظيفي التكوين الأصل كحجة ويعيد تكوينا جديدا يمكنك تخصيصه بنفسك. رقم 
 
### مثال على تكوين الكائن [#object-configuration-example]

```jsx
import { SWRConfig, useSWRConfig } from 'swr'

function App() {
  return (
    <SWRConfig
      value={{
        dedupingInterval: 100,
        refreshInterval: 100,
        fallback: { a: 1, b: 1 },
      }}
    >
      <SWRConfig
        value={{
          dedupingInterval: 200, // will override the parent value since the value is primitive
          fallback: { a: 2, c: 2 }, // will merge with the parent value since the value is a mergeable object
        }}
      >
        <Page />
      </SWRConfig>
    </SWRConfig>
  )
}

function Page() {
  const config = useSWRConfig()
  // {
  //   dedupingInterval: 200,
  //   refreshInterval: 100,
  //   fallback: { a: 2,  b: 1, c: 2 },
  // }
}
```

### مثال التكوين الوظيفي [#functional-configuration-example]

```jsx
import { SWRConfig, useSWRConfig } from 'swr'

function App() {
  return (
    <SWRConfig
      value={{
        dedupingInterval: 100,
        refreshInterval: 100,
        fallback: { a: 1, b: 1 },
      }}
    >
      <SWRConfig
        value={parent => ({
          dedupingInterval: parent.dedupingInterval * 5,
          fallback: { a: 2, c: 2 },
        })}
      >
        <Page />
      </SWRConfig>
    </SWRConfig>
  )
}

function Page() {
  const config = useSWRConfig()
  // {
  //   dedupingInterval: 500,
  //   fallback: { a: 2, c: 2 },
  // }
}
```

## واجهات برمجة التطبيقات الإضافية [#extra-apis] 

### مزود ذاكرة التخزين المؤقت [#cache-provider] 

إلى جانب جميع [options](/docs/api) المدرجة، يقبل `SWRConfig` أيضا وظيفة `provider` الاختيارية. يرجى الرجوع إلى قسم [Cache] (/docs/advanced/cache) لمزيد من التفاصيل.

```jsx
<SWRConfig value={{ provider: () => new Map() }}>
  <Dashboard />
</SWRConfig>
```

### الوصول إلى التكوينات العالمية [#الوصول إلى التكوينات العالمية] رقم 
 
يمكنك استخدام خطاف `useSWRConfig` للحصول على التكوينات العالمية، بالإضافة إلى [`mutate`](/docs/mutation) و[`cache`](/docs/advanced/cache):

```jsx
import { useSWRConfig } from 'swr'

function Component () {
  const { refreshInterval, mutate, cache, ...restConfig } = useSWRConfig()

  // ...
}
```

سيتم تمديد التكوينات المتداخلة. إذا لم يتم استخدام `<SWRConfig>`، فسيؤدي ذلك إلى إرجاع الإعدادات الافتراضية.
