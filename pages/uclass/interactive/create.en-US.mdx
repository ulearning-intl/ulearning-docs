# Create a Class

Click Create Session > Name the class > Select the class that needs to attend > Complete creation.

![pic10](/Uclass_pics/en/pic10.png)

Select "Start Now" to begin the class; selecting "Start Later" will enter the class preparation page, allowing you to create classroom activities in advance.
