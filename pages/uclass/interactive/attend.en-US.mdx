# Start Class Session

Scenario 1: Create a New Class and Start Teaching

Click the "Create Class" button to create a class, select the class, and then click "Start Now" to enter the class session page.

![pic13](/Uclass_pics/en/pic13.png)

![pic14](/Uclass_pics/en/pic14.png)

Scenario 2: Starting a Pre-prepared Class

Select a class with the status **"Not Started"** from the class list. Click to enter the class page, then tap the **"Start Class"** button at the bottom right to begin teaching.

![pic15](/Uclass_pics/en/pic15.png)
