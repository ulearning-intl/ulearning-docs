# Session Report

The system automatically generates a session report. Click "Session Report" in the class list to view the detailed report for that session.

![pic47](/Uclass_pics/en/pic47.png)

The session report includes:  
(1) Class Information: Name of the session, class/group details, etc.  
(2) Attendance Status: Check-in records for the session.  
(3) Activities and Interaction Details: Information on activities published and interactions during the class.  
(4) Student Points Earned: Points awarded to students during the class.  
(5) Teacher Remarks: The teacher can add notes to document the session.
