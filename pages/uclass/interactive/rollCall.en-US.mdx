# 课堂点名

The large-screen roll call includes two methods:

QR Code Sign-in: Students use their mobile phones to scan the QR code displayed on the large screen to complete sign-in.  

Digital Code Sign-in: Students enter the number code shown on the screen into their mobile phones to complete sign-in.

To prevent students from checking in on behalf of others, the number code and QR code on the large screen automatically refresh at regular intervals,  The teacher can configure the time interval for the refresh.
