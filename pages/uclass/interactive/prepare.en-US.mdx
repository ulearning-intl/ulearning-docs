# Prepare Teaching Content

After creating a class, select “Start Later” or choose a class that is "Not Started" to enter the class preparation page. By clicking the buttons at the bottom of the interface, the teacher can create activities such as pick, polls, grading, and quizzes. Activities created before the class starts will be saved as drafts. The teacher can view all activities created for the current class on the page.  Once the class begins, the teacher can manually start and end the activities.

![pic12](/Uclass_pics/en/pic12.png)

Roll call and pick are activities started during class and do not support creation in advance.
