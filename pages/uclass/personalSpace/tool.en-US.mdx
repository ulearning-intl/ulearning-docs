# Common Teaching Tools

## 1.Whiteboard

Teachers can use the whiteboard within the large-screen teaching software for writing. Teachers can modify pen color and stroke. Supports text input, insertion of basic shapes such as lines, rectangles, circles, etc., supports image insertion, and supports subject tools like rulers, triangles, and compasses.

![pic4](/Uclass_pics/en/pic4.png)

## 2.Open File

Teachers can open local files or resources from their personal cloud drive for classroom demonstration and teaching. Clicking "Send to Students" can distribute the courseware to the classroom for students to view.

![pic5](/Uclass_pics/en/pic5.png)

## 3.Screen Mirroring

Supports teachers mirroring the screen of mobile devices such as phones or tablets to the large screen. Supports both iOS and Android systems.
