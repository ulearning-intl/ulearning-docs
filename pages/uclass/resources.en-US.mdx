# Resources Center

The Resource Center contains the resource libraries activated by the school. Teachers can access these libraries and use the resources within them for teaching. The school's resource libraries are configured by the school administrator in the management backend. If additional resource libraries are needed, please request the school administrator to configure them.

![pic49](/Uclass_pics/en/pic49.png)

![pic50](/Uclass_pics/en/pic50.png)

Teachers can click the "Favorites" icon to bookmark frequently used resource libraries for easier access in the future.
