## I. Chemin d'accès

【Membres】-【Gestion des classes/étudiants/équipe pédagogique】

## II. Étapes d'opération

Ici, vous pouvez gérer les classes, les étudiants et les équipes pédagogiques associées à ce cours (l'administrateur du cours peut voir toutes les classes et étudiants).

### 1. Équipe pédagogique

Via 【Ajouter un enseignant】, vous pouvez ajouter des enseignants qui participent à l'enseignement de ce cours ensemble. Si c'est un mode d'enseignement individuel, vous n'avez pas besoin d'ajouter une équipe pédagogique.

Note : Les enseignants associés sont des personnes déjà créées dans l'administration centrale. Si vous ne trouvez pas l'enseignant que vous souhaitez ajouter, veuillez contacter l'administrateur de l'école.

Permissions des enseignants : Les cours peuvent définir trois types de permissions : administrateur, enseignant et assistant pédagogique ; les différences spécifiques des permissions peuvent être comprises via 【Voir les permissions pédagogiques】.

Gestion des enseignants et classes : Vous pouvez utiliser 【Attribuer une classe】 pour attribuer des enseignants aux classes qu'ils enseignent, et vous pouvez aussi utiliser 【Supprimer du cours】 pour dissocier les enseignants des classes.

### 2. Gestion des classes

#### (1) Créer une classe

**Méthode 1** : Vous pouvez créer via 【Gestion des classes】-【Créer une classe】. Lors de la création de plusieurs classes simultanément, il suffit de les saisir sur des lignes différentes. Vous pouvez également définir si une approbation est requise lorsque les étudiants rejoignent/quittent la classe, et si les étudiants d'autres écoles peuvent rejoindre.

**Méthode 2** : Importer depuis la base de données des étudiants, importer des classes depuis d'autres cours

#### (2) Approbation de classe

Si une approbation est requise pour rejoindre la classe, vous pouvez approuver dans 【Demandes d'adhésion/retrait de classe】.

#### (3) Plus d'opérations

- **Liste des étudiants** : Entrée vers 【Gestion des étudiants】, cliquez pour accéder à la liste correspondante.
- **Ajouter un étudiant** : Associer les étudiants de l'école à une classe d'un cours.
- **Attribuer un enseignant** : Associer les enseignants de l'école à une classe d'un cours.
- **Modifier la classe** : Modifier les informations de la classe.
- **Supprimer la classe** : Tous les étudiants de la classe seront supprimés du cours, et tous les enregistrements d'activités des étudiants dans la classe seront effacés. Veuillez procéder avec précaution.

### 3. Gestion des étudiants

Les étudiants rejoignent la classe indépendamment : 【Membres】-【Gestion des classes】-【Code de classe】, cliquez pour voir le code QR.

La liste affiche tous les étudiants du cours, et vous pouvez effectuer les opérations de gestion suivantes :

- **Changer de classe** : Prend en charge le changement d'étudiants vers d'autres classes créées de ce cours ;
- **Supprimer du cours** : Après avoir supprimé un étudiant du cours, les devoirs, discussions, examens et autres enregistrements de l'étudiant dans le cours seront supprimés ;
- **Réinitialiser le mot de passe** : Les enseignants peuvent réinitialiser les mots de passe des étudiants. Le mot de passe réinitialisé est un mot de passe unifié. Vous pouvez rappeler aux étudiants de changer le mot de passe après connexion ;
- **Profil de l'étudiant** : Voir les informations de synthèse du cours des étudiants et les enregistrements d'apprentissage, de devoirs, d'examens, etc. ;
- **Exporter la liste** : Prend en charge l'exportation des données de la liste des étudiants.

### 4. Autres gestionnaires de personnel

#### (1) Groupes de classe

Cliquez sur 【Plan de groupes de classe】 pour diviser les membres de la classe en plusieurs groupes, qui peuvent être utilisés pour 【Devoirs】-【Devoirs de groupe】.

#### (2) Authentification faciale

L'authentification faciale est utilisée pour les examens des étudiants. Si la vérification par reconnaissance faciale est activée, vous devez télécharger des photos d'authentification des candidats à l'avance. Les photos peuvent être importées en masse par les enseignants ou téléchargées par les étudiants eux-mêmes. Les photos personnelles téléchargées par les étudiants doivent être examinées par les enseignants, tandis que les photos d'étudiants téléchargées par les enseignants n'ont pas besoin d'examen.
