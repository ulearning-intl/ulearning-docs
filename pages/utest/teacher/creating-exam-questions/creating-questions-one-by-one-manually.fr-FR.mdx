# Créer des questions une par une manuellement 
 
 Si vous souhaitez créer quelques questions rapidement, vous pouvez le faire en utilisant cette méthode. 
 
 ### Étape 1 
 Sur l'écran d'examen du cours, cliquez sur le bouton Banque de questions [1]. 
 ![]( /img/utest/teacher/question01.png) 
 
 ### Étape 2 
 Sur l'écran Question Bank, cliquez sur le bouton Ajouter une question [1]. 
 ![]( /img/utest/teacher/question02.png) 
 
 ### Étape 3 

 Il ouvre l'écran de l'éditeur de questions dans une nouvelle fenêtre/onglet. 
 
 ![]( /img/utest/teacher/question03.png) 
 
 - Sélectionnez le type de question et le numéro que vous souhaitez ajouter et cliquez sur le bouton Ok [1] pour ajouter la ou les questions. 
 - **Tente de la question :** Cliquez pour entrer l'instruction question/question [2]. 
     - *Notes : Il s'agit d'un champ de contenu riche. Vous pouvez formater le contenu et ajouter de l'image, de l'audio, de la vidéo et même une formule selon vos besoins.* 
 - **Modéments de réponse :** Cliquez pour entrer le choix [3]. 
     - *Notes : Il s'agit d'un champ de contenu riche. Vous pouvez formater le contenu et ajouter de l'image, de l'audio, de la vidéo et même une formule selon vos besoins.* 
 - Pour ajouter d'autres choix de réponses, cliquez sur le bouton Ajouter un élément [4]. 
 - **Réponse :** Définissez la bonne réponse à la question [5]. 
 - **Extorsion de la réponse (facultative) :** Cliquez pour entrer l'explication de la réponse. C'est un moyen de renforcement de l'apprentissage. Les étudiants peuvent consulter l'explication de la réponse après soumission [6]. 
     - *Notes : Il s'agit d'un champ de contenu riche. Vous pouvez formater le contenu et ajouter de l'image, de l'audio, de la vidéo et même une formule selon vos besoins.* 
 - **Points :** Entrez les points possibles pour la question [7]. 
 - **Difficulté :** Sélectionnez le niveau de difficulté pour la question [8] 
     - Options : 
         - Très facile 
         - Facile 
         - Moyenne 
         - Difficile 
         - Très difficile 
 - **Partager :** Sélectionnez la façon dont vous souhaitez partager la question [9]. 
     - Options : 
         - Privé 
         - Enseignants sélectionnés 
         - Enseignants 
         - Tous les enseignants 
 - **Tag (facultatif) :** Cliquez pour entrer une balise de moins de 20 caractères pour la question [10]. Vous pouvez entrer jusqu'à 3 balises pour chaque question. 
 - **Module connexe :** Vous pouvez relier la question à un ou plusieurs modules de didacticiels. Il peut être utilisé par un examen pour dessiner automatiquement les questions en fonction de la relation question-module/courseware [11]. 
 - Vous pouvez répéter les étapes ci-dessus pour créer autant de questions que vous le souhaitez. 
 - Une fois que vous avez terminé, cliquez sur le bouton Enregistrer [12] pour enregistrer les questions dans la banque de questions. 
 
 **Notes :** 
 
 Vous pouvez créer manuellement 13 types de questions d'examen 
 - Choix multiple 
 - Réponse multiple 
 - Vrai/Faux 
 - Remplissez le champ vide 
 - Réponse courte 
 - Formule calculée 
 - Séquence 
 - Banque de mots 
 - Menu Déroulant Multiple 
 - Test Cloze 
 - Langue - Lire à haute voix 
 - Langue - Orale 
 - Combinaison
