# Importation de questions par lots - Utilisation d'un modèle Excel 
 
 Si vous avez un grand nombre de questions à créer, vous pouvez utiliser la fonction d'importation par lots dans ULearning. Suivez les étapes ci-dessous pour importer des questions à l'aide d'un modèle Excel : 
 
 ### Étape 1 : Accédez à la fonction d'importation par lots 
 - Sur l'écran Question Bank, localisez et cliquez sur le bouton Importation par lots [1]. 
 - Cette action ouvrira la fenêtre contextuelle "Importer des questions par lots". 
 
 ![]( /img/utest/teacher/question05.png) 
 
 ### Étape 2 : Télécharger le modèle 
 
 - Dans la fenêtre contextuelle "Importer des questions par lots", cliquez sur le bouton Télécharger le modèle [2]. 
 - Le modèle, nommé "Importer des questions par lots", sera téléchargé sur votre ordinateur local. 
 - Ouvrez le modèle et suivez les instructions fournies pour entrer vos questions. 
 - Une fois que vous avez entré toutes les questions, enregistrez le fichier. 
 
 ### Étape 3 : Télécharger le fichier terminé 

 - Retournez à la fenêtre contextuelle "Importer des questions par lots". 
 - Cliquez sur le bouton Télécharger le fichier [3]. 
 - Sélectionnez le fichier que vous venez de remplir et cliquez sur Ouvrir. 
 - Les questions du fichier seront téléchargées et ajoutées à la banque de questions. 

 ### Instructions pour l'utilisation du modèle : 

 ![]( /img/utest/teacher/question06.png) 
 
 - Le modèle vous permet d'ajouter jusqu'à 200 questions à la fois. 
 - Tous les champs marqués d'un astérisque (*) sont obligatoires. Certains champs sans astérisque peuvent encore être requis pour certains types de questions. Reportez-vous aux instructions ci-dessous pour identifier ces champs. 
 - Le modèle prend en charge l'importation de sept types de questions : choix multiple, réponse multiple, vrai/faux, remplir le blanc, réponse courte, séquence et combinaison. 
 
 - Les questions combinées sont des questions avec plusieurs sous-questions. Les sous-questions doivent être l'un des six types de questions mentionnés ci-dessus (choix multiple, réponse multiple, vrai/faux, remplir le blanc, réponse courte et séquence). 
 - La capture d'écran du modèle ci-dessus comprend un exemple de question combinée et ses sous-questions surlignées en bleu. 
 - Pour les questions combinées, vous n'avez pas besoin de remplir les champs Difficulté, Partage et Module connexe pour les sous-questions. 
 - Le champ Numéro de série est utilisé pour différencier les questions combinées des autres types de questions. La question de combinaison et ses sous-questions doivent avoir le même numéro de série. 
 - Le champ Points est obligatoire. S'il est laissé vide ou avec une valeur non valide, le système attribuera 2 points par défaut. Pour les questions de combinaison, entrez les points pour chaque sous-question, et le système calculera le total de la question de combinaison. 
 - Entrez une valeur de 1 à 5 dans le champ Difficulté : 1 = Très facile, 2 = Facile, 3 = Moyenne, 4 = Difficile, 5 = Très difficile. 
 - Le champ Partager peut avoir des valeurs : Privé, Enseignants sélectionnés, Enseignants scolaires ou Tous les enseignants. 
 - Utilisez le format "Nom du cours>Nom du module" pour le champ Module connexe (facultatif). Par exemple : "Insurance 101>Introduction" signifie que la question est liée au module "Introduction" du didacticiel "Insurance 101". 
 - Le champ Tag (facultatif) est utilisé pour la recherche et la classification. Entrez jusqu'à 3 balises, chaque balise doit comporter moins de 11 caractères, séparés par # sans espaces. Par exemple : "Insurance#Coverage#Discount". 
 - Le champ Réponse : 
     - Choix multiple : une seule lettre de A à O (en majuscule). 
     - Réponse multiple : lettres de A à O (en majuscules) séparées par un point-virgule (;) sans espaces. 
     - Vrai/Faux : Vrai ou Faux. 
     - Remplissez le blanc : la réponse pour chaque blanc séparé par un point-virgule (;) sans espaces. 
     - Séquence : lettres de A à O (en majuscules) dans l'ordre et séparées par un point-virgule (;) sans espaces. 
 - Le champ Question Stem : Utilisez cinq traits de soulignement consécutifs (_____) pour représenter un blanc. Par exemple : "San Francisco est une ville de l'État _____. La capitale de l'État est _____." 
 - Les champs de choix A-O : prend en charge jusqu'à 15 champs de choix et sont obligatoires pour les questions ou sous-questions à choix multiple, à réponse multiple et de séquence. 
 - Le champ Instructions de la question n'est requis que pour les questions combinées, et vous pouvez le laisser vide pour d'autres types de questions.
