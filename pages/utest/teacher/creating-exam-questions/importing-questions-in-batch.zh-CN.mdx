# 批量导入问题-使用Excel模板 
 
 如果您有大量问题要创建，您可以使用ULearning中的批量导入功能。 按照以下步骤使用Excel模板导入问题： 
 
 ### 第 1 步：访问批量导入功能 
 
 - 在问题库屏幕上，找到并单击批量导入按钮[1]。 
 - 此操作将打开“批量导入问题”弹出窗口。 
 
 ![](/img/utest/teacher/question05.png)

 ### 第 2 步：下载模板 
 
 - 在“批量导入问题”弹出窗口中，单击下载模板按钮[2]。 
 - 名为“批量导入问题”的模板将下载到您的本地计算机。 
 - 打开模板并按照提供的说明输入您的问题。 
 - 输入所有问题后，保存文件。 
 
 ### 第 3 步：上传已完成的文件 
 
 - 返回“批量导入问题”弹出窗口。 
 - 单击上传文件按钮[3]。 
 - 选择您刚刚完成的文件，然后单击打开。 
 - 文件中的问题将被上传并添加到问题库。 
 
 ### 使用模板的说明： 
 
 ![](/img/utest/teacher/question06.png)
 
 - 该模板允许您一次添加多达200个问题。 
 - 所有标有星号（*）的字段都是必需的。 对于某些问题类型，某些没有星号的字段可能仍然需要。 请参考以下说明来识别这些栏位。 
 - 该模板支持导入七种问题类型：多项选择、多响应、真/假、填空、简短答案、序列和组合。 
 
 - 组合问题是指包含多个子问题的问题。 子问题必须是上述六个问题类型之一（多项选择、多个答案、正确/错误、填写空白、简短答案和顺序）。 
 - 上面的模板屏幕截图包括一个示例组合问题及其以蓝色突出显示的子问题。 
 - 对于组合问题，您无需填写子问题的难度、共享和相关模块字段。 
 - “序列号”栏位用于区分组合问题和其他类型的问题。 组合问题及其子问题应具有相同的序列号。 
 - 必须填写“积分”栏位。 如果留空或值无效，系统将默认分配2分。 对于组合问题，输入每个子问题的积分，系统将计算组合问题的总数。 
 - 在难度字段中输入1到5的值：1 = 非常容易，2 = 简单，3 = 平均，4 = 困难，5 = 非常困难。 
 - Share字段可以有值：私人教师、选定教师、学校教师或所有教师。 
 - 对相关模块字段使用“库尔软件名称>模块名称”格式（可选）。 例如：“保险101>简介”是指问题与“保险101”课件的“简介”模块有关。 
 - 标签字段（可选）用于搜索和分类。 输入最多3个标签，每个标签应小于11个字符，用#分隔，没有空格。 例如：“保险#Coverage#Discount”。 
 - 答案字段： 
     - 多项选择：从A到O的单个字母（大写）。 
     - 多重响应：从A到O（大写）的字母由分号（;）分隔，没有空格。 
     - 判断正误：判断正误。 
     - 填空：每个空白的答案由分号（;）分隔，没有空格。 
     - 排序：按顺序从A到O（大写）的字母，并用分号（;）分隔，没有空格。 
 - 问题干字段：使用五个连续的下划线（_____）表示空白。 例如：“旧金山是 _____ 州的一个城市。 该州的首都是_____。” 
 - 选择A-O字段：支持最多15个选择字段，并且需要多个选择、多个响应和序列问题或子问题。 
 - “问题说明”字段仅适用于组合问题，对于其他问题类型，您可以将其留空。
