 # Gestion des questions 
 
 Pour gérer efficacement vos questions d'examen dans la banque de questions, vous pouvez utiliser des dossiers. Voici comment vous pouvez créer et gérer des dossiers : 

 ### Étape 1 : Créer un dossier 
 
 - Sur l'écran Banque de questions, localisez et cliquez sur le bouton Ajouter un dossier [1]. 
 - Suivez les instructions fournies pour créer un dossier. 
 - Par exemple, vous pouvez créer un dossier nommé "ULearning Basics" [2] et placer toutes les questions d'examen pour ce cours dans ce dossier. 
 - Vous pouvez même créer un dossier à l'intérieur d'un autre dossier pour établir une structure de dossiers à plusieurs niveaux pour une meilleure organisation et gestion de vos questions d'examen. 
 
 ### Étape 2 : Effectuer des actions sur les dossiers 
 
 Une fois que vous avez créé un dossier, vous pouvez effectuer diverses actions dessus [3] : 
 
 - **Renommer :** Si vous souhaitez changer le nom du dossier, sélectionnez le dossier et choisissez l'option de renommer. Suivez les instructions pour fournir un nouveau nom pour le dossier. 
 - **Déplacer :** Si vous souhaitez déplacer le dossier vers un autre emplacement dans la banque de questions, sélectionnez le dossier et choisissez l'option de déplacement. Suivez les instructions pour sélectionner la nouvelle destination du dossier. 
 - **Supprimer :** Si vous n'avez plus besoin d'un dossier, vous pouvez le supprimer. Sélectionnez le dossier et choisissez l'option de suppression. Confirmez la suppression lorsque vous y êtes invité. 
 - **Partager :** Vous pouvez partager des dossiers avec d'autres utilisateurs si nécessaire. Sélectionnez le dossier et choisissez l'option de partage. Suivez les instructions pour spécifier les utilisateurs ou les groupes avec lesquels vous souhaitez partager le dossier.

 En utilisant des dossiers, vous pouvez organiser et gérer efficacement vos questions d'examen dans la banque de questions. Cela permet de rationaliser le processus de localisation et d'accès à des questions spécifiques dans votre cours.
