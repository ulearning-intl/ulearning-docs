 # 在批处理中创建问题（单击以插入/复制并粘贴） 
 
 如果您需要为多项选择、多答案、正确/错误、填空和简短回答问题类型创建大量问题，您可以使用ULearning中的批量创建功能。 按照以下步骤批量创建问题： 
 
 ### 第 1 步：访问问题编辑器 
 
 - 在问题编辑器屏幕上，找到并单击批处理创建按钮[1]。 
 
 ![](/img/utest/teacher/question07.png)
 
 - 通过单击相应的选项来插入它，选择要创建的问题类型[1]。 
 - 问题模板将插入左侧窗格[2]，基于模板的实际考试问题将显示在右侧窗格[3]中。 
 - 将问题模板（以黄色高亮显示）中的虚拟占位符编辑或替换为您的实际内容[2]。 右侧面板[3]中的实际考试问题将立即更新。 
 - 您可以插入任意数量的问题，并批量设置问题点[4]。 
 - 完成后，单击保存按钮将问题添加到问题库中。 
 
 ![](/img/utest/teacher/question08.png)
 
 ### 第2步：复制/粘贴方法 
 
 如果您想创建大量问题（例如50个），您可以使用复制/粘贴方法。 
 - 选择您要创建的问题类型，然后插入每种类型 [1]。 
 - 复制您插入的所有问题模板[2]，并将其粘贴到Word文档或记事本中。 
 - 编辑Word文档或记事本中的问题。 编辑完成后，复制问题并将其粘贴回问题库[2]。 
 - 单击保存按钮，所有问题都将添加到问题库中。 

 ![](/img/utest/teacher/question09.png)

 **提示：** 
 - 您可以使用问题编辑器中提供的问题模板创建自己的模板，并将其保存为本地计算机上的Word文档。 每当您想将考试问题添加到问题库时，您可以使用自己的模板快速创建这些问题，然后将其复制并粘贴到问题编辑器中。 
 - 此方法仅支持五类问题。 为方便起见，以下列出了这五种类型的模板。 您可以将它们复制并粘贴到Word文档或记事本文档中，以创建自己的模板。 
 
 ### 模板示例： 

```
1. (2) Multiple Choice question placeholder
   *
   (*) Choice 1 placeholder
   Choice 2 placeholder
   Choice 3 placeholder
   Choice 4 placeholder
   ## Answer Explanation placeholder (Optional)

2. (1) Multiple Response question placeholder
   *
   (*) Choice 1 placeholder
   (*) Choice 2 placeholder
   (*) Choice 3 placeholder
   Choice 4 placeholder
   ## Feedback placeholder (Optional)

3. (1) True/False question placeholder
   (*) True
   ## Feedback placeholder (Optional)

4. (1) Fill in the Blank question placeholder
   ( ) 、 ( ) 、 ( ) 。
   (*) = Answer 1 placeholder || Answer 2 placeholder || Answer 3 placeholder
   ## Feedback placeholder (Optional)

5. (1) Short Answer question placeholder
   (*) Answer placeholder
   ## Feedback placeholder (Optional)
```

 **注：** 

 - 问题词干前面括号内的数字表示分配给问题的点。 例如，（2）意味着问题有2分，您可以根据需要编辑数字。 
 - 问题词干后面的独立*标记表示它是多项选择或多响应问题。 
 - 答案选择前面的括号内*表示该选择是正确的答案。 例如，（*）选择1占位符。 您可以移动或删除它来重新选择正确的答案。 
 - ##后面的占位符用于答案解释，作为学习强化。 学生可以在提交后查看答案解释，但它是可选的。 您可以在##之后删除占位符，也可以将其替换为您的实际内容。 
 - 在填写空白问题词干中，一组括号（）表示空白。 (*) = 后的内容表示问题的答案，如果问题有多个空白，答案将以 || 分隔。 
 - 简短答案问题占位符允许单个答案占位符。 
 - ##反馈占位符是可选的，可用于就问题提供反馈。
