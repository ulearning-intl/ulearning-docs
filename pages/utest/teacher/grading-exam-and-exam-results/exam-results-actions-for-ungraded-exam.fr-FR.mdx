
# Actions pour l'examen non noté 

 Pour un examen non noté dans ULearning, en plus de l'action "Examen de note", vous pouvez effectuer les actions suivantes à partir du menu déroulant Actions : 

 ### Étape 1 : Accès aux actions pour l'examen non noté 
 
 ![]( /img/utest/teacher/result05.png) 
 - Sur l'écran Résultats de l'examen, localisez l'examen non noté [1]. 
 - Cliquez sur le menu déroulant des actions pour cet examen. 
 
 - Afficher le journal [2] : exactement le même journal que vous pouvez consulter à partir de la station de surveillance des examens. 
 ![]( /img/utest/teacher/result06.png) 
 
 - Vous pouvez cliquer sur le bouton Définir la reprise [3] pour donner à l'étudiant une chance de repasser l'examen. 
 Remarque : une fois la reprise définie, tous les enregistrements d'examen précédents seront supprimés pour cet étudiant. 
 
 - Pour supprimer les enregistrements d'examen de cet étudiant, cliquez sur le bouton Supprimer les enregistrements. 
 Remarque : la suppression des résultats des examens est irrécupérable. Soyez prudent lorsque vous enregez cette action. 
 
 En utilisant ces actions, vous pouvez consulter le journal d'examen, définir une reprise pour l'étudiant ou supprimer les dossiers d'examen si nécessaire. Ces options offrent une flexibilité dans la gestion des examens non notés et garantissent des résultats d'examen exacts et à jour dans la plate-forme ULearning.
