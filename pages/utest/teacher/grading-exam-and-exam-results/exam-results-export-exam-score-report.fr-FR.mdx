# Exportation du rapport de score d'examen 

 Pour exporter le rapport de score d'examen dans ULearning, procédez comme suit : 
 
 ### Étape 1 : Exportation du rapport de score d'examen 
 
 - Sur l'écran Résultats de l'examen, cliquez sur le menu déroulant. 
 - Dans le menu déroulant, sélectionnez le bouton "Exporter les scores". 
 ![]( /img/utest/teacher/result11.png) 
 
 Le rapport de score d'examen sera téléchargé sur votre ordinateur local. 
 ![]( /img/utest/teacher/result12.png) 

 **Remarque :** Seuls les scores avec les informations sur la classe/l'étudiant sont exportés dans un fichier Excel. 

 En suivant ces étapes, vous pouvez facilement exporter le rapport de score d'examen à partir d'ULearning et y accéder sur votre ordinateur local pour une analyse plus approfondie et une tenue de dossiers.
