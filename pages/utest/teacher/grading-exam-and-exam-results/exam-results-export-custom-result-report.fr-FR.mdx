# Exportation du rapport de résultats personnalisé 
 
 Pour exporter un rapport de résultats personnalisé dans ULearning, procédez comme suit : 
 
 ### Étape 1 : Exportation du rapport de résultats personnalisé 
 - Sur l'écran Résultats de l'examen, cliquez sur le menu déroulant. 
 - Dans le menu déroulant, sélectionnez le bouton "Exporter les résultats personnalisés". 
 
 ![]( /img/utest/teacher/result16.png) 
 
 Il ouvre l'écran Rapport personnalisé de l'examen. 
 ![]( /img/utest/teacher/result17.png) 
 
 - Vous pouvez prendre l'enseignant dont vous souhaitez inclure les classes dans le rapport [1]. 

 Remarque : vous ne pouvez en sélectionner qu'un seul. 
 - Ensuite, vous pouvez sélectionner les classes que vous souhaitez inclure les données de l'examen dans le rapport personnalisé [2]. 
 - Vérifiez toutes les données que vous souhaitez signaler [3]. 
 - Si plusieurs classes sont incluses, vous pouvez sélectionner si vous voulez mettre toutes les classes dans un fichier Excel ou si vous voulez les séparer dans différents fichiers (une classe Excel) [4]. 
 - Ensuite, vous pouvez configurer la façon dont les données doivent être commandées/triées [5]. 
 - Enfin, cliquez sur le bouton Exporter les résultats personnalisés pour exporter votre rapport personnalisé. 
 
 
 Voici un exemple de rapport personnalisé : 
 ![]( /img/utest/teacher/result18.png) 
 
 En suivant ces étapes, vous pouvez exporter un rapport de résultats personnalisé à partir d'ULearning, ce qui vous permet de générer un rapport spécialisé en fonction des critères et préférences souhaités. Cela offre une flexibilité dans l'analyse et le rapport des données d'examen en fonction de vos besoins spécifiques.
