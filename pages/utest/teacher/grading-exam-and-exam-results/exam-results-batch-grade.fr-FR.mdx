
# Note du lot pour les résultats de l'examen 

 Pour noter les examens par lots dans ULearning, suivez ces étapes : 

 ### Étape 1 : Accès à l'écran Batch Grade 

 Dans l'écran Résultats de l'examen, cliquez sur le bouton Batch Grade. 
 ![]( /img/utest/teacher/result09.png) 
 
 Il ouvre l'écran Batch Grade. 
 ![]( /img/utest/teacher/result10.png) 
 
 - Utilisez les liens de navigation du volet gauche pour passer à la question suivante [1]. 
 - Vous pouvez entrer une note pour l'étudiant [2], puis cliquer sur le bouton Enregistrer pour enregistrer la note [3]. 
 - Ou vous pouvez cliquer sur le bouton Batch Grade [4] pour donner la même note à tous les élèves.
