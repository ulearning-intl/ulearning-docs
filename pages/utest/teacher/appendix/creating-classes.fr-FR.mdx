
# Création de cours 

 Pour créer des classes dans un cours dans ULearning, procédez comme suit : 
 
 ### Étape 1 : Accédez à l'onglet Membres 
 - Cliquez sur l'onglet "Membres" dans le volet de gauche. 
 ![]( /img/utest/teacher/appendix05.png) 
 
 ### Étape 2 : Créer une classe 
 - Cliquez sur le bouton "Créer une classe". 
 
 **Remarque :** Lorsque vous créez un cours, le système crée automatiquement une classe nommée "Classe par défaut" pour vous. Ce guide explique comment créer une classe supplémentaire, car il est courant d'enseigner plusieurs classes pour un cours au cours d'un semestre. 
 ![]( /img/utest/teacher/appendix06.png) 
 
 ### Étape 3 : Remplissez les détails de la classe 
 - Dans la fenêtre contextuelle Créer une classe : 
     - Entrez le nom de la classe, par exemple, "Monday_9am". 
         - Remarque : Vous pouvez créer plusieurs classes à la fois en entrant le nom de chaque classe sur une ligne distincte. 
     - Cochez ou décochez la case devant "L'étudiant a besoin d'une approbation pour rejoindre la classe" en fonction de vos préférences. 
     - Cliquez sur le bouton "Confirmer" en bas. 
 
 ![]( /img/utest/enseignant/annexe07.png) 
 
 ### Étape 4 : Vérifier la classe créée 
 - Vous remarquerez que la classe nouvellement créée a été ajoutée à la liste des classes de cours. 
 
 ![]( /img/utest/enseignant/annexe08.png) 
 
 En suivant ces étapes, vous pouvez créer plusieurs classes dans un cours dans ULearning. Cela vous permet d'organiser et de gérer différentes sections de classe ou sessions pour le même cours. N'oubliez pas de saisir correctement les noms de classe et de préciser si l'approbation de l'élève est requise pour rejoindre la classe.
