
# Créer un nouveau cours 
 
 Pour créer un nouveau cours dans ULearning, procédez comme suit : 
 
 ### Étape 1 : Accéder à la page Créer un cours 
 - Cliquez sur le bouton "Créer un cours". 
 
 ![]( /img/utest/teacher/appendix01.png) 
 
 ### Étape 2 : Détails du cours 
 - Entrez le nom du cours. 
 - Sélectionnez une couverture de cours pour votre cours. 
 - Cliquez sur le bouton "Créer". 
 
 ![]( /img/utest/teacher/appendix02.png) 
 
 Le cours nouvellement créé est répertorié comme le premier dans votre liste de cours. 
 ![]( /img/utest/teacher/appendix03.png) 
 
 **Remarque :** Assurez-vous de suivre la politique de nommage des cours de votre école lorsque vous donnez un nom à votre cours. 
 
 **Astuce :** Vous pouvez télécharger votre propre image comme couverture du cours. Assurez-vous que la taille de l'image est de 800px * 450px pour un affichage optimal. 
 
 ### Étape 3 : Accès à la page d'accueil du cours 
 - Cliquez sur le cours nouvellement créé dans la liste des cours. 
 - Cela ouvrira la page d'accueil du cours, où vous pourrez commencer à ajouter du contenu et à gérer le cours. 
 ![]( /img/utest/teacher/appendix04.png) 
 
 **Remarque :** Le cours nouvellement créé sera répertorié comme le premier de votre liste de cours. 
 
 En suivant ces étapes, vous pouvez créer un nouveau cours dans ULearning, définir un nom de cours, sélectionner une image de couverture de cours et accéder à la page d'accueil du cours pour commencer à ajouter du contenu et à gérer le cours. N'oubliez pas de respecter toutes les politiques de nommage définies par votre école et de vous assurer que l'image de couverture du cours est correctement dimensionnée pour un affichage optimal.
