# Creating an Exam Manually

If you prefer to manually add questions to an exam one by one, you can follow these steps:

### Step 1: Access the Exam Editor

- From the course homepage, click on the Exams tab.
![](/img/utest/teacher/exam01.png)

### Step 2: Open the Exam Bank

- Click the Exam Bank button.
![](/img/utest/teacher/exam02.png)

### Step 3: Add a New Exam

- Click the Add Exam button.
![](/img/utest/teacher/exam03.png)

### Step 4: Configure Exam Details

- On the Exam Editor screen:
  - Enter the exam name.
  - Optionally, enter points in batch for each question by selecting the desired number of points and clicking the Confirm button. This ### step allows you to assign the same number of points to each question. If you want to assign different numbers of points, you can skip this step.
  - Select the number of plays you want to set up for audio (if applicable) and click the Confirm button.
![](/img/utest/teacher/exam04.png)

### Step 5: Add an Exam Section

- Click the "+" sign to add an exam section.
![](/img/utest/teacher/exam05.png)

### Step 6: Configure Section Details

- In the section popup window:
  - Enter the section details or select options as desired.
  - Click the Confirm button.
![](/img/utest/teacher/exam06.png)

### Step 7: Select Questions

- Click the "Select Questions" link.

![](/img/utest/teacher/exam07.png)

### Step 8: Choose Questions

- From the Select Question screen:
  - Select the questions you want to add to this exam section.
  - Click the Insert button.

![](/img/utest/teacher/exam08.png)

### Step 9: Adjust Questions and Points

- The selected questions will be inserted into the exam section:
  - You can assign the same points to all questions at once or set a different score for each question individually.
  - You can use the Up/Down arrow to adjust the order of the questions.
![](/img/utest/teacher/exam09.png)

### Step 10: Finish Exam Creation

- Click the Finish button at the bottom of the Exam Editor screen.
![](/img/utest/teacher/exam10.png)

### Step 11: Configure Exam Settings

- On the Exam Settings screen:
  - Enter the Suggestion Duration in minutes.
  - The Exam Points will be automatically populated based on the points you assigned to each question in the previous step. If you want to make changes to the Exam Points, click the Go Back button to return to the previous step.
  - Select the Share option.
  - Click the Submit button.

![](/img/utest/teacher/exam11.png)
### Step 12: View Exam in Exam Bank

- The newly created exam will now be listed in the Exam Bank screen.

![](/img/utest/teacher/exam12.png)

By following these steps, you can manually create an exam by selecting questions, assigning points, and configuring other settings to customize the exam according to your requirements.
