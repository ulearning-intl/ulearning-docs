
# Création manuelle d'un examen 
 
 Si vous préférez ajouter manuellement des questions à un examen une par une, vous pouvez suivre ces étapes : 
 
 ### Étape 1 : Accéder à l'éditeur d'examen 
 
 - Sur la page d'accueil du cours, cliquez sur l'onglet Examens. 
 ![]( /img/utest/teacher/exam01.png) 
 
 ### Étape 2 : Ouvrir la banque d'examens 
 
 - Cliquez sur le bouton Banque d'examens. 
 ![]( /img/utest/teacher/exam02.png) 
 
 ### Étape 3 : Ajouter un nouvel examen 
 
 - Cliquez sur le bouton Ajouter un examen. 
 ![]( /img/utest/teacher/exam03.png) 
 
 ### Étape 4 : Configurer les détails de l'examen 
 
 - Sur l'écran de l'éditeur d'examen : 
     - Entrez le nom de l'examen. 
     - En option, entrez des points par lots pour chaque question en sélectionnant le nombre de points souhaité et en cliquant sur le bouton Confirmer. Cette étape ### vous permet d'attribuer le même nombre de points à chaque question. Si vous souhaitez attribuer un nombre différent de points, vous pouvez ignorer cette étape. 
     - Sélectionnez le nombre de lectures que vous souhaitez configurer pour l'audio (le cas échéant) et cliquez sur le bouton Confirmer. 
 ![]( /img/utest/teacher/exam04.png) 
 
 ### Étape 5 : Ajouter une section d'examen 
 
 - Cliquez sur le signe "+" pour ajouter une section d'examen. 
 ![]( /img/utest/teacher/exam05.png) 
 
 ### Étape 6 : Configurer les détails de la section 
 
 - Dans la fenêtre contextuelle de la section : 
     - Entrez les détails de la section ou sélectionnez les options comme vous le souhaitez. 
     - Cliquez sur le bouton Confirmer. 
 ![]( /img/utest/teacher/exam06.png) 
 
 ### Étape 7 : Sélectionnez les questions 
 
 - Cliquez sur le lien "Sélectionner des questions". 
 
 ![]( /img/utest/teacher/exam07.png) 
 
 ### Étape 8 : Choisissez des questions 
 
 - À partir de l'écran Sélectionner une question : 
     - Sélectionnez les questions que vous souhaitez ajouter à cette section de l'examen. 
     - Cliquez sur le bouton Insérer. 
 
 ![]( /img/utest/teacher/exam08.png) 
 
 ### Étape 9 : Ajuster les questions et les points 
 - Les questions sélectionnées seront insérées dans la section de l'examen : 
     - Vous pouvez attribuer les mêmes points à toutes les questions à la fois ou définir un score différent pour chaque question individuellement. 
     - Vous pouvez utiliser la flèche haut/bas pour ajuster l'ordre des questions. 
 ![]( /img/utest/teacher/exam09.png) 
 
 ### Étape 10 : Terminer la création de l'examen 
 
 - Cliquez sur le bouton Terminer en bas de l'écran de l'éditeur d'examen. 
 ![]( /img/utest/teacher/exam10.png) 
 
 ### Étape 11 : Configurer les paramètres de l'examen 
 
 - Sur l'écran Paramètres de l'examen : 
     - Entrez la durée de la suggestion en quelques minutes. 
     - Les points d'examen seront automatiquement remplis en fonction des points que vous avez attribués à chaque question à l'étape précédente. Si vous souhaitez apporter des modifications aux points d'examen, cliquez sur le bouton Revenir en arrière pour revenir à l'étape précédente. 
     - Sélectionnez l'option Partager. 
     - Cliquez sur le bouton Soumettre. 
 
 ![]( /img/utest/teacher/exam11.png) 
 ### Étape 12 : Voir l'examen dans la banque d'examens 
 
 - L'examen nouvellement créé sera désormais répertorié dans l'écran de la banque d'examens. 
 
 ![]( /img/utest/teacher/exam12.png) 
 
 En suivant ces étapes, vous pouvez créer manuellement un examen en sélectionnant des questions, en attribuant des points et en configurant d'autres paramètres pour personnaliser l'examen en fonction de vos besoins.
