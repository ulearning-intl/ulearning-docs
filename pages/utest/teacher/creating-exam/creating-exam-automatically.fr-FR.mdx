# Création automatique d'un examen 
 
 Si vous préférez définir la structure de l'examen ou utiliser une structure d'examen prédéfinie et que le système génère automatiquement l'examen pour vous, vous pouvez suivre ces étapes : 
 
 ### Étape 1 : Accéder à l'éditeur d'examen aléatoire 
 
 - À partir de l'écran Banque d'examens, cliquez sur le bouton Examen aléatoire. 
 
 ![]( /img/utest/teacher/exam13.png) 
 
 Remarque : cette méthode vous permet de créer plusieurs examens en un seul coup en fonction de la même structure d'examen. Ces examens peuvent être utilisés dans la même offre d'examen pour éviter la tricherie. Les examens générés seront stockés dans la banque d'examens et peuvent être utilisés plusieurs fois. Vous pouvez également partager ces examens avec d'autres personnes. 
 
 ### Étape 2 : Configurer les détails de l'examen 

 L'éditeur d'examen aléatoire (examen généré automatiquement) s'ouvrira dans une nouvelle fenêtre. 
 ![]( /img/utest/teacher/exam14.png) 
 
 - Entrez le nom de l'examen [1]. 
 - Entrez une durée d'examen suggérée en quelques minutes [2]. 
 - Sélectionnez le nombre d'examens que vous souhaitez générer [3]. 
 - Vérifiez la source de la question à partir de laquelle vous souhaitez tirer les questions [4]. 
 - Spécifiez les didacticiels/modules auxquels les questions doivent être liées [5]. 
 - Vous pouvez sélectionner un modèle de structure d'examen prédéfini [6]. Le modèle définit la façon dont l'examen doit être structuré et comment les questions doivent être tirées. 
 - Vous pouvez cliquer sur "Gérer mon modèle" pour modifier vos modèles [7]. 
 - Si vous choisissez un modèle prédéfini, il structurera automatiquement votre examen. 
 - Si vous choisissez de définir la structure à la volée : 
     - Cliquez pour entrer le nom de la section [8]. 
     - Cliquez pour entrer les instructions de la section [9]. 
     - Cliquez sur le lien "Ajouter un type de question" pour ajouter un type de question à dessiner [10]. Par exemple, vous pouvez ajouter le type à choix multiples. 
         - Entrez le nombre de questions à dessiner [11]. 
         - Entrez les points pour chaque question [12]. 
     - Répétez l'étape "Ajouter un type de question" pour ajouter d'autres types de questions. 
     - Pour ajouter une autre section, cliquez sur le lien "+Ajouter une nouvelle section". 
     - Si vous souhaitez enregistrer la structure en tant que modèle pour une utilisation future, cochez la case "Enregistrer le modèle après avoir généré l'examen". 
 - Cliquez sur le bouton Soumettre pour générer l'examen. 
 
 ### Étape 3 : Voir l'examen généré 
 
 - L'examen nouvellement généré sera ajouté à votre banque d'examens. 
 
 ![]( /img/utest/teacher/exam15.png) 
 
 En suivant ces étapes, vous pouvez générer automatiquement des examens en fonction de la structure d'examen que vous définissez ou choisir parmi des modèles prédéfinis. Cela vous permet de créer rapidement plusieurs examens avec des questions aléatoires tout en conservant le contrôle de la structure des examens et des sources de questions.
