
# Rapport d'analyse de visualisation/exportation 
 
 Pour afficher ou exporter un rapport d'analyse dans ULearning, procédez comme suit : 
 
 ### Étape 1 : Accès au rapport d'analyse 

 - Sur l'écran Gestion des examens, cliquez sur l'onglet "Rapport d'analyse" [1]. 
 - Cela remplira les données d'analyse pour l'examen sélectionné. 
 
 ![]( /img/utest/teacher/analysis01.png) 
 
 - Vous pouvez sélectionner l'analyse de la classe que vous souhaitez présenter dans le rapport [2]. 
 - La section Informations de base [3] comprend l'analyse de : 
     - Taux de réussite/d'examen 
     - Le temps utilisé par les candidats pour passer l'examen 
     - Les appareils utilisés par les candidats pour passer l'examen 
 - Vous pouvez choisir la façon dont les données sont représentées pour cette section : Comptes vs Pourcentage [6]. 
 - La section Analyse des examens [4] comprend l'analyse pour : 
     - Score moyen 
     - Score d'écart type 
     - Répartition des scores 
 - La section Analyse des examens [5] comprend l'analyse pour : 
     - Statistiques sur l'utilisation et le score des examens 
     - Types de questions d'examen et statistiques de scores 
 - Vous pouvez choisir comment les données sont représentées pour cette section : Graphique vs Tableau [7]. 
 - Pour enregistrer une copie du rapport d'analyse au format PDF dans votre ordinateur local, cliquez sur le bouton Exporter le rapport d'analyse. 
 
 En suivant ces étapes, vous pouvez consulter et analyser les données de l'examen à l'aide de la fonction Rapport d'analyse dans ULearning. Le rapport fournit des informations sur les taux d'achèvement des examens, le temps passé, l'utilisation de l'appareil, les scores moyens, la distribution des scores, et plus encore. Vous pouvez personnaliser le format de représentation des données et exporter le rapport d'analyse sous forme de fichier PDF pour plus de référence ou de partage avec d'autres. 
 
 N'oubliez pas d'examiner et d'interpréter attentivement le rapport d'analyse pour obtenir des informations précieuses sur les performances et les tendances liées à l'examen.
