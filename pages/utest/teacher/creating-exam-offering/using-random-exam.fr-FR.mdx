# Création d'une offre d'examen (examen de cours) - Utilisation d'un examen aléatoire 
 
 En plus d'utiliser un examen pré-créé, vous pouvez également créer une offre d'examen à l'aide d'une structure d'examen générée aléatoirement. Cette méthode permet au système de générer instantanément un examen aléatoire pour chaque étudiant qui passe l'examen. Voici les étapes : 
 
 ### Étape 1 : Accéder à l'écran d'examen 
 - Sur l'écran de l'examen, cliquez sur le bouton Ajouter un examen de cours. 
 ![]( /img/utest/teacher/offering07.png) 
 
 ### Étape 2 : Configurer les détails de l'offre d'examen 
 ![]( /img/utest/teacher/offering08.png) 
 
 - Sur l'écran Ajouter l'examen du cours, entrez le nom de l'examen du cours [1], tel que "CA106A Midterm Exam". 
 - Sélectionnez l'option "Examen aléatoire" [2]. 
 - Cliquez sur le bouton Définir la structure de l'examen [3]. Cela ouvrira l'écran Définir la structure de l'examen. 
 
 ### Étape 3 : Définir la structure de l'examen 
 
 - Sur l'écran Définir la structure de l'examen : 
     - Entrez un nom pour la structure de l'examen, tel que "Midterm - Random". 
     - Cliquez sur le bouton + Ajouter une section pour ajouter une section d'examen à la structure. 
         ![]( /img/utest/teacher/offering09.png) 
         
         - Entrez le nom de la section, par exemple "Insurance Introduction". 
         - Cliquez sur le bouton + Ajouter un type de question pour ajouter un type de question pour cette section. Vous pouvez ajouter plusieurs types de questions pour une section. 
         ![]( /img/utest/teacher/offering10.png) 
 
         ![]( /img/utest/teacher/offering11.png) 
         - Entrez le nombre de questions que vous souhaitez tirer de la banque de questions pour ce type de question. 
         - Entrez les points par question. 
         - Cliquez sur le lien Sélectionner un répertoire pour choisir le(s) dossier(s) à partir duquel vous souhaitez dessiner les questions (Ma banque de questions/Questions scolaires). 
         ![]( /img/utest/teacher/offering12.png) 
 
         - Cliquez sur le bouton Confirmer. 
         ![]( /img/utest/teacher/offering13.png) 

     - Vous pouvez ajouter d'autres types de questions en cliquant sur le bouton + Ajouter un type de question, et d'autres sections en cliquant sur le bouton + Ajouter une section. 
     - Cliquez sur le bouton Aperçu de l'examen pour voir un aperçu de l'examen généré. 
     - Pour enregistrer un exemple d'examen généré sur la base de la structure de l'examen dans votre My Exam Bank, cliquez sur le bouton Exporter l'échantillon d'examen. 
     - Si vous êtes satisfait de la structure de l'examen, cliquez sur le bouton Enregistrer en bas de l'écran Définir la structure de l'examen. 
     ![]( /img/utest/teacher/offering14.png) 
 
 - La structure d'examen définie sera ajoutée à l'offre d'examen. 
 
 Remarques : Les étapes ci-dessous seront exactement les mêmes que les étapes que nous avons déjà couvertes dans Création d'une offre d'examen (examen de cours) - Utilisation de l'examen pré-créé. 
 
 - Entrez le score de passage [4], par exemple, 60 points. 
 - Sélectionnez quand afficher les résultats de l'examen [5], par exemple, "Là après la soumission". 
 - Entrez la durée de l'examen en minutes [6], par exemple, 90 minutes. 
 - Sélectionnez ce qu'il faut montrer pour les résultats de l'examen [7], par exemple, "Score". 
 - Sélectionnez le nombre de tentatives d'examen autorisées [8], par exemple, 3 tentatives. 
 - Sélectionnez les appareils d'examen autorisés [9], par exemple, "PC et appareil mobile". 
 - Sélectionnez le paramètre de démarrage tardif [10], par exemple, "5 minutes autorisées". 
 - Sélectionnez la limite de temps pour la soumission de l'examen [11], par exemple, "5 minutes" (un étudiant ne peut soumettre l'examen que 5 minutes après son début). 
 - Choisissez si vous souhaitez autoriser les élèves à télécharger des pièces jointes pour les questions à réponse courte [12]. 
 - Cochez les cases pour activer les fonctionnalités anti-tcheating [13] : surveillance de l'appareil, surveillance IP, vérification du visage et surveillance de la caméra. 
     - Remarque : il s'agit de fonctionnalités optionnelles avec un coût supplémentaire. Par défaut, ils sont désactivés. Si vous souhaitez activer ces fonctionnalités, contactez ULearning à service.sg@ulearning.cn. 
 - Entrez l'heure de disponibilité (début/fin) [14] pour la session d'examen. 
 - Cliquez sur le bouton Sélectionner une classe [15]. Cela ouvrira la fenêtre contextuelle Sélectionner une classe. 
     - Sélectionnez la ou les classes à qui vous souhaitez attribuer l'examen. 
     - Cliquez sur le bouton Confirmer. 
     ![]( /img/utest/teacher/offering15.png) 
 
     - La classe sélectionnée sera ajoutée à la session d'examen. 
     ![]( /img/utest/teacher/offering16.png) 
 - Vous pouvez supprimer la session ou développer/réduire la session pour afficher plus ou moins d'informations sur la session [17]. 
 - Pour ajouter d'autres sessions, cliquez sur le lien "+ Ajouter une session" [18] et répétez le processus. Par exemple, vous pouvez créer une session pour les étudiants à temps plein et une autre session pour les étudiants en formation continue. 
 - Enfin, cliquez sur le bouton Publier l'examen du cours [19]. 
 
 ### Étape 4 : Voir l'offre d'examen créée 
 
 - L'examen de cours nouvellement créé sera répertorié sur l'écran des examens. 
 ![]( /img/utest/teacher/offering17.png) 
 
 En suivant ces étapes, vous pouvez créer une offre d'examen en utilisant une structure d'examen générée aléatoirement. Cela permet au système de générer un examen unique pour chaque étudiant, en tirant des questions à partir des dossiers de la banque de questions spécifiés. Vous pouvez configurer divers paramètres pour l'offre d'examen, y compris la note de passage, la durée de l'examen et les fonctionnalités anti-tcheating. De plus, vous pouvez attribuer l'offre d'examen à des classes ou à des groupes d'étudiants spécifiques.
