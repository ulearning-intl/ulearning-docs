
# 创建考试课程（课程考试）-使用预创建的考试 
 
 在创建考试课程或课程时，您有两种选择：使用预先创建的考试或创建即时生成的随机考试。 在本节中，我们将重点介绍使用预先创建的考试来创建考试产品。 以下步骤如下： 

 ### 第 1 步：访问考试屏幕 
 
 - 在考试屏幕上，单击添加课程考试按钮。 
 ![](/img/utest/teacher/offering01.png)
 
 ### 第 2 步：配置考试课程详情 
 
 ![](/img/utest/teacher/offering02.png)
 
 - 在添加课程考试屏幕上，输入课程考试名称[1]，例如“CA106A期中考试”。 
 - 为课程考试选择“预定义考试”选项[2]。 
 - 单击选择考试按钮[3]。 这将打开选择考试屏幕。 
     - 选择您要包含在课程考试中的考试。 
     - 点按“插入考试”按钮。 
     - 所选考试将添加到“添加课程考试”屏幕中。 
 ![](/img/utest/teacher/offering03.png)
 
 - 输入及格分数[4]，例如，60分。 
 - 选择何时显示考试结果[5]，例如“提交后立即”。 
 - 以分钟为单位输入考试时长[6]，例如90分钟。 
 - 选择要显示的考试结果 [7]，例如“评分”。 
 - 选择允许的考试尝试次数[8]，例如3次尝试。 
 - 选择允许的考试设备[9]，例如“PC和移动设备”。 
 - 选择延迟开始设置[10]，例如“允许5分钟”。 
 - 选择提交考试的时间限制[11]，例如“5分钟”（学生只能在考试开始后5分钟提交考试）。 
 - 选择是否允许学生上传简短回答问题的附件[12]。 
 - 选中复选框以启用防作弊功能[13]：设备监控、IP监控、人脸验证和相机监控。 
     - 注意：这些是可选功能，需支付额外费用。 默认情况下，它们将被停用。 如果您想激活这些功能，请通过service.sg@ulearning.cn联系ULearning。 
 - 输入考试课程的空闲时间（开始/结束）[14]。 
 - 单击选择班级按钮[15]。 这将打开选择类弹出窗口。 
     - 选择要将考试分配给的班级。 
     - 点按“确认”按钮。 
 ![](/img/utest/teacher/offering04.png)
 
 - 所选班级将被添加到考试课程中。 
 ![](/img/utest/teacher/offering05.png)
 
 - 您可以删除会话或展开/折叠会话以查看或多或少的会话信息[17]。 
 - 要添加更多会话，请单击“+添加会话”链接[18]并重复该过程。 例如，您可以为全日制学生创建一个会话，并为继续教育学生创建另一个会话。 
 - 最后，单击发布课程考试按钮[19]。 
 
 ### 第 3 步：查看已创建的考试产品 
 
 - 新创建的课程考试将列在考试屏幕上。 
 ![](/img/utest/teacher/offering06.png)
 
 通过按照以下步骤操作，您可以使用预先创建的考试来创建考试产品。 这允许您选择现有的考试，并配置特定于考试产品的设置，例如及格分数、考试持续时间和反作弊功能。 您还可以将考试课程分配给特定班级或学生群体。
