
# 创建考试课程（课程考试）-使用随机考试 
 
 除了使用预先创建的考试外，您还可以使用随机生成的考试结构创建考试课程。 这种方法允许系统立即为每个参加考试的学生生成随机考试。 以下步骤如下： 
 
 ### 第 1 步：访问考试屏幕 
 
 - 在考试屏幕上，单击添加课程考试按钮。 
![](/img/utest/teacher/offering07.png)

 ### 第 2 步：配置考试课程详情 
![](/img/utest/teacher/offering08.png)
 
 - 在添加课程考试屏幕上，输入课程考试名称[1]，例如“CA106A期中考试”。 
 - 选择“随机考试”选项[2]。 
 - 单击设置考试结构按钮[3]。 这将打开“设置考试结构”屏幕。 
 
 ### 第 3 步：定义考试结构 
 - 在设置考试结构屏幕上： 
     - 输入考试结构的名称，例如“Midterm - Random”。 
     - 单击+添加部分按钮将考试部分添加到结构中。 
     ![](/img/utest/teacher/offering09.png)
 
     - 输入部分名称，例如“保险简介”。 
     - 单击+添加问题类型按钮为本节添加问题类型。 您可以为一个部分添加多种问题类型。 
     ![](/img/utest/teacher/offering10.png) 
 
     ![](/img/utest/teacher/offering11.png)
     - 输入您要从此问题类型的问题库中提取的问题数量。 
     - 输入每个问题的要点。 
     - 单击选择目录链接以选择要从中绘制问题的文件夹（我的问题库/学校问题）。 
     ![](/img/utest/teacher/offering12.png)

     - 点按“确认”按钮。 
     ![](/img/utest/teacher/offering13.png)
 
 - 您可以通过单击+添加问题类型按钮添加更多问题类型，通过单击+添加部分按钮添加更多部分。 
 - 单击预览考试按钮以查看生成的考试的预览。 
 - 要将根据考试结构生成的考试样本保存到您的“我的考试银行”，请单击“导出考试样本”按钮。 
 - 如果您对考试结构感到满意，请单击设置考试结构屏幕底部的保存按钮。 
  ![](/img/utest/teacher/offering14.png) 
 
 - 定义的考试结构将添加到考试课程中。 
 
 注意：以下步骤与我们在创建考试优惠（课程考试）-使用预创建的考试中已经介绍的步骤完全相同。 
 
 - 输入及格分数[4]，例如，60分。 
 - 选择何时显示考试结果[5]，例如“提交后立即”。 
 - 以分钟为单位输入考试时长[6]，例如90分钟。 
 - 选择要显示的考试结果 [7]，例如“评分”。 
 - 选择允许的考试尝试次数[8]，例如3次尝试。 
 - 选择允许的考试设备[9]，例如“PC和移动设备”。 
 - 选择延迟开始设置[10]，例如“允许5分钟”。 
 - 选择提交考试的时间限制[11]，例如“5分钟”（学生只能在考试开始后5分钟提交考试）。 
 - 选择是否允许学生上传简短回答问题的附件[12]。 
 - 选中复选框以启用防作弊功能[13]：设备监控、IP监控、人脸验证和相机监控。 
     - 注意：这些是可选功能，需支付额外费用。 默认情况下，它们将被停用。 如果您想激活这些功能，请通过service.sg@ulearning.cn联系ULearning。 
 - 输入考试课程的空闲时间（开始/结束）[14]。 
 - 单击选择班级按钮[15]。 这将打开选择类弹出窗口。 
     - 选择要将考试分配给的班级。 
     - 点按“确认”按钮。 
 ![](/img/utest/teacher/offering15.png)

 - 所选班级将被添加到考试课程中。 
 ![](/img/utest/teacher/offering16.png)
 - 您可以删除会话或展开/折叠会话以查看或多或少的会话信息[17]。 
 - 要添加更多会话，请单击“+添加会话”链接[18]并重复该过程。 例如，您可以为全日制学生创建一个会话，并为继续教育学生创建另一个会话。 
 - 最后，单击发布课程考试按钮[19]。 

 ### 第4步：查看创建的考试产品 

 - 新创建的课程考试将列在考试屏幕上。 
 ![](/img/utest/teacher/offering17.png)

 通过按照这些步骤操作，您可以使用随机生成的考试结构创建考试课程。 这允许系统为每个学生生成一个唯一的考试，从指定的问题库文件夹中绘制问题。 您可以为考试产品配置各种设置，包括及格分数、考试持续时间和反作弊功能。 此外，您可以将考试课程分配给特定班级或学生群体。
