
# Création d'une offre d'examen (examen de cours) - Utilisation d'un examen pré-créé 
 
 Lors de la création d'une offre ou d'une session d'examen, vous avez deux options : utiliser un examen pré-créé ou créer un examen aléatoire généré instantanément. Dans cette section, nous nous concentrerons sur la création d'une offre d'examen à l'aide d'un examen pré-créé. Voici les étapes : 
 
 ### Étape 1 : Accéder à l'écran d'examen 
 
 - À partir de l'écran Examen, cliquez sur le bouton Ajouter un examen de cours. 
 ![]( /img/utest/teacher/offering01.png) 
 
 ### Étape 2 : Configurer les détails de l'offre d'examen 
 
 ![]( /img/utest/teacher/offering02.png) 
 
 - Sur l'écran Ajouter un examen de cours, entrez le nom de l'examen de cours [1], tel que "CA106A Examen de mi-parcours". 
 - Sélectionnez l'option "Examen prédéfini" pour l'examen de cours [2]. 
 - Cliquez sur le bouton Sélectionner l'examen [3]. Cela ouvrira l'écran Sélectionner l'examen. 
     - Sélectionnez l'examen que vous souhaitez inclure dans l'examen de cours. 
     - Cliquez sur le bouton Insérer l'examen. 
     - L'examen sélectionné sera ajouté à l'écran Ajouter un examen de cours. 
 ![]( /img/utest/teacher/offering03.png) 
 
 - Entrez le score de passage [4], par exemple, 60 points. 
 - Sélectionnez quand afficher les résultats de l'examen [5], par exemple, "Juste après la soumission". 
 - Entrez la durée de l'examen en minutes [6], par exemple, 90 minutes. 
 - Sélectionnez ce qu'il faut afficher pour les résultats de l'examen [7], par exemple, "Score". 
 - Sélectionnez le nombre de tentatives d'examen autorisées [8], par exemple 3 tentatives. 
 - Sélectionnez les appareils d'examen autorisés [9], par exemple, "PC et appareil mobile". 
 - Sélectionnez le paramètre de démarrage tardif [10], par exemple, "5 minutes autorisées". 
 - Sélectionnez la limite de temps pour la soumission de l'examen [11], par exemple, "5 minutes" (un étudiant ne peut soumettre l'examen que 5 minutes après son début). 
 - Sélectionnez si vous souhaitez autoriser les étudiants à télécharger des pièces jointes pour les questions à réponse courte [12]. 
 - Cochez les cases pour activer les fonctions anti-triche [13] : Surveillance de l'appareil, Surveillance de l'IP, Vérification du visage et Surveillance de la caméra. 
     - Remarque : il s'agit de fonctionnalités optionnelles avec un coût supplémentaire. Par défaut, ils sont désactivés. Si vous souhaitez activer ces fonctionnalités, contactez ULearning à l'adresse service.sg@ulearning.cn. 
 - Entrez l'heure de disponibilité (début/fin) [14] pour la session d'examen. 
 - Cliquez sur le bouton Sélectionner la classe [15]. Cela ouvrira la fenêtre contextuelle Sélectionner la classe. 
     - Sélectionnez la ou les classes à laquelles vous souhaitez attribuer l'examen. 
     - Cliquez sur le bouton Confirmer. 
 ![]( /img/utest/teacher/offering04.png) 
 
 - La classe sélectionnée sera ajoutée à la session d'examen. 
 ![]( /img/utest/teacher/offering05.png) 

 - Vous pouvez supprimer la session ou développer/réduire la session pour afficher plus ou moins d'informations sur la session [17]. 
 - Pour ajouter plus de sessions, cliquez sur le lien "+ Ajouter une session" [18] et répétez le processus. Par exemple, vous pouvez créer une session pour les étudiants à temps plein et une autre session pour les étudiants en formation continue. 
 - Enfin, cliquez sur le bouton Publier l'examen de cours [19]. 

 ### Étape 3 : Voir l'offre d'examen créée 
 
 - L'examen de cours nouvellement créé sera répertorié sur l'écran Examens. 
 ![]( /img/utest/teacher/offering06.png) 
 
 En suivant ces étapes, vous pouvez créer une offre d'examen à l'aide d'un examen pré-créé. Cela vous permet de sélectionner un examen existant et de configurer les paramètres spécifiques à l'offre d'examen, tels que la note de passage, la durée de l'examen et les fonctions anti-triche. Vous pouvez également attribuer l'offre d'examen à des classes ou à des groupes d'étudiants spécifiques.
