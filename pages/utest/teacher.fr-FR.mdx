# Vue d'ensemble 
 
Ceci est un guide de l'utilisateur pour les enseignants. Le but de ce guide est de donner aux utilisateurs enseignants d'ULearning les connaissances et les compétences nécessaires pour utiliser la fonction d'examen ULearning LMS - U-Test. En suivant les étapes de ce guide, vous pouvez commencer rapidement votre voyage pour effectuer les tâches U-Test les plus courantes en tant qu'enseignant. 
 
Ce guide est organisé autour des tâches nécessaires que vous effectuerez, étape par étape accompagnées de captures d'écran de la plate-forme U-Test.
