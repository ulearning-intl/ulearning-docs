# Création d'un examen dans la banque d'examens 
 
 ### Étape 1 
 Sur la page d'accueil du cours, cliquez sur l'onglet "Examens". 
 ![]( /img/ulearning/teacher/exam01.png) 

 ### Étape 2 
 Cliquez sur le bouton "Banque d'examens". 
 ![]( /img/ulearning/teacher/exam02.png) 
 
 ### Étape 3 
 Cliquez sur le bouton "Ajouter un examen". 
 ![]( /img/ulearning/teacher/exam03.png) 
 
 ### Étape 4 
 Sur la page de l'éditeur d'examen : 
 - Entrez le nom de l'examen. 
 - Entrez des points en masse pour chaque question et cliquez sur le bouton "Confirmer". 
 - Remarque : cette étape est facultative. Il vous permet d'attribuer le même nombre de points pour chaque question. Si vous souhaitez attribuer différents points, vous pouvez sauter cette étape. 
 - Sélectionnez le nombre de lectures que vous souhaitez configurer pour l'audio et cliquez sur le bouton "Confirmer". 
 ![]( /img/ulearning/teacher/exam04.png) 
 
 ### Étape 5 
 Cliquez sur le signe + pour ajouter une section d'examen. 
 ![]( /img/ulearning/teacher/exam05.png) 
 
 ### Étape 6 
 Sur la fenêtre contextuelle de la section : 
 - Entrez les détails de la section ou sélectionnez les options comme vous le souhaitez. 
 - Cliquez sur le bouton "Confirmer". 
 ![]( /img/ulearning/teacher/exam06.png) 
 
 ### Étape 7 
 Cliquez sur le lien "Sélectionner des questions". 
 ![]( /img/ulearning/teacher/exam07.png) 
 
 ### Étape 8 
 À partir de l'écran "Sélectionner une question" : 
 - Sélectionnez les questions que vous souhaitez ajouter à cette section de l'examen. 
 - Cliquez sur le bouton "Insérer". 
 ![]( /img/ulearning/teacher/exam08.png) 
 
 Les questions sélectionnées sont maintenant insérées dans la section de l'examen : 
 - Vous pouvez attribuer les mêmes points à toutes les questions à la fois ou définir un score différent pour chaque question individuellement. 
 - Vous pouvez utiliser la flèche haut/bas pour ajuster l'ordre des questions. 
 ![]( /img/ulearning/teacher/exam09.png) 
 
 Cliquez sur le bouton Terminer en bas de l'écran de l'éditeur d'examen. 
 ![]( /img/ulearning/teacher/exam10.png) 
 
 Sur l'écran "Paramètres de l'examen" : 
 - Entrez la durée de la suggestion en minutes. 
 - Les points de l'examen sont automatiquement remplis en fonction des points que vous avez attribués à chaque question à l'étape précédente. 
 - Remarque : si vous souhaitez apporter des modifications aux points d'examen, cliquez sur le bouton Revenir en arrière pour revenir à l'étape précédente. 
     - Sélectionnez l'option "Partager". 
 - Cliquez sur le bouton "Soumettre". 
 ![]( /img/ulearning/teacher/exam11.png) 
 
 L'examen nouvellement créé est maintenant affiché dans l'écran "Banque d'examens". 
 ![]( /img/ulearning/teacher/exam12.png)
