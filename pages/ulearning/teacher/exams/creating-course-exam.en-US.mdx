# Creating Course Exam 

### Step 1 
From the Exam screen, click the Add Course Exam button.
![](/img/ulearning/teacher/exam13.png)

### Step 2 
On the Add Course Exam page:
- Enter the course exam name, for example, "CA106A Midterm Exam."
- Select the Pre-defined Exam option for Course Exam.
- Click the Select Exam button.
![](/img/ulearning/teacher/exam14.png)

### Step 3 
On the Select Exam screen:
- Select the Exam you want to include in the course exam.
  - Note: You can include multiple exam objects into a course exam as desired.
- Click the Insert Exam button.
![](/img/ulearning/teacher/exam15.png)

### Step 4 
The selected exam is added to the Add Course Exam page:
- Enter the passing score, for example, 10 points.
- Select when to show exam results, for example, Immediately.
- Enter the exam duration in minutes, for example, 20 minutes.
- Select what to show, for example, Score.
- Select the exam attempts allowed, for example, 3 attempts.
- Select the late start setting, for example, 2 minutes allowed.
- Select if you want to enable face verification.
  - Note: Face verification is an optional feature with an extra cost. By default, this function is deactivated. If you want to activate this function, ask your school to contact ULearning at sevice.sg@ulearning.cn.
- Click the + Add Session link.
![](/img/ulearning/teacher/exam16.png)

### Step 5 
Enter the availability time (Start/End) for the exam session and then click the Select Class button.
- Note: If you want to assign this course exam to a single student or a group of selected students, use the Select Student option instead.
![](/img/ulearning/teacher/exam17.png)

### Step 6 
On the Select Class popup window:
- Select the class(es) you want to assign to.
- Click the Confirm button.
![](/img/ulearning/teacher/exam18.png)

### Step 7 
The selected class is added to this exam session:
- To add more sessions, click the + Add Session again and repeat the process. For example, you can create a session for all full-time students and another one for continuing education students.
- Finally, click the Publish Course Exam button.
![](/img/ulearning/teacher/exam19.png)

### Step 8 
The newly created course exam is now listed on the Exams screen.
![](/img/ulearning/teacher/exam20.png)
