# Notation d'examen du cours 

 ### Étape 1 
 À partir de l'écran "Examens" : 
 - Survolez la souris/Cliquez sur le menu déroulant des actions pour l'examen de cours que vous souhaitez noter. 
 - Cliquez sur l'option "Afficher les résultats". 
 ![]( /img/ulearning/teacher/exam21.png) 
 
 ### Étape 2 
 Sur l'écran des "résultats d'examen" de l'examen : 
 - Localisez le candidat/étudiant que vous souhaitez noter. 
 - Survolez la souris/Cliquez sur le menu déroulant des actions pour cet étudiant. 
 - Cliquez sur l'option "Noter l'examen". 
 ![]( /img/ulearning/teacher/exam22.png) 
 
 ### Étape 3 
 Sur la page de notation des examens : 
 - Les numéros de série des questions pour les questions que vous pouvez ou devez noter manuellement sont affichés en haut de l'écran. Par exemple, les questions 1, 2, 5, 6, 13, 14, 20 et 21. 
 - Localisez ces questions une par une sur l'écran. 
 - Passez en revue la réponse de l’étudiant et saisissez manuellement une note pour chaque question.  
 ![]( /img/ulearning/teacher/exam23.png) 
 
 ### Étape 4 
 Pour donner un commentaire personnalisé à la notation que vous avez fait pour la question : 
 - Cliquez sur le lien Ajouter/Modifier des commentaires. 
 ![]( /img/ulearning/teacher/exam24.png) 
 
 - Saisissez vos commentaires dans le champ "Commentaires de l'enseignant". 
 - Cliquez sur le lien "Terminer". 
 ![]( /img/ulearning/teacher/exam25.png) 
 
 - Les commentaires sont ajoutés à la notation de cette question. 
 ![]( /img/ulearning/teacher/exam26.png) 
 
 ### Étape 5 
 S'il y a plusieurs sections pour cet examen, assurez-vous de naviguer dans toutes les sections et de noter toutes les questions du test. 
 - Cliquez sur le bouton "Enregistrer la notation" en haut de l'écran de l'examen. 
 ![]( /img/ulearning/teacher/exam27.png) 
 
 ### Étape 6 
 Dans la fenêtre contextuelle "Fermer", cliquez sur le bouton "Fermer". 
 ![]( /img/ulearning/teacher/exam28.png) 
 
 ### Étape 7 
 Sur l'écran "Résultats de l'examen", les résultats de l'examen de l'élève sont mis à jour et rafraîchis : 
 - Le statut est changé de "Non noté" à "Noté". 
 - Le score manuel/score total est passé d'en attente aux scores réels. Par exemple, 95/50. 
 - Le score total est passé de "En attente" au "score réel". 
 ![]( /img/ulearning/teacher/exam29.png) 
 
 ### Étape 8 
 Pour noter l'examen pour d'autres étudiants, répétez les étapes. 
 
 Avant de noter, assurez-vous d'avoir sélectionné l'examen de cours approprié et l'étudiant à noter.
