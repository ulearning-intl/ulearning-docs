# Création d'un examen de cours 
 
 ### Étape 1 
 Dans l'écran de l'examen, cliquez sur le bouton Ajouter un examen de cours. 
 ![]( /Img/ulearning/teacher/exam13.png) 
 
 ### Étape 2 
 Sur la page Ajouter un examen de cours : 
 - Entrez le nom de l'examen du cours, par exemple, "CA106A Examen de mi-parcours". 
 - Sélectionnez l'option Examen prédéfini pour l'examen de cours. 
 - Cliquez sur le bouton Sélectionner l'examen. 
 ![]( /Img/ulearning/teacher/exam14.png) 
 
 ### Étape 3 
 Sur l'écran Sélectionner l'examen : 
 - Sélectionnez l'examen que vous souhaitez inclure dans l'examen du cours. 
 - Remarque : vous pouvez inclure plusieurs objets d'examen dans un examen de cours comme vous le souhaitez. 
 - Cliquez sur le bouton Insérer l'examen. 
 ![]( /Img/ulearning/teacher/exam15.png) 
 
 ### Étape 4 
 L'examen sélectionné est ajouté à la page Ajouter un examen de cours : 
 - Entrez le score de passe, par exemple, 10 points. 
 - Sélectionnez quand afficher les résultats de l'examen, par exemple, Immédiatement. 
 - Entrez la durée de l'examen en minutes, par exemple 20 minutes. 
 - Sélectionnez ce qu'il faut afficher, par exemple, Score. 
 - Sélectionnez les tentatives d'examen autorisées, par exemple, 3 tentatives. 
 - Sélectionnez le paramètre de démarrage tardif, par exemple, 2 minutes autorisées. 
 - Sélectionnez si vous souhaitez activer la vérification faciale. 
 - Remarque : la vérification du visage est une fonctionnalité facultative avec un coût supplémentaire. Par défaut, cette fonction est désactivée. Si vous souhaitez activer cette fonction, demandez à votre école de contacter ULearning à l'adresse sevice.sg@ulearning.cn. 
 - Cliquez sur le lien + Ajouter une session. 
 ![]( /Img/ulearning/teacher/exam16.png) 
 
 ### Étape 5 
 Entrez l'heure de disponibilité (début/fin) de la session d'examen, puis cliquez sur le bouton Sélectionner la classe. 
 - Remarque : si vous souhaitez attribuer cet examen de cours à un seul étudiant ou à un groupe d'étudiants sélectionnés, utilisez plutôt l'option Sélectionner un étudiant. 
 ![]( /Img/ulearning/teacher/exam17.png) 
 
 ### Étape 6 
 Dans la fenêtre contextuelle Sélectionner une classe : 
 - Sélectionnez la ou les classes que vous souhaitez attribuer. 
 - Cliquez sur le bouton Confirmer. 
 ![]( /Img/ulearning/teacher/exam18.png) 
 
 ### Étape 7 
 La classe sélectionnée est ajoutée à cette session d'examen : 
 - Pour ajouter plus de sessions, cliquez à nouveau sur le bouton + Ajouter une session et répétez le processus. Par exemple, vous pouvez créer une session pour tous les étudiants à temps plein et une autre pour les étudiants en formation continue. 
 - Enfin, cliquez sur le bouton Publier l'examen de cours. 
 ![]( /Img/ulearning/teacher/exam19.png) 
 
 ### Étape 8 
 L'examen de cours nouvellement créé est maintenant répertorié sur l'écran Examens. 
 ![]( /Img/ulearning/teacher/exam20.png)
