# Creating Exam in Exam Bank

### Step 1 
From the course homepage, click the Exams tab.
![](/img/ulearning/teacher/exam01.png)

### Step 2 
Click the Exam Bank button.
![](/img/ulearning/teacher/exam02.png)

### Step 3 
Click the Add Exam button.
![](/img/ulearning/teacher/exam03.png)

### Step 4 
On the Exam Editor page:
- Enter the exam name.
- Enter points in batch for each question and click the Confirm button.
  - Note: This step is optional. It allows you to assign the same number of points for each question. If you want to assign different points, you can skip this step.
- Select the number of plays you want to set up for audio and click the Confirm button.
![](/img/ulearning/teacher/exam04.png)

### Step 5 
Click the + sign to add an exam section.
![](/img/ulearning/teacher/exam05.png)

### Step 6 
On the section popup window:
- Enter section details or select options as desired.
- Click the Confirm button.
![](/img/ulearning/teacher/exam06.png)

### Step 7 
Click the Select Questions link.
![](/img/ulearning/teacher/exam07.png)

### Step 8 
From the Select Question screen:
- Select the questions you want to add to this exam section.
- Click the Insert button.
![](/img/ulearning/teacher/exam08.png)

The selected questions are now inserted into the exam section:
- You can assign the same points to all questions at once or set a different score for each question individually.
- You can use the Up/Down arrow to adjust the order of the questions.
![](/img/ulearning/teacher/exam09.png)

Click the Finish button at the bottom of the Exam Editor screen.
![](/img/ulearning/teacher/exam10.png)

On the Exam Settings screen:
- Enter the Suggestion Duration in minutes.
- Exam Points are automatically populated based on the points you assigned to each question in the previous step.
  - Note: If you want to make changes to Exam Points, click the Go Back button to return to the previous step.
- Select the Share option.
- Click the Submit button.
![](/img/ulearning/teacher/exam11.png)

The newly created exam is now listed in the Exam Bank screen.
![](/img/ulearning/teacher/exam12.png)
