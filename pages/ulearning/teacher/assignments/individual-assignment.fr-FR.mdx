# Création et publication d'affectations individuelles

 ### Étape 1 
 Sur la page d'accueil du cours, cliquez sur l'onglet Affectation. 
 ![]( /Img/ulearning/teacher/assignments01.png) 
 
 ### Étape 2 
 
 - Cliquez sur le bouton + Affectation. 
 - Cliquez sur l'option Affectation individuelle. 
 ![]( /Img/ulearning/teacher/assignments02.png) 
 
 ### Étape 3 
 
 - Remplissez les détails de l'affectation ou sélectionnez les options comme vous le souhaitez. 
 ![]( /Img/ulearning/teacher/assignments03.png) 
 
 - Cliquez sur le bouton Sélectionner la classe. 
 ![]( /Img/ulearning/teacher/assignments04.png) 
 
 ### Étape 4 
 
 - Sélectionnez la ou les classes à laquelles vous souhaitez attribuer cette affectation. 
 - Cliquez sur le bouton Confirmer. 
 ![]( /Img/ulearning/teacher/assignments05.png) 
 
 ### Étape 5 
 La ou les classes sont ajoutées et cliquez sur le bouton Publier. 
 ![]( /Img/ulearning/teacher/assignments06.png) 
 
 ### Étape 6 
 L'affectation est maintenant ajoutée à l'écran Affectations. 
 ![]( /Img/ulearning/teacher/assignments07.png)
