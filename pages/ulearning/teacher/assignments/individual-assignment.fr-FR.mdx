# Création et publication des devoirs individuelles

 ### Étape 1 
 Sur la page d'accueil du cours, cliquez sur l'onglet devoirs. 
 ![]( /img/ulearning/teacher/assignments01.png) 
 
 ### Étape 2 
 
 - Cliquez sur le bouton + Devoirs. 
 - Cliquez sur l'option devoirs individuelle. 
 ![]( /img/ulearning/teacher/assignments02.png) 
 
 ### Étape 3 
 
 - Remplissez les détails du devoirs ou sélectionnez les options comme vous le souhaitez. 
 ![]( /img/ulearning/teacher/assignments03.png) 
 
 - Cliquez sur le bouton Sélectionner la classe. 
 ![]( /img/ulearning/teacher/assignments04.png) 
 
 ### Étape 4 
 
 - Sélectionnez la où les classes auxquelles vous souhaitez attribuer ce devoirs. 
 - Cliquez sur le bouton Confirmer. 
 ![]( /img/ulearning/teacher/assignments05.png) 
 
 ### Étape 5 
 Les classes sont ajoutées, cliquez sur le bouton Publier. 
 ![]( /img/ulearning/teacher/assignments06.png) 
 
 ### Étape 6 
 Le devoirs est maintenant ajoutée à l'écran des devoirs. 
 ![]( /img/ulearning/teacher/assignments07.png)
