# Creating and Publishing Quiz

### Step 1 
From the course homepage, click the Assignment tab.
![](/img/ulearning/teacher/assignments18.png)

### Step 2

- Click or mouseover the + Assignment button.
- Click the Quizzes option.
![](/img/ulearning/teacher/assignments19.png)

### Step 3

- Enter the quiz name.
- Enter the quiz instructions.
- Click the Select Questions button.
![](/img/ulearning/teacher/assignments20.png)

### Step 4 
On the Select Question page:
- Select the questions you want to add to this quiz.
- Click the Insert button.
![](/img/ulearning/teacher/assignments21.png)

### Step 5 
The questions are added to the quiz. Click the Next button at the bottom of the Create Quiz page.
![](/img/ulearning/teacher/assignments22.png)

### Step 6

- Enter details or select options for the quiz as desired.
- Click the Select Classes button.
![](/img/ulearning/teacher/assignments23.png)

On the Select Class popup window:
- Select the class(es) you want to assign this quiz to.
- Click the Ok button.
![](/img/ulearning/teacher/assignments24.png)

Click the Publish button.
![](/img/ulearning/teacher/assignments25.png)

The newly created quiz is now listed on the Assignments screen.
![](/img/ulearning/teacher/assignments26.png)
