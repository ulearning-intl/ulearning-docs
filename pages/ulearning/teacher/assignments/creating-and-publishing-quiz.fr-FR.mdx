 #la création et la publication d'un quiz

 ### Étape 1 
 Sur la page d'accueil du cours, cliquez sur l'onglet Devoirs.
![](/img/ulearning/teacher/assignments18.png)

 ### Étape 2 

 - Cliquez ou passez la souris sur le bouton + Devoirs. 
 - Cliquez sur l'option Quiz.
![](/img/ulearning/teacher/assignments19.png)

 ### Étape 3 

 - Entrez le nom du quiz. 
 - Entrez les instructions du quiz. 
 - Cliquez sur le bouton Sélectionner les questions.
![](/img/ulearning/teacher/assignments20.png)

 ### Étape 4 
 Sur la page "Sélectionner une question" : 
 - Sélectionnez les questions que vous souhaitez ajouter à ce quiz. 
 - Cliquez sur le bouton Insérer.
![](/img/ulearning/teacher/assignments21.png)

 ### Étape 5 
 Les questions sont ajoutées au quiz. Cliquez sur le bouton Suivant en bas de la page "Créer un quiz".
![](/img/ulearning/teacher/assignments22.png)

 ### Étape 6 、

 - Entrez les détails ou sélectionnez les options pour le quiz comme vous le souhaitez. 
 - Cliquez sur le bouton "Sélectionner les classes".
![](/img/ulearning/teacher/assignments23.png)

 Dans la fenêtre contextuelle "Sélectionner une classe" : 
 - Sélectionnez là où les classes auxqlles vous souhaitez attribuer ce quiz. 
 - Cliquez sur le bouton Ok.
![](/img/ulearning/teacher/assignments24.png)

Cliquez sur le bouton "Publier".
![](/img/ulearning/teacher/assignments25.png)

Le quiz nouvellement créé est maintenant affiché sur l'écran "Devoirs".
![](/img/ulearning/teacher/assignments26.png)
