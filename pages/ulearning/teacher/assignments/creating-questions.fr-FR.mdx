 # Création de questions 

 ### Étape 1 
 Dans l'écran Affectations, cliquez sur le lien « Retour à la liste des cours ».
![](/img/ulearning/teacher/assignments08.png)

### Étape 2 
 Sur l'écran Liste des cours, cliquez sur le lien "Banque de questions".
![](/img/ulearning/teacher/assignments09.png)

### Étape 3 
 Sur la page Banque de questions, cliquez sur le bouton "Ajouter une question".
![](/img/ulearning/teacher/assignments10.png)

### Étape 4 
 Sur la page de l'éditeur de questions : 
 - Passez la souris sur le type de question à partir du panneau gauche que vous souhaitez créer. 
 - Entrez le nombre de questions que vous souhaitez créer. 
 - Cliquez sur le bouton "Ok".
![](/img/ulearning/teacher/assignments11.png)

 ### Étape 5 
 La ou les questions sont ajoutées dans la zone de contenu de la question du milieu.
![](/img/ulearning/teacher/assignments12.png)

### Étape 6 

 - Entrez les détails de la question comme vous le souhaitez. 
 - Cliquez sur le lien "Sélectionner le module" au bas de la question.
![](/img/ulearning/teacher/assignments13.png)

### Étape 7 
 Dans la fenêtre contextuelle Sélectionner le cours et le module : 
 - Sélectionnez le didacticiel auquel la question est liée, et les titres des modules du didacticiel rempliront la colonne Sélectionner un module. 
 - Sélectionnez le(s) module(s) à laquelle la question est liée. 
 - Cliquez sur le bouton "Ok".
![](/img/ulearning/teacher/assignments14.png)

### Étape 8 

 - Le champ Module connexe est rempli avec les titres de cours/module que vous avez sélectionnés. 
 - Cliquez sur le bouton "Enregistrer".
![](/img/ulearning/teacher/assignments15.png)

### Étape 9 

 - Répétez le processus pour créer/ajouter autant de questions que vous le souhaitez. 
 - Lorsque vous avez terminé, cliquez sur le bouton "Retour" dans le coin supérieur droit de l'écran.
![](/img/ulearning/teacher/assignments16.png)

### Étape 10 

 - Les questions nouvellement créées sont ajoutées à la banque de questions. 
 - Cliquez sur le bouton "Fermer".
![](/img/ulearning/teacher/assignments17.png)
