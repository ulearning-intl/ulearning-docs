# Ajout d'une ressource 

 ### Étape 1 
 Sur l'écran d'accueil du cours, cliquez sur l'onglet Ressources. 
 
 ![]( /img/ulearning/teacher/resource01.png) 

 ### Étape 2 
 À partir de l'écran Ressources : 
 - Cliquez sur le menu déroulant Ajouter une ressource. 
 - Cliquez sur l'option Télécharger le fichier. 
 ![]( /img/ulearning/teacher/resource02.png) 

 ### Étape 3 
 Dans la fenêtre contextuelle Ajouter une ressource, cliquez sur le bouton Téléchargement local. 
 ![]( /img/ulearning/teacher/resource03.png) 
 
 ### Étape 4 
 Dans la fenêtre contextuelle Télécharger, sélectionnez le fichier que vous souhaitez télécharger, puis cliquez sur le bouton Ouvrir. 
 ![]( /img/ulearning/teacher/resource04.png) 
 
 ### Étape 5 
 Dans la fenêtre de téléchargement : 
 - Vous verrez que le fichier que vous venez de télécharger est sélectionné. 
     - Remarque : vous pouvez continuer à télécharger plus de fichiers avant de passer à l'étape suivante. 
 - Cliquez sur le bouton Confirmer (n). 
 ![]( /img/ulearning/teacher/resource05.png) 
 
 ### Étape 6 
 Le fichier est maintenant répertorié sur l'écran Ressources. 
 ![]( /img/ulearning/teacher/resource06.png)
