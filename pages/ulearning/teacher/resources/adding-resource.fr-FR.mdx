# Ajout d'une ressource 

 ### Étape 1 
 Sur l'écran d'accueil du cours, cliquez sur l'onglet Ressources. 
 
 ![]( /img/ulearning/teacher/resource01.png) 

 ### Étape 2 
 À partir de l'écran Ressources : 
 - Cliquez sur le menu déroulant Ajouter une ressource. 
 - Cliquez sur l'option Téléverser le fichier. 
 ![]( /img/ulearning/teacher/resource02.png) 

 ### Étape 3 
 Dans la fenêtre contextuelle Ajouter une ressource, cliquez sur le bouton Téléversement local. 
 ![]( /img/ulearning/teacher/resource03.png) 
 
 ### Étape 4 
 Dans la fenêtre contextuelle Téléverser, sélectionnez le fichier que vous souhaitez Téléverser, puis cliquez sur le bouton Ouvrir. 
 ![]( /img/ulearning/teacher/resource04.png) 
 
 ### Étape 5 
 Dans la fenêtre de téléversement : 
 - Vous verrez que le fichier que vous venez de Téléverser est sélectionné. 
     - Remarque : vous pouvez continuer à Téléverser plus de fichiers avant de passer à l'étape suivante. 
 - Cliquez sur le bouton Confirmer (n). 
 ![]( /img/ulearning/teacher/resource05.png) 
 
 ### Étape 6 
 Le fichier est maintenant lister sur l'écran Ressources. 
 ![]( /img/ulearning/teacher/resource06.png)
