 # Création d'un micro cours 


 ### Étape 1 
 Sur l'écran Ressources : 
 - Cliquez sur le menu déroulant Actions à côté de la ressource que vous souhaitez convertir en micro-cours. 
 - Cliquez sur l'option Enregistrer le micro-cours. 
 ![]( /Img/ulearning/teacher/resource07.png) 

 ### Étape 2 
 Il ouvre la fenêtre Micro Course Editor avec le fichier chargé. 
 - Cliquez sur le bouton Enregistrer la voix off pour la diapositive que vous souhaitez ajouter de l'audio. 
 ![]( /Img/ulearning/teacher/resource08.png) 

 ### Étape 3 
 Lorsque vous avez terminé l'enregistrement, cliquez sur le bouton Arrêter pour terminer l'enregistrement. 
 - Remarque : l'heure d'enregistrement s'affiche à côté du bouton Stop. 
 ![]( /Img/ulearning/enseignant/resource09.png) 

 ### Étape 4 
 Cliquez sur le bouton Lecture pour écouter votre enregistrement. 
 - Si vous n'êtes pas satisfait, cliquez sur le bouton Effacer la voix off et refaites l'enregistrement. 
 ![]( /Img/ulearning/teacher/resource10.png) 

 ### Étape 5 
 Si vous souhaitez effacer la voix off de cette diapositive, cliquez sur le bouton Confirmer. 
 ![]( /Img/ulearning/teacher/resource11.png) 

 ### Étape 6 
 Cliquez sur la diapositive suivante dans le volet des vignettes de la diapositive de gauche et enregistrez la voix off pour cela. 
 - Répétez ce processus pour ajouter la voix off aux diapositives à laquel vous souhaitez ajouter la voix off. 
 ![]( /Img/ulearning/teacher/resource12.png) 

 ### Étape 7 
 Pour ajouter plus de diapositives à ce micro-cours : 
 - Cliquez sur le signe + dans le coin supérieur gauche de l'écran. 
 - Sélectionnez le type de fichier que vous souhaitez ajouter, puis cliquez. 
 ![]( /Img/ulearning/teacher/resource13.png) 

 ### Étape 8 
 Il ouvre la fenêtre contextuelle Ajouter une ressource avec tous les fichiers disponibles du type que vous avez choisi, ou vous pouvez télécharger un fichier à partir de votre ordinateur local : 
 - Sélectionnez le(s) fichier(s). 
 - Cliquez sur le bouton Confirmer (n). 
 ![]( /Img/ulearning/teacher/resource14.png) 

 ### Étape 9 
 Suivez le même processus pour enregistrer la voix off de la ou des diapositives nouvellement ajoutées. 
 ![]( /Img/ulearning/teacher/resource15.png) 

 ### Étape 10 
 Pour ajuster l'ordre des diapositives, faites glisser la diapositive à l'endroit où vous souhaitez qu'elle aille. 
 ![]( /Img/ulearning/teacher/resource16.png) 

 ### Étape 11 
 Pour supprimer une diapositive de votre micro-cours, cliquez sur le petit bouton Supprimer dans le coin supérieur droit de la vignette de la diapositive dans le volet de navigation des diapositives. 
 - Remarque : Soyez prudent avec cette action puisque la suppression est irrécupérable. 
 ![]( /Img/ulearning/teacher/resource17.png) 

 ### Étape 12 
 Cliquez sur le bouton Aperçu pour prévisualiser le micro cours dans l'édition. 
 - Remarque : Le micro-cours est automatiquement enregistré dans la bibliothèque de ressources. Si vous n'êtes pas prêt à publier ce micro-cours, vous avez terminé la création du micro-cours. Vous pouvez fermer la fenêtre de l'éditeur de cours Micro et revenir à la fenêtre Ressource. 
 ![]( /Img/ulearning/teacher/resource18.png) 
 
 ### Étape 13 
 L'aperçu est le même que la vue lorsque vos élèves regardent ce micro-cours. Vous pouvez utiliser le bouton Haut/Bas pour revenir en arrière/avancer dans les diapositives. 
 ![]( /Img/ulearning/teacher/resource19.png)
