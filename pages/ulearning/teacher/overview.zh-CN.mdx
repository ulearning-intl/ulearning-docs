# 概述

 此为面向教师的快速入门用户指南。 本指南为ULearning教师用户提供使用ULearning LMS所需的知识和技能。 通过遵循本指南中的步骤，您可以快速掌握教师最常见的LMS任务，开启您的教师旅程。
 
 本指南围绕您将执行的必要任务展开，并以ULearning平台屏幕截图一步步引导您。
 
 ## 使用ULearning教学的快速入门清单 
 
 此清单简要概述了您在ULearning从线下教学过渡到在线教学时需要进行的操作。 虽然没有魔杖可以自动将您的线下课程转换为在线课程，但我们希望通过这些简单的操作步骤帮助您入门。 
 
 熟悉这些基本步骤后，您可以充分发挥您的创造力，探索ULearning的各种可能性。 
 **步骤/任务** 
 
 ### LMS 登录 
 - 首先：平台URL、您的 username和密码 
 
 ### 课程设置 
 - 在课程开始之前 
 - 创建课程 
     - 作为ULearning的新用户，这是您的起点 
 - 创建班级 
     - 当您创建新课程时，平台将为您的课程创建一个默认班级
     - 您可以根据需要创建更多班级 
 - 注册学生 
     - 此操作通常由您学校的行政人员处理 
     - 但有时也需要您来执行 
 - 将老师添加到课程中 
     - 一门课程可以由多名教师共同教授或共同管理
 - 为班级分配老师 
     - 一个班级可以由多个教师以不同身份共同授课： 
         - 课程管理员 
         - 老师 
         - 助教（教师助理） 
 
 ### 课件设置 
 - 通常在课程开始前设置，但您也可以在课程开始后创建并添加更多课件 
 - 制作课件 
     - 可重复使用和可共享的HTML格式数字化学习内容 
     - 使用ULearning专有创作工具创作：U-Author 
     - 分为5层：课件、模块、主题、页面和内容元素（文本、图像、视频、练习题等） 
     - 可以由其他人制作，然后与您共享 
     - 也可以由一个团队共同制作 
 - 预览课件 
     - 在编辑时查看数字化学习内容，浏览视角为学生视角 
 - 发布课件 
     - 向学生开放，在发布之前，只对您可见 
 - 将课件插入课程 
     - 课件本身可以单独使用 
     - 更常见的使用方法是与课程一起使用，可被多个课程不断重复使用
 
 ### 教学/学习活动和资源 
 - 课程开始后将开展各种教学/学习活动和事件，包括课程中添加的额外学习资源。 
 - 公告 
     -创建/发布公告 
         - 您可以使用公告执行各种操作 
         - 您可以按班级发布公告 
 - 资源 
     - 添加资源 
     - 创建微课程 
         -课件以外的额外学习资源 
         -多种不同格式：图像、视频、音频、文档、链接、ULearning页面等。 
         -您可以通过易于使用的ULearning内置微课程创作工具，将PPT/PDF和图像转换为微课程 
 - 作业 
     - 创建作业
     - 发布作业 
     - 作业评分 
         - 三种类型的作业：个人作业、小组作业和随堂测验 
     - 首次创建随堂测验
         - 创建问题并将其添加到问题库 
         - 创建随堂测验 
         - 将问题库中的问题插入随堂测验中 
     - 评分和未评分（0分） 
     - 您可以按班级发布公告 
 - 讨论 
     - 创建/发布讨论 
     - 评分讨论 
     - 回复讨论 
     - 你可以利用讨论创执行各种操作
     - 你可以按班级发布讨论 
     - 您可以根据您设置的评分规则手动加权分数
 
 ### 考核与评价 
 - 课程开始后，甚至课程结束后均可 
 - 考试 
     - 创建考试题 
     - 创建考试 
     - 创建班级考试和课程 
     - 班级考试评分 
         - 两种类型的班级考试 
             - 使用预定的考卷进行班级考试 
             -使用随机生成的考卷进行班级考试，根据您定义的抽题规则直接从问题库中抽选问题。 
     - 考试活动发生在课程开始后，可在课程开始之前准备考题 
     - 你可以按班级和学生区分讨论 
- 成绩
     - 制定评分规则 
     - 查看成绩 
     - 导出成绩 
     - 理论上，您可以在课程结束时设置评分规则。 但实际上，您应该尽早（甚至在课程开始之前）制定评分规则，以便学生清楚地了解学习期望以及如何评估课程学习 
     - 您可以按班级区分评分规则 
 
 ### 数据和分析 
 - 分析 
     - 查看数据 
     - 分析数据 
 - 请随着课程的进展，持续关注课程数据 
 - 寻找趋势并找出问题 
 - 可进行更改
