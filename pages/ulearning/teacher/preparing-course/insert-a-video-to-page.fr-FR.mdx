# Insertion d'une vidéo dans la page 
 
 ### Étape 1 
 Cliquez sur l'icône Vidéos dans la barre d'élément de contenu de droite. 
 ![]( /img/ulearning/teacher/prepare22.png) 
 
 ### Étape 2 
 Cliquez sur le bouton Téléchargement local. 
 ![]( /img/ulearning/teacher/prepare23.png) 
 
 ### Étape 3 
 Sélectionnez la vidéo que vous souhaitez télécharger et cliquez sur le bouton Ouvrir. 
 ![]( /img/ulearning/teacher/prepare24.png) 

 ### Étape 4 
 - Dans la fenêtre contextuelle Sélectionner une vidéo, cliquez sur la vidéo téléchargée pour la sélectionner. 
     - Remarques : Selon la taille de la vidéo, cela peut prendre un certain temps pour que la vidéo s'affiche ici. 
 - Cliquez sur le bouton Confirmer pour insérer la vidéo dans votre page de contenu. 
 
 ![]( /img/ulearning/teacher/prepare25.png) 

 ### Étape 5 
 - Dans la fenêtre contextuelle Ajouter des sous-titres, cliquez sur le bouton Confirmer pour insérer la vidéo dans votre page de contenu. 
     - Remarques : Vous pouvez également importer des sous-titres pour cette vidéo ici ou vous pouvez le faire après l'insertion de la vidéo. 
 
 ![]( /img/ulearning/teacher/prepare26.png) 
 
 - La vidéo insérée est affichée dans la zone de contenu, et vous pouvez : 
     - Ajoutez une légende ou des notes à la vidéo dans le coin inférieur gauche. 
     - Utilisez les outils de la barre d'outils sous le lecteur vidéo pour éditer/supprimer la vidéo, éditer/ajouter des sous-titres, etc., et même insérer des questions d'exercice dans la vidéo. 
 ![]( /img/ulearning/teacher/prepare27.png)
