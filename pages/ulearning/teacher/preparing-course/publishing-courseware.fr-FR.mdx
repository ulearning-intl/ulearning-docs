# Publication de didacticiels 

 ### Étape 1 
 - À partir de l'écran de l'éditeur de didacticiels, cliquez sur le lien **Retour à la page d'accueil des didacticiels**. 
 ![]( /Img/ulearning/teacher/prepare35.png) 
 
 ### Étape 2 
 - À partir de l'écran de votre page d'accueil de Courseware, localisez le module que vous souhaitez publier. 
 - Cliquez sur le bouton **Publier** à côté du module. 
 - Notes : Avant de publier le didacticiel, vous pouvez voir que les modules sont tous marqués comme "Non publié" juste après le titre du module. 
 ![]( /Img/ulearning/teacher/prepare36.png) 
 
 ### Étape 3 
 - Dans la fenêtre contextuelle Alerte qui apparaît, cliquez sur le bouton **Skip**. 
 - Notes : Vous n'avez pas besoin de remplir les informations sur le didacticiel si vous avez seulement l'intention d'utiliser le didacticiel à l'intérieur du cours que vous enseignez. 
 ![]( /Img/ulearning/teacher/prepare37.png) 
 
 ### Étape 4 
 - Sur la fenêtre contextuelle Publier : 
 - Sélectionnez le lieu de publication dans la fonction **Où publier ? ** liste déroulante. Par défaut, il est publié sur le portail LMS ULearning de votre école. 
 - Cliquez sur le bouton **Publier**. 
 
 ![]( /Img/ulearning/teacher/prepare38.png) 
 
 ### Étape 5 
 - La page d'accueil des didacticiels est partiellement actualisée. Vous remarquerez que le bouton **Publier** est remplacé par deux nouveaux boutons : **Mettre à jour** et **Dépublier**. 
 ![]( /Img/ulearning/teacher/prepare39.png) 

 ### Étape 6 
 - Pour actualiser complètement la page d'accueil du didacticiel, cliquez sur l'icône **Actualiser** du navigateur devant la barre d'adresse du navigateur. Vous remarquerez que la marque d'état "Non publié" est supprimée pour chaque module. 
 ![]( /Img/ulearning/teacher/prepare40.png)
