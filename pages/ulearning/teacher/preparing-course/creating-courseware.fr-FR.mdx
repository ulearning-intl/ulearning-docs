# Création de matériel de cours 
 
 ## Action par étapes 
 1. Cliquez sur l'onglet **Courseware**. 
 
 ![]( /img/ulearning/teacher/prepare04.png) 
 
 2. Cliquez sur le lien **Modifier/Créer un cours**. 
 
 ![]( /img/ulearning/teacher/prepare05.png) 
 
 3. À partir de l'écran de votre liste du matériel de cours  , cliquez sur le bouton **Créer un matériel de cours**. 
 ![]( /img/ulearning/teacher/prepare06.png) 
 
 4. - Entrez le nom du matériel de cours. 
    - Ensuite, cliquez sur le bouton **Créer du matériel de cours**. 
 
 ![]( /img/ulearning/teacher/prepare07.png) 
 
 5. - Pour créer rapidement le plan de contenu, il suffit d'utiliser les titres du module, du sujet et de la page pour remplacer les espaces réservés et garder les panneaux "#" intacts.

![](/img/ulearning/teacher/prepare08.png)

   Voici un exemple : 

   Espaces réservés au modèle | Exemple de titre de module, de sujet et de page réel
   --- | ---
   #New Module | #Lecture 1
   ##New Topic | ##Topic 1: Karel Program
   ###Page 1 | ###An Algorithm vs Program
   ###Page 2 | ###Syntax of a Karel Program
   #New Module | #Lecture 2
   ##New Topic | ##Topic 1: Karel and Java
   ###Page 1 | ###Common Errors
   ###Page 2 | ###Decomposition

6. Cliquez sur le bouton **Créer**. 
 
 ![]( /img/ulearning/teacher/prepare09.png) 
 
 7. Sur l'écran de l'éditeur de matériel de cours  : 
    - La zone de gauche est le volet Structure, où vous pouvez ajouter/supprimer un module, un sujet ou une page, et les réorganiser.
    - La grande zone centrale est la zone de contenu, où vous pouvez créer du contenu riche en texte.
    - La zone de droite est la barre d’outils des éléments de contenu, où vous pouvez sélectionner un élément de contenu et l’insérer dans la zone de contenu.
 
 ![]( /img/ulearning/teacher/prepare10.png) 
 
 Dans le volet Structure, vous pouvez : 
 - Cliquez sur **Nouveau module**, **Nouveau sujet** ou **Nouvelle page** pour ajouter un module/sujet/page. 
 - Ou cliquez sur le menu déroulant d'un élément, puis sélectionnez le bouton **Ajouter un sujet** ou **Ajouter une page** pour un élément correspondant. 
 - Dans le menu déroulant, cliquez sur le bouton **Renommer** pour modifier le module, le sujet ou le titre de la page. 
 
 ![]( /img/ulearning/teacher/prepare11.png) 
 
 - Faites glisser un élément pour réorganiser la structure du contenu. 
 
 ![]( /img/ulearning/teacher/prepare12.png) 
 
 Dans la barre d'outils de l'élément de contenu, vous pouvez sélectionner et insérer 5 objets de contenu différents dans la zone de contenu du milieu. 
 
 ![]( /img/ulearning/teacher/prepare13.png)
