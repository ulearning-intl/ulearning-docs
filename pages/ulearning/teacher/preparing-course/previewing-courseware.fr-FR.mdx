# Aperçu du matériel de cours 

 Dans votre fenêtre d'édition de matériel de cours, cliquez sur le bouton Aperçu pour afficher votre matériel de cours dans la modification en tant que rôle d'étudiant. 
 
 
![](/img/ulearning/teacher/prepare33.png)
 
 Il ouvre la page de contenu dans une nouvelle fenêtre. Vous pouvez utiliser la page précédente et la page suivante pour naviguer dans l'ensemble du matériel de cours. 
 
![](/img/ulearning/teacher/prepare34.png)
