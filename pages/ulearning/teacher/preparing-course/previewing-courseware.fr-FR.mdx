# Aperçu du matériel de cours 

 Dans votre fenêtre d'édition de didacticiel, cliquez sur le bouton Aperçu pour afficher votre didacticiel dans l'édition en tant que rôle d'étudiant. 
 
 
![](/img/ulearning/teacher/prepare33.png)
 
 Il ouvre la page de contenu dans une nouvelle fenêtre. Vous pouvez utiliser la page précédente et la page suivante pour naviguer dans l'ensemble du didacticiel. 
 
![](/img/ulearning/teacher/prepare34.png)
