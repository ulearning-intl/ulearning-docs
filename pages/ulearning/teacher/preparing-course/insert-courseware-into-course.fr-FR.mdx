# Insertion du matériel de cours dans le cours 

 ### Étape 1 

 Après avoir créé, enregistré et publié votre didacticiel, vous pouvez fermer la page du didacticiel. Vous êtes maintenant de retour sur votre page de cours dans l'onglet Courseware. 
 Cliquez sur le lien Importer mon didacticiel. 

![](/img/ulearning/teacher/prepare28.png)

 ### Étape 2 
 - Cliquez sur la case à cocher devant le didacticiel que vous souhaitez insérer. 
 - Cliquez sur le bouton Importer en bas. 

 ![]( /img/ulearning/teacher/prepare29.png) 

 ### Étape 3 

 Dans la fenêtre contextuelle Modifier le didacticiel, cliquez sur le bouton Confirmer. 
 ![]( /img/ulearning/teacher/prepare30.png) 

 ### Étape 4 

 Vous remarquerez que le cours est inséré et que tous les modules du didacticiel sont répertoriés à l'écran. 
 
 ![]( /img/ulearning/teacher/prepare31.png) 

 **Notes de l'utilisateur** 

 Vous n'êtes pas limité à l'insertion d'un seul didacticiel ou à l'insertion de votre propre didacticiel uniquement. En fonction des besoins du cours/de l'apprentissage, vous pouvez ajouter plusieurs didacticiels provenant de différentes sources.
