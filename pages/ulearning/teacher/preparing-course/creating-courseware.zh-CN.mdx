# 创建课程软件

## 步骤操作
1. 点击**软件**选项卡。

![](/img/ulearning/teacher/prepare04.png)

2. 点击 **编辑/创建课程** 链接。

![](/img/ulearning/teacher/prepare05.png)

3. 在您的课件列表屏幕上，单击**创建课件**按钮。
![](/img/ulearning/teacher/prepare06.png)

4. - 输入课程软件名称。
   - 然后，单击**创建课程软件**按钮。

![](/img/ulearning/teacher/prepare07.png)

5. - 要快速创建内容大纲，只需使用实际模块、主题和页面标题来替换占位符，并保持“#”符号不变。

![](/img/ulearning/teacher/prepare08.png)

   这里有一个例子：
   
   模板占位符 | 实际模块、主题和页面标题的示例
   --- | ---
   #New Module | #Lecture 1
   ##New Topic | ##Topic 1: Karel Program
   ###Page 1 | ###An Algorithm vs Program
   ###Page 2 | ###Syntax of a Karel Program
   #New Module | #Lecture 2
   ##New Topic | ##Topic 1: Karel and Java
   ###Page 1 | ###Common Errors
   ###Page 2 | ###Decomposition

6. 点击 **创建** 按钮。 
 
![](/img/ulearning/teacher/prepare09.png)
 
 7. 在课件编辑器屏幕上： 
    - 左侧区域是结构窗格，您可以在其中添加/删除模块/主题/页面并重新排序。 
    - 大中间区域是内容区域，您可以在其中创建富文本内容。 
    - 正确的区域是内容元素工具栏，您可以在其中选择内容元素并将其插入内容区域。 
 
 ![](/img/ulearning/teacher/prepare10.png)
 
 在结构窗格上，您可以： 
 - 单击 **新模块**、**新主题** 或 **新页面** 以添加模块/主题/页面。 
 - 或单击项目下拉菜单，然后为相应项目选择 **添加主题** 或 **添加页面** 按钮。 
 - 从下拉菜单中，单击 **重命名** 按钮以编辑模块、主题或页面标题。 
 
 ![](/img/ulearning/teacher/prepare11.png)
 
 - 拖动一个项目以重新排序内容结构。 

 ![](/img/ulearning/teacher/prepare12.png)
 
 从内容元素工具栏中，您可以选择5个不同的内容对象并将其插入中间内容区域。 

 ![](/img/ulearning/teacher/prepare13.png)
