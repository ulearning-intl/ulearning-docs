# 发布课件 

 ### 第 1 步 
 - 在课件编辑器页面上，单击 **返回课件主页** 链接。 
 ![](/img/ulearning/teacher/prepare35.png)
 
 ### 第 2 步 
 - 从课件主页页面中，找到要发布的模块。 
 - 单击模块旁边的 **发布** 按钮。 
 - 注意：在发布课件之前，您可以看到模块标题后面的所有模块都被标记为“未发布”状态。 
 ![](/img/ulearning/teacher/prepare36.png) 
 
 ### 第3步 
 - 在警报弹出窗口中，单击 **跳过** 按钮。 
 - 注意：如果您只想在教授的课程中使用课件，则无需填写课件信息。 
 ![](/img/ulearning/teacher/prepare37.png) 

 ### 第4步 
 - 在发布弹出窗口上： 
     - 从 **在哪里发布？** 下拉菜单中选择发布位置。 默认情况下，将发布到您学校的ULearning LMS门户网站。 
     - 单击 **发布** 按钮。 

 ![](/img/ulearning/teacher/prepare38.png)
 
 ### 第5步 
 - 课件主页已部分刷新。 您将注意到 **发布** 按钮被两个新按钮替换：**更新** 和 **取消发布** 。 
 ![](/img/ulearning/teacher/prepare39.png)
 
 ### 第6步 
 - 要完全刷新课件主页，请单击浏览器地址栏前的浏览器 **刷新** 图标。 您将注意到每个模块的 **未发布** 状态标记已被移除。 
 ![](/img/ulearning/teacher/prepare40.png)
