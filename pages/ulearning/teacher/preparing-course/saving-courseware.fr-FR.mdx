# Enregistrer les didacticiels 
 
 Le didacticiel en cours d'édition est automatiquement enregistré toutes les 10 secondes ou chaque fois que vous quittez la page. 
 
 Vous pouvez également cliquer sur le bouton Enregistrer pour l'enregistrer manuellement à tout moment.
