# Enregistrer matériel de cours 
 
 Le matériel de cours en cours d'édition est automatiquement enregistré toutes les 10 secondes ou chaque fois que vous quittez la page. 
 
 Vous pouvez également cliquer sur le bouton Enregistrer pour l'enregistrer manuellement à tout moment.
