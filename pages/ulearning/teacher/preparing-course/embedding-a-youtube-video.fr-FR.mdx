# Intégrer une vidéo Youtube 

 ### Étape 1 
 - Cliquez sur le bouton **Ajouter un plugin**. 
 
 ![]( /img/ulearning/teacher/prepare15.png) 
 
 - Il s'ouvre dans la fenêtre **Insérer le plugin**. 
 
 ![]( /img/ulearning/teacher/prepare16.png) 
 
 ### Étape 2 
 - Allez sur le site vidéo YouTube et cliquez sur le lien **PARTAGER** sous le lecteur vidéo. 
 ![]( /img/ulearning/teacher/prepare17.png) 
 
 ### Étape 3 
 - Cliquez sur l'icône **Intégrer** (la première icône dans la fenêtre contextuelle Partager). 
 
 ![]( /img/ulearning/teacher/prepare18.png) 
 
 ### Étape 4 
 - Cliquez sur le lien **Copier** dans le coin inférieur droit de la fenêtre contextuelle **Intégrer la vidéo**. 
 
 ![]( /img/ulearning/teacher/prepare19.png) 
 
 ### Étape 5 
 - Revenez à l'écran de votre éditeur Courseware et collez le code copié dans la fenêtre contextuelle **Insérer le plugin**. 
 
 ![]( /img/ulearning/teacher/prepare20.png) 
 
 - Cliquez sur le bouton **Ok**. La vidéo YouTube est intégrée à votre page de contenu. 
 
 ![]( /img/ulearning/teacher/prepare21.png)
