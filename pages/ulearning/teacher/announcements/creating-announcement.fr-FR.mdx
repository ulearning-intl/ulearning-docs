# Création d'une annonce 

### Étape 1 
 Sur l'écran d'accueil du cours, cliquez sur le bouton Créer une annonce.
![](/img/ulearning/teacher/announcement01.png)

### Étape 2 
 À partir de la fenêtre contextuelle Créer une annonce : 
 - Entrez le titre de l'annonce. 
 - Entrez le contenu de l'annonce. 
     - Remarque : il s'agit d'un champ d'éditeur de texte enrichi. Vous pouvez entrer des textes, des images, des tableaux, des liens, etc. Vous pouvez également télécharger des pièces jointes ici. 
 - Cliquez sur le lien Sélectionner une classe.

![](/img/ulearning/teacher/announcement02.png)

 ### Étape 3 
 
 Dans la fenêtre contextuelle Sélectionner une classe : 
 - Sélectionnez la ou les classes pour qui vous souhaitez publier l'annonce. 
 - Cliquez sur le bouton Confirmer.


![](/img/ulearning/teacher/announcement03.png)

### Étape 4 
 Dans la fenêtre contextuelle Créer une annonce, cliquez sur le bouton Confirmer.

![](/img/ulearning/teacher/announcement04.png)

### Étape 5 
 Vous verrez que votre annonce nouvellement créée est publiée sur l'écran d'annonce du cours.

![](/img/ulearning/teacher/announcement05.png)
