# Établissement des règles de notation 
 
 ### Étape 1 
 Sur la page d'accueil du cours, cliquez sur l'onglet "Notes". 
 ![]( /img/ulearning/teacher/grade01.png) 
 
 ### Étape 2 
 Sur la page "Notes", cliquez sur le bouton "Définir les règles de notation". 
 ![]( /img/ulearning/teacher/grade02.png) 
 
 ### Étape 3 
 Sur l'écran "Définir les règles de Notation", configurez les éléments mis en évidence suivants : 
 - Entrez la note final possible en points pour ce cours, par exemple, 100 points. 
 - Entrez la date limite à laquelle l'enseignant doit terminer la notation pour ce cours, par exemple, le 18/12/2020 (le dernier jour du semestre d'automne). 
 - Sélectionnez quand afficher les notes : 
     - Oui 
     - Non 
     - Uniquement après la date limite 
 - Attribuez un poids à chaque élément noté. Si vous attribuez 0 à un élément, cela signifie qu’il ne sera pas pris en compte dans la note finale de l’élève.
Par défaut, le système inclut 8 éléments notés :
     - Apprentissage des materiel de cours - Moyenne de modules/sujets 
     - Temps passé sur les materiel de cours – 100 ou 0 point (basé sur le temps passé à apprendre)
     - Progression des materiel de cours - Comptée en pourcentage 
     - Devoirs - Points réels selon les barèmes définis par l’enseignant
     - Discussions - Notées sur un score cumulé avec un plafond
     - Présence – Notée selon les pénalités appliquées
     - Participation - Noté sur un score cumulé avec un plafond 
     - Examens - Points réels selon les barèmes définis par l’enseignant
 ![]( /img/ulearning/teacher/grade03.png) 
 
 **Remarques :** 
 - Vous pouvez ajouter vos propres éléments personnalisés à la note finale en cliquant sur le lien + Ajouter un élément de note.
 - Assurez-vous que la somme des poids attribués est bien de 100 %.
 
 ### Étape 4 
 Pour la notation d'un groupe des materiel de cours sur la page "Définir les règles de notation" : 
 - Entrez les détails ou sélectionnez les options comme vous le souhaitez. 
 - Pour inclure/exclure un contenu spécifique du matériel de cours, cliquez sur le lien "Paramètres". 
 
 ![]( /img/ulearning/teacher/grade04.png) 
 
 - Dans la fenêtre contextuelle de la portée du contenu, sélectionnez ou désélectionnez les modules/sujets à inclure/exclure. 
 - Cliquez sur le bouton "Confirmer". 
 ![]( /img/ulearning/teacher/grade05.png) 
 
 ### Étape 5 
 Pour inclure/exclure des devoirs pour la note finale du cours : 
 - Cliquez sur le lien de réglage pour le "devoirs". 
 ![]( /img/ulearning/teacher/grade06.png) 
 
 - Dans la fenêtre contextuelle "Devoirs", incluez ou excluez les devoirs comme vous le souhaitez. 
 - Cliquez sur le bouton "Confirmer". 
 ![]( /img/ulearning/teacher/grade07.png) 
 
 ### Étape 6 
 Entrez les détails souhaités pour les éléments notés suivants :
 - Discussions : Entrez une valeur, par exemple 50. Si le total des points obtenus par un étudiant lors des discussions atteint ou dépasse 50, il recevra l’intégralité des points attribués à cet élément (par exemple, 10 points) dans sa note finale. Si l’étudiant a obtenu 25 points (soit 50 % des 50 points attendus), il recevra 5 points (soit 50 % des 10 points) dans sa note finale.
 - Présence : Attribuez un poids ou une contribution en points à la note finale du cours. Définissez la réduction de points pour chaque type de pénalité. Par exemple, si un étudiant a 1 absence et un départ anticipé, il contribuera 3 points de présence à la note finale. S’il a 2 absences et 3 retards, la contribution sera de 0 point.
 - Participation : La règle pour la participation est similaire à celle des discussions.
 ![]( /img/ulearning/teacher/grade08.png) 
 
 ### Étape 7 
 Pour inclure/exclure les examens dans la note finale du cours : 
 - Cliquez sur le lien "Paramètres" de l'élément Examens. 
 ![]( /img/ulearning/teacher/grade09.png) 
 
 - Dans la fenêtre contextuelle Examens, cochez ou décochez les examens selon vos préférences.
 - Cliquez sur le bouton Confirmer. 
 ![]( /img/ulearning/teacher/grade10.png) 
 
 ### Étape 8 
 Après avoir configuré tous les éléments, cliquez sur le bouton Enregistrer en bas de la page Définir les règles de notation.
 ![]( /img/ulearning/teacher/grade11.png) 
 
 ### Étape 9 
 Dans la fenêtre contextuelle "Enregistré avec succès", cliquez sur le bouton OK.
 ![]( /img/ulearning/teacher/grade12.png) 
 
 ### Étape 10 
 Sur l'écran Notes, les résultats des étudiants sont générés en fonction des règles de notation que vous venez de définir.
