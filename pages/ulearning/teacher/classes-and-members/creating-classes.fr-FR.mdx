# Création de classes

### Étape 1 
 Cliquez sur l'onglet Membres dans le volet à gauche. 
 ![]( /img/ulearning/teacher/class01.png) 
 
 ### Étape 2 
 
 Cliquez sur le bouton Créer une classe. 
 - Remarques : Lorsque vous créez un cours, le système crée également une classe nommée "Classe par défaut" pour vous. Ce guide vous montre comment créer une classe supplémentaire, car il est très probable que vous enseignerez plusieurs classes pour un même cours au cours d’un semestre.  
 ![]( /img/ulearning/teacher/class02.png) 
 
 ### Étape 3 
 
 À partir de la fenêtre contextuelle Créer une classe : 
 - Entrez le nom de la classe, par exemple, Lundi_9am. 
 - Remarques : Vous pouvez créer plusieurs classes en un seul plan. Assurez-vous simplement d'entrer le nom de chaque classe par ligne. 
 - Cochez ou décochez la case devant "L'étudiant a besoin d'approbation pour rejoindre la classe". 
 - Cliquez sur le bouton Confirmer en bas. 
 
 ![]( /img/ulearning/teacher/class03.png) 
 ### Étape 4 
 
 Vous remarquerez que la classe nouvellement créée a été ajoutée à la liste des cours. 
 
 ![]( /img/ulearning/teacher/class04.png)
