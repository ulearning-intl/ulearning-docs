# Adding Teacher to Course

### Adding Teacher to Course

### Step 1 
From the Class List screen, click the Teachers tab at the top of the screen.
![](/img/ulearning/teacher/class09.png)

### Step 2 
It will take you to the Teachers screen. Click the Add Teacher button.
![](/img/ulearning/teacher/class10.png)

### Step 3
From the Add Teacher popup window:
- Select all the teachers you want to add to the course.
- Note: As you select the teachers you want to add, you'll see they are added to the Selected Teachers column on the right. If you want to remove a teacher, click the X icon right after the teacher's name.
- Click the Save button.

![](/img/ulearning/teacher/class11.png)

### Step 4
In the Teacher screen, the teachers you just added will be listed.
![](/img/ulearning/teacher/class12.png)

**User Notes**
- You can assign a class to a teacher using the Assign class action in the screen above. For details, please refer to the complete version of ULearning Teacher Guide.
