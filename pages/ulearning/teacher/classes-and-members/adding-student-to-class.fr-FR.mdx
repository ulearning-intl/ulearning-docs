# Ajout d'étudiants à la classe


### Étape 1 
 
 À partir de l'écran Liste des classes : 
 - Cliquez sur le menu déroulant Actions à la fin de la ligne de classe. 
 - Sélectionnez l'action Ajouter un étudiant et cliquez sur. 
 ![]( /Img/ulearning/teacher/class05.png) 
 
 ### Étape 2 
 
 À partir de la fenêtre contextuelle Sélectionner les étudiants du pool d'étudiants : 
 - Sélectionnez tous les élèves que vous souhaitez ajouter à la classe. 
 - Cliquez sur le bouton Confirmer. 
 ![]( /Img/ulearning/teacher/class06.png)
