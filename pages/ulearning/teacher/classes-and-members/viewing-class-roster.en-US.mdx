# Viewing Class Roster

### Step 1

From the Class List screen:
- Click the Actions dropdown menu at the end of the class row.
- Select the Roster action and click.
![](/img/ulearning/teacher/class07.png)

### Step 2 
It will take you to the Student Roster screen where the students you added to the class will be listed.
![](/img/ulearning/teacher/class08.png)
