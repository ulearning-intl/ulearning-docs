# Adding Students to Class


### Step 1 

From the Class List screen:
- Click the Actions dropdown menu at the end of the class row.
- Select the Add Student action and click.
![](/img/ulearning/teacher/class05.png)

### Step 2 

From the Select Students from Student Pool popup window:
- Select all the students you want to add to the class.
- Click the Confirm button.
![](/img/ulearning/teacher/class06.png)
