# Création de discussions

### Étape 1 
 Sur la page d'accueil du cours, cliquez sur l'onglet Discussions. 
 ![]( /img/ulearning/teacher/discussion01.png) 
 
 ### Étape 2 
 Cliquez sur le bouton + Discussion. 
 ![]( /img/ulearning/teacher/discussion02.png) 
 
 ### Étape 3 
 - Entrez le titre de la discussion. 
 - Entrez le sujet/les instructions de discussion. 
 - Entrez les détails ou sélectionnez les options pour tous les autres éléments comme vous le souhaitez. 
 - Cliquez sur le bouton Sélectionner une classe. 
 ![]( /img/ulearning/teacher/discussion03.png) 
 
 ### Étape 4 
 Sur la page Sélectionner une classe : 
 - Sélectionnez la ou les classes à laquelle vous souhaitez attribuer cette discussion. 
 - Cliquez sur le bouton Confirmer. 
 ![]( /img/ulearning/teacher/discussion04.png) 
 
 ### Étape 5 
 Cliquez sur le bouton Soumettre en bas de la page Créer une discussion. 
 ![]( /img/ulearning/teacher/discussion05.png) 
 
 ### Étape 6 
 La discussion est maintenant ajoutée à l'écran de discussion. 
 ![]( /img/ulearning/teacher/discussion06.png)
