# Affichage de l'analyse du cours

### Étape 1 
 Sur la page d'accueil du cours, cliquez sur l'onglet Analytique.
![](/img/ulearning/teacher/analytic01.png)

 ### Étape 2 
 La première partie de l'analyse des données du cours est un aperçu de haut niveau du cours dans 8 catégories : 
 - Membres : Fournit des informations sur les membres du cours. 
 - Annonces : Affiche les données relatives aux annonces de cours. 
 - Courseware : Affiche les données liées au contenu du courseware. 
 - Ressources : fournit des informations sur les ressources du cours. 
 - Affectations : affiche les données relatives aux affectations de cours. 
 - Discussions : Affiche les données relatives aux discussions sur les cours. 
 - Salle de classe : Fournit des informations sur les activités en classe. 
 - Examens : Affiche les données relatives aux examens de cours.
![](/img/ulearning/teacher/analytic02.png)

 ### Étape 3 
 La deuxième partie de l'analyse des données du cours est le top 5 du classement de : 
 - Vidéos : Affiche les 5 meilleures vidéos en fonction de l'engagement ou d'autres mesures. 
 - Ressources : Affiche les 5 principales ressources auxquelles les étudiants ont accès ou sont utilisées. 
 - Discussions : Affiche les 5 principales discussions en fonction de l'engagement ou d'autres mesures.
![](/img/ulearning/teacher/analytic03.png)

 ### Étape 4 
 La troisième partie de l'analyse des données de cours est les données d'apprentissage des didacticiels module par module, y compris : 
 - Taux moyen d'achèvement : indique le taux moyen auquel les étudiants terminent chaque module. 
 - Temps moyen d'apprentissage : fournit des informations sur le temps moyen passé par les étudiants sur chaque module.
![](/img/ulearning/teacher/analytic04.png)

 ### Étape 5 
 La quatrième partie de l'analyse des données du cours est les données relatives aux activités liées à la salle de classe dans 4 catégories : 
 - Appel nominal : fournit des informations sur l'assiduité ou la participation aux activités d'appel nominal. 
 - Sondage : Affiche les données relatives aux sondages effectués en classe. 
 - Réponse rapide (concurrencier pour répondre) : Affiche les données relatives aux activités de réponse rapide où les élèves s'affrontent pour répondre aux questions. 
 - Pick Student (Random Pick) : fournit des informations sur les activités où les étudiants sont sélectionnés au hasard.
![](/img/ulearning/teacher/analytic05.png)

 ### Étape 6 
 La dernière partie de l'analyse des données de cours est les données statistiques des actions liées à l'enseignement effectuées par les enseignants du cours. Les données fournies sont simples et explicites, donnant un aperçu des mesures prises par les enseignants dans le cours.
![](/img/ulearning/teacher/analytic06.png)
