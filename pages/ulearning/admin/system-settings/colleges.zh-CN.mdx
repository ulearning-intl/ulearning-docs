# 学院-建立学术组

超级管理员在ULearning上设置和管理学术组，请使用此功能添加、编辑和删除学术组和子组。

**要点**
- 将学校的学术组/子组与ULearning中的团体对应
- 应当基于你学校的教学学术结构
- 一个学术组最多可以添加3层结构，通常为两层。例如，
     - 第一层 - 学校/学院
     - 第二层 - 部门
     - 第三层 - 项目/计划

## 添加学术组/子组

### 添加一个学术组

![](/img/ulearning/admin/college01.png)
1. 在系统设置屏幕上，单击添加按钮。

![](/img/ulearning/admin/college02.png)

2. 点击输入学术组名称[1]，例如商学院。
3. 点击确认按钮[2]。

![](/img/ulearning/admin/college03.png)
4. 新添加的学术组已列在您的学术组列表的最后。

### Add an Academic Sub-Group

### 添加学术子组 

![](/img/ulearning/admin/college04.png)

 1. 如要将子组添加到学术组中，请单击该学术组名称最右侧的+图标。 
![](/img/ulearning/admin/college05.png)
 
 2. 点按以输入子团体的名称[1]，例如财务。 
 3. 点按确认按钮[2]。

![](/img/ulearning/admin/college06.png)

 4. 新添加的子组已列入。


## 编辑学术组/子组

![](/img/ulearning/admin/college07.png)

1. 如需编辑组或子组，请单击该学术组名称最右侧的编辑图标。

![](/img/ulearning/admin/college08.png)

2. 根据需要进行任何更改[1]，然后单击确认按钮保存[2]。

![](/img/ulearning/admin/college09.png)
3. 所做的更改已被保存。


## 删除学术组/子组

![](/img/ulearning/admin/college10.png)
1. 要删除组或子组，请单击该学术组名称最右侧的删除（X）图标。

![](/img/ulearning/admin/college11.png)
2. 单击弹出窗口上的确定按钮。

![](/img/ulearning/admin/college12.png)
3. 学术组或子组已被删除。
