# Gestion des comptes du rôle administrateur

Cette fonction vous permet d'ajouter, de modifier et de supprimer les comptes de rôle d'administrateur. 
 
- Seul le Super Admin peut ajouter un compte administrateur 
 
![]( /img/ulearning/admin/admin01.png) 

 ## Ajouter un compte administrateur 
 1. Sur l'écran Paramètres du système, cliquez sur l'onglet Administrateurs [1]. 
 2. Pour ajouter un nouveau compte de rôle d'administrateur, cliquez sur le bouton Ajouter un administrateur [2]. 
 3. Le rôle nouvellement ajouté sera ajouté au tableau. 
 ![]( /img/ulearning/admin/admin02.png) 

 ## Modifier le compte administrateur 
 Vous pouvez modifier un compte de rôle d'administrateur en cliquant sur le boutton Modifier [3].

## Supprimer le compte administrateur 
Pour supprimer un compte de rôle d'administrateur, cliquez sur le boutton Supprimer [4].

<Callout type="info" emoji="ℹ️">
  Remarque : le compte de rôle d'administrateur système (Super Admin) est créé/ajouté par ULearning uniquement et un seul compte de rôle d'administrateur système (Super Admin) peut être créé.
</Callout>
