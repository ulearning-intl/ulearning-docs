# Collèges - Mise en place de groupes académiques 

Pour que les super-administrateurs configurent et gèrent des groupes académiques sur ULearning, utilisez cette fonction pour ajouter, modifier et supprimer des groupes et sous-groupes académiques. 
 
**Points clés :** 
 - Mapper les groupes/sous-groupes académiques de l'école à ULearning 
 - Il devrait être basé sur la structure académique de votre école pour l'enseignement 
 - Peut ajouter jusqu'à 3 couches d'un groupe académique, généralement deux couches. Pour l'examen, 
     - Première couche - École/Collèges 
     - Deuxième couche - Départements 
     - Troisième couche - Programmes/Plans


## Ajouter un groupe/sous-groupe académique 

### Ajouter un groupe académique

![](/img/ulearning/admin/college01.png)
1. Sur l'écran Paramètres système, cliquez sur le bouton Ajouter.

![](/img/ulearning/admin/college02.png)

2. Cliquez pour entrer le nom du groupe académique [1], par exemple, Faculté de commerce. 
3. Cliquez sur le bouton Confirmer [2].

![](/img/ulearning/admin/college03.png)
4. Le groupe académique nouvellement ajouté est répertorié comme le dernier dans votre liste de groupe académique.

### Ajouter un sous-groupe académique

![](/img/ulearning/admin/college04.png)

1. Pour ajouter un sous-groupe au groupe académique, cliquez sur l'icône + à l'extrême droite du nom de ce groupe académique.

![](/img/ulearning/admin/college05.png)

2. Cliquez pour entrer le nom du sous-groupe [1], par exemple, Finance. 
3. Cliquez sur le bouton Confirmer [2].

![](/img/ulearning/admin/college06.png)

4. Le sous-groupe nouvellement ajouté est répertorié.

## Modifier un groupe/sous-groupe académique

![](/img/ulearning/admin/college07.png)

1. Pour modifier un groupe ou un sous-groupe, cliquez sur l'icône Modifier à l'extrême droite du nom de ce groupe académique.

![](/img/ulearning/admin/college08.png)

2. Effectez toute modification comme vous le souhaitez [1], puis cliquez sur le bouton Confirmer pour enregistrer [2].

![](/img/ulearning/admin/college09.png)
3. La modification apportée est enregistrée.


## Supprimer un groupe/sous-groupe académique

![](/img/ulearning/admin/college10.png)
1. Pour supprimer un groupe ou un sous-groupe, cliquez sur l'icône Supprimer (X) à l'ex-droite du nom de ce groupe académique.

![](/img/ulearning/admin/college11.png)
2. Cliquez sur le bouton OK dans la fenêtre contextuelle.

![](/img/ulearning/admin/college12.png)
3. Le groupe ou le sous-groupe académique est supprimé.
