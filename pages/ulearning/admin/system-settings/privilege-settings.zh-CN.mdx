# 权限设置

使用此功能添加/编辑角色，设置角色权限。

![](/img/ulearning/admin/privilege01.png)

1. 在系统设置页面点击权限设置选项卡[1]。
2. 单击添加角色按钮创建新角色[2]。
3. 新建的角色将添加在表格中。
4. 然后您可以在表格中修改权限设置[3]，或点击铅笔（编辑）图标修改角色名称[3]。
5. 点击删除图标[4]删除角色。
6. 完成所有设置后点击保存改变按钮[5]以保存您所作的改变。

**注意:** 
- 系统管理员（超级管理员）由Ulearning设置，用户不可修改。
- 系统设置、子系统设置以及云会议功能仅对系统管理员（超级管理员）开放。
