# Paramètres de privilège

Utilisez cette fonction pour ajouter/modifier des rôles et personnaliser les privilèges d'un rôle.

![](/img/ulearning/admin/privilege01.png)

1. Sur l'écran Paramètres du système, cliquez sur l'onglet Paramètres des privilèges [1]. 
2. Pour ajouter un nouveau rôle, cliquez sur le bouton Ajouter un rôle [2]. 
3. Le rôle nouvellement ajouté sera ajouté au tableau. 
4. Ensuite, vous pouvez modifier les paramètres de privilèges [3] en un rôle dans le tableau ou cliquer sur l'icône Crayon (Modifier) pour modifier le nom du rôle [3]. 
5. Pour supprimer un rôle, cliquez sur l'icône Supprimer [4]. 
6. Une fois que vous avez terminé, cliquez sur le bouton Enregistrer les modifications [5] pour enregistrer toutes les modifications que vous avez apportées.

**Remarque :** 
- Le rôle d'administrateur système (Super Admin) est créé par ULearning et n'est pas modifiable par les utilisateurs. 
- Les modules Paramètres système, Paramètres sous-système et Cloud Meeting ne sont disponibles que pour le rôle d'administrateur système (Super Admin).
