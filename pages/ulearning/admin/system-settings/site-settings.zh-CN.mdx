# 网站设置


![](/img/ulearning/admin/site01.png)

此功能模块可供超级管理员为学校定制ULearning实例网站。在系统管理页面上[1]，单击网站管理选项[2]。使用此功能，您可以：
- 设置网站名称 [3] – 为网站分配一个有意义的名称。
- 设置网站URL[4] – 以https://yourname.ulearningsa.net 的格式为网站定义URL
- 选择网站语言[5] – 设置网站默认语言并选择要支持的语言。
- 从两个选项[6]中选择网站主页样式，并为所选样式编辑外观和风格以及内容显示[7]。

## 选择/编辑网站主页样式 – 菜单样式

菜单样式主页是用户登录平台之前的门户主页。 如果您想自定义页面的菜单和内容，建议您使用此功能。 
 
### 第 1 步：设置菜单样式
![](/img/ulearning/admin/site02.png)

在网站设置页面 **[1]** 上，单击以选择菜单样式图标 **[2]** ，然后单击编辑按钮 **[3]** 以修改外观和风格以及菜单样式显示的内容。

### 第2步：自定义主页概述

![](/img/ulearning/admin/site03.png)

打开可编辑的菜单样式主页（任何带有 **✎** 或 **📷** 图标的组件都是可编辑的）。 您可以： 
- 自定义您的学校徽标 **[1]**。 
- 自定义顶部导航选项卡 **[2]**。 
- 编辑轮播头图-头图滑动播放样式 **[3]**。 
- 添加/编辑/显示/隐藏/删除页面组件，两种类型的组件： 
     -系统内置-只能显示/隐藏 **[4]** 或可编辑 **[5]** ：如学院/系、公告等。 
     -用户添加 **[6]** **[7]** 由系统定义的固定格式-可以编辑/显示/隐藏/删除：包括3种类型：课程|图像|视频。

### 第3步：自定义您的学校徽标

![](/img/ulearning/admin/site04.png)

在菜单样式编辑页面上，单击徽标编辑图标。

![](/img/ulearning/admin/site05.png)
- 请单击上传徽标按钮，上传学校徽标。
     - 格式：png、bmp、jpg、gif。
     - 大小：不超过2 MB。
     - 建议宽度x高度：300x50像素。

![](/img/ulearning/admin/site06.png)
  - 从电脑中选择学校徽标，然后单击打开按钮上传。

![](/img/ulearning/admin/site07.png)
  - 上传的校徽取代了目前使用的学徽 **[1]**。 
  - 点按确认按钮以完成设置 **[2]**。


### 第4步：自定义顶部导航选项卡

![](/img/ulearning/admin/site08.png)
在菜单样式编辑页面上，单击顶部导航栏的编辑图标。

![](/img/ulearning/admin/site09.png)
- 您将打开编辑模块弹出窗口。
- 显示的菜单项在第一行 **[1]** ，您可以：
     - 单击  - （减号）图标以隐藏菜单项。
         - 需要保留至少2个项目。
     - 单击铅笔（编辑）图标以修改菜单项。
     - 您可以拖动以移动项目以重新排序。
 - 隐藏的菜单项在第二行 **[2]** ，您可以：
     - 单击+（加号）图标以显示菜单项。
     - 单击X（删除）图标以删除菜单项。
 - 要添加菜单按钮，请单击添加菜单项按钮 **[3]** 。


![](/img/ulearning/admin/site10.png)
 您可以添加2种类型的菜单项 **[1]**： 
 - 外部链接-可以添加/编辑/显示/隐藏。 
 - 图文内容-可以添加/编辑/显示/隐藏。 
 - 添加外部链接项： 
     - 从下拉菜单中选择外部链接 **[1]**。 
     - 单击并输入项目名称 **[2]**。 
     - 单击并以http://或https:// **[3]** 格式输入链接URL。 
     - 然后单击确认按钮完成设置 **[4]**。


![](/img/ulearning/admin/site11.png)

 - 要添加图文内容（文本和图像）项： 
     - 从下拉菜单中选择图文内容 **[1]**。 
     - 单击以输入菜单项名称 **[1]**。 
     - 单击以输入内容（格式化文本、图像、链接等） **[2]**。 
     - 然后单击确认按钮完成 **[3]**。


![](/img/ulearning/admin/site12.png)

    - 菜单项已添加。

![](/img/ulearning/admin/site13.png)

第三种类型的菜单项-系统内置菜单项 **[1]** **[2]** ：
 - 您可以通过单击编辑图标 **[1]** 修改菜单项名称，或通过单击-（减号）图标将其从菜单栏中隐藏。
 - 但您无法删除该菜单项 **[2]** 。


### 第5步：编辑轮播头图-头图滑动播放样式

![](/img/ulearning/admin/site14.png)
 - 单击轮播头图编辑按钮。

![](/img/ulearning/admin/site15.png)
 - 单击编辑模块链接。

![](/img/ulearning/admin/site16.png)

您将打开编辑模块弹出窗口。显示的图像在第一行[1]，您可以： 

- 拖动图像以重新排序。 
- 单击-（减号）图标以隐藏图像。 
- 输入图像的URL。 

 隐藏的图像在第二行[2]，您可以： 
- 单击+（加号）图标以显示图像。 
- 单击X（删除）图标以删除图像。 
- 输入图像的URL。 

 如需添加菜单按钮，请单击上传图像按钮[3]。 
 - 支持的图像格式：png、bmp、jpg、jpeg、gif。 
 - 单个图像大小：不超过2 MB。 
 - 推荐宽度x高度：1170x320像素-否则系统将裁剪。 

 注意：当启用多个图像显示时，默认情况下会为启用图像轮播。

### 第6步：添加/编辑/显示/隐藏/删除页面组件

![](/img/ulearning/admin/site17.png)
有两种类型的组件： 
 - 用户[1]添加系统定义的固定格式-可以编辑/显示/隐藏/删除。 
 - 系统内置[2] – 只能显示/隐藏或可编辑，但用户无法删除：如学院/院系、公告等。

![](/img/ulearning/admin/site18.png)
对于用户添加的组件，单击设置图标[1]，您可以： 
 - 通过单击编辑模块链接[2]来编辑组件，然后按照页面说明进行编辑。 
 - 单击隐藏模块链接[3]来隐藏组件。 
 - 通过单击删除模块链接[4]删除组件。

![](/img/ulearning/admin/site19.png)
通过单击向上/向下移动图标[1][2]来重新排序组件。

![](/img/ulearning/admin/site20.png)

对于系统内置组件-可编辑项目，单击设置图标[1]，您可以： 
 - 通过单击编辑模块链接[2]编辑组件，然后按照页面说明进行编辑。 
 - 通过单击隐藏模块链接[3]隐藏组件。 
 
注意：您无法删除此类组件。


![](/img/ulearning/admin/site21.png)
对于系统内置组件-仅显示/隐藏项目，单击设置图标[1]，然后您可以： 
 - 通过单击隐藏/显示模块链接来隐藏/显示组件[2]。 
 
注意：您无法删除此类组件。


![](/img/ulearning/admin/site22.png)

如需添加新组件，请转到页面底部，然后单击添加模块按钮。

![](/img/ulearning/admin/site23.png)
在打开的添加模块弹出窗口中，选择要添加的组件类型： 
 - 课程（课件）类型 [1]。 
 - 图像类型[2]。 
 - 视频类型 [3]。 
 
 样本外观将根据您选择的类型显示在下面[4]。 
 
 然后单击确认按钮[5]打开编辑模块弹出窗口，以添加您选择的组件类型。

### 第7步：添加课程（课件）类型组件

![](/img/ulearning/admin/site24.png)

1. 点击以输入组件名称/标题 [1]。 
2. 选择您想要添加的课程[2]。 
3. 所选课程显示在右栏[3]。 
4. 完成后，单击确认按钮[4]。

![](/img/ulearning/admin/site25.png)

课程组件已添加在页面底部。



### 第8步：添加图像类型组件

![](/img/ulearning/admin/site26.png)

1. 单击以输入组件名称/标题[1]。
2. 单击上传图像按钮[2]。

![](/img/ulearning/admin/site27.png)

上传的图像已列出，您可以单击-（减号）图标将不想要的图片删除[1]。 
 1. 您可以输入图像的名称[2]。 
 2. 您还可以输入图像的URL[3]。 
 3. 单击上传图像按钮[4]在组件中添加更多图像。 
 4. 完成后，单击确认按钮[5]。 
 
 注意： 
 - 推荐图像宽高比：16:9。 
 - 格式：png、jpg、gif、jpeg、bmp。 
 - 单个图像大小：不超过2 MB。


![](/img/ulearning/admin/site28.png)
图像组件已添加在页面底部。


### 第9步：添加视频类型组件

![](/img/ulearning/admin/site29.png)

1. 单击以输入组件名称/标题[1]。 
2. 单击上传视频按钮[2]。

![](/img/ulearning/admin/site30.png)

上传的视频已列出，您可以单击-（减号）图标将不想要的视频删除[1]。 
1. 您可以输入视频的名称[2]。 
2. 单击上传视频按钮[3]在组件中添加更多视频。 
3. 完成后，单击确认按钮[4]。 
 
注：推荐视频宽高比：16:9；格式：mp4；单个视频大小：不超过500 MB。

![](/img/ulearning/admin/site31.png)
视频组件已添加在页面底部。

## 选择/编辑网站主页样式 – 登录样式 

如果您想使用登录窗口保持主页清晰，建议您使用此功能。

### 第1步：设置登录样式

![](/img/ulearning/admin/site32.png)
1. 在网站设置页面上，单击以选择登录样式图标 **[1]** （选择后网站主页将设置为登录样式）。 
2. 然后，单击编辑按钮 **[2]** 以修改外观和风格以及要显示的登录样式的内容。

### 第2步：登录页面

![](/img/ulearning/admin/site33.png)
此操作将打开可编辑的登录样式主页。 您可以： 
 1. 自定义您的学校徽标和页面背景图像 **[1]**。 
 2. 自定义底部 **[2]** 的常用链接。 
 
### 第3步：自定义您的学校徽标

![](/img/ulearning/admin/site34.png)
在登录样式编辑页面上，单击左上角的徽标/背景编辑图标。

![](/img/ulearning/admin/site35.png)
请点按“上传”按钮，上传校徽。 

 注意： 
 - 格式：png、bmp、jpg、gif。 
 - 尺寸：不超过2 MB。 
 - 建议宽度x高度：120x120像素。


![](/img/ulearning/admin/site36.png)
徽标上传后，单击确定按钮。

![](/img/ulearning/admin/site37.png)
徽标现已更新。

### 第 4 步：自定义页面背景图像

![](/img/ulearning/admin/site38.png)

在登录样式编辑屏幕上，单击左上角的徽标/背景编辑图标。

![](/img/ulearning/admin/site39.png)

1. 单击背景选项卡[1]。 
2. 要上传图像，请单击上传图像按钮[2]。

 注意： 
 - 图像格式：png、bmp、jpg、jpeg、gif。 
 - 大小：不超过5MB。 
 - 宽度x高度：1920x1080像素及以上（较小的图像仍将以原始尺寸显示并放置在页面中央）。 
 - 图像将按照您上传的顺序显示。 
 
 3. 选择页面背景的图像[3]。 
 注意：选择多个图像将启用轮播-所选图像将滑动播放 
 
 4. 完成后，单击确定按钮[4]。

![](/img/ulearning/admin/site40.png)

页面背景现已更新。


### 第5步：编辑常用链接

![](/img/ulearning/admin/site41.png)

- 单击页面底部常用链接的铅笔（编辑）图标。

![](/img/ulearning/admin/site42.png)
此操作将打开编辑链接弹出窗口。 
 - 单击添加按钮添加常用链接。

![](/img/ulearning/admin/site43.png)

1. 单击以输入链接名称 [1]。 
2. 单击以http://或https:// **[2]** 格式输入URL。 
3. 要添加更多链接，请单击 **添加按钮[3]**。 
4. 要移除链接，请点按 **删除图标 [4]**。 

 注意： 
 - 最多添加10个链接 
 - 链接名称不超过20个字符 
 - 可以拖动链接以重新排序

![](/img/ulearning/admin/site44.png)
常用链接已添加。
