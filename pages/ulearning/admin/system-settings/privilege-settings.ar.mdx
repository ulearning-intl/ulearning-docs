# إعدادات الصلاحيات أو الامتيازات

استخدم هذه الوظيفة لإضافة/تحرير الأدوار وتخصيص الصلاحيات لكل دور.

![](/ar/img/ulearning/admin/Privilege_settings.png)

1.	على شاشة إعدادات النظام، انقر فوق علامة التبويب "إعدادات الامتيازات" [1].
2.	لإضافة دور جديد، انقر فوق زر "إضافة دور" [2].
3.	سيتم إضافة الدور الجديد إلى الجدول.
4.	يمكنك بعد ذلك تعديل إعدادات الامتيازات [3] لدور في الجدول أو النقر فوق رمز القلم (تحرير) لتعديل اسم الدور [3].
5.	لحذف دور، انقر فوق رمز الحذف [4].
6.	بعد الانتهاء، انقر فوق زر "حفظ التغييرات" [5] لحفظ جميع التغييرات التي أجريتها.

**ملاحظة:** 
-	يتم إنشاء دور "مدير النظام" (Super Admin) بواسطة ULearning ولا يمكن تعديله من قبل المستخدمين.
-	إعدادات النظام، إعدادات النظام الفرعي، ووحدات الاجتماعات السحابية متاحة فقط لدور "مدير النظام" (Super Admin).
