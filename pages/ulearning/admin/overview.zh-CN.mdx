# ULearning Admin概述

![](/img/ulearning/admin/overview01.png)

ULearning平台有四个角色，即超级管理员、管理员（用户定义）、教师和学生。 本快速入门用户指南适用于超级管理员和管理员（用户定义）的角色。 

本指南的目的是为ULearning管理员用户提供使用ULearning LMS的必要知识和技能。 通过遵循本指南中的步骤，您可以快速开始作为管理员执行最常见的LMS任务的旅程。 

本指南围绕您将执行的必要任务进行组织，一步一步地伴随着ULearning平台屏幕截图。 



## 管理员快速入门清单

![](/img/ulearning/admin/overview02.png)

此清单提供了超级管理员或管理员（用户定义）可以使用ULearning管理模块执行的操作的高级概述。 我们旨在通过这个易于遵循的指南指导您完成初始步骤。

一旦你熟悉了这些基本步骤，你就可以释放你的创造力，并在ULearning中探索各种可能性。 

关于超级管理员或管理员可以做什么的高级介绍： 

 - **系统设置** 
 - **教学管理** 
 - **数据和报告** 
 - **应用程序** 

 ### 系统设置 

 - **学院** 
   - 学术团体管理 
 - **网站设置** 
   - 登录页面的基本设置 
 - **特权设置** 
   - 添加、编辑和删除角色 
 - **管理员** 
   - 添加、编辑和删除管理员 

 ### 教学管理 

 - **课程** 
   - 合并、转让和打开课程；查看和导出课程数据；添加、编辑、删除课程类别 
 - **考试** 
   - 打开考试管理控制台并查看/执行任务 
 - **虚拟课堂** 
   - 开始和播放 
 - **教师** 
   - 添加/导入、编辑、删除、担任教师 
 - **学生** 
   - 添加/导入/更新、编辑、删除、充当学生 
 - **行政舱** 
   - 添加、导入、编辑和删除类；查看名册 
 - **公告** 
   - 创建、查看、编辑和删除 

 ### 数据和报告 

 - **数据快照** 
   - 过滤、查看和导出 
 - **课程分析** 
   - 过滤和查看 
 - **活动分析** 
   - 过滤和查看 
 - **大学比较** 
   - 过滤器 

 ### 应用程序 

 - **LTI应用程序** 
   - 添加 LTI App


要点 

 - **超级管理员（系统管理员角色）：** 
   - 具有执行上述所有任务的权限；每个ULearning实例只有一个固定且不可自定义的超级管理员。 
 - **其他自定义管理员角色：** 
   - 他们的功能取决于超级管理员分配的权限；一个ULearning实例可以有多个管理员。 
 - **系统设置：** 
   - 仅供超级管理员访问；不适用于其他管理员。 

 接下来，我们将提供有关管理员可以做什么的详细说明。 为了说明起见，我们将使用超级管理员帐户来演示各种管理任务/流程的步骤。 超级管理员独有的任务/进程将被显式高亮显示。
