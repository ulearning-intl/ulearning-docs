# Announcement Management

Announcement serve as a means to communicate school-wide information in a rich content format. They can be posted to both students and teachers, When users log in, announcements are visible on their landing page.

Use this function to create and post a school-wide announcement:
- Create an announcement
- View an announcement
- Edit an announcement
- Delete an announcement

![](/img/ulearning/admin/announcement01.png)

## Create Announcement

1. Click the **Announcements** tab **[1]**.
   - Utilize the search feature to find specific announcements **[2]**.
   - To create a new announcement, click the **New Announcements** button **[3]**.

![](/img/ulearning/admin/announcement02.png)

## View Announcement
To view an announcement, click the **View** link **[4]** for that announcement.

## Edit Announcement 
To edit an announcement, click the **Edit** link **[5]** for that announcement.

## Delete Announcement
To delete an announcement, click the **Delete** link **[6]** for that announcement.
