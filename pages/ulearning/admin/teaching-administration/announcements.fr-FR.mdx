# Gestion des annonces 

 L'annonce sert de moyen de communiquer des informations à l'échelle de l'école dans un format de contenu riche. Ils peuvent être affichés à la fois aux étudiants et aux enseignants. Lorsque les utilisateurs se connectent, les annonces sont visibles sur leur page de destination. 
 
 Utilisez cette fonction pour créer et publier une annonce à l'échelle de l'école : 
 - Créer une annonce 
 - Voir une annonce 
 - Modifier une annonce 
 - Supprimer une annonce 

 ![]( /img/ulearning/admin/announcement01.png) 

 ## Créer une annonce 

 1. Cliquez sur l'onglet **Annoncements** **[1]**. 
   - Utilisez la fonction de recherche pour trouver des annonces spécifiques **[2]**. 
   - Pour créer une nouvelle annonce, cliquez sur le bouton **Nouvelles annonces** **[3]**. 
 
 ![]( /img/ulearning/admin/announcement02.png) 

 ## Voir l'annonce 
 Pour afficher une annonce, cliquez sur le lien **Voir** **[4]** pour cette annonce. 
 
 ## Modifier l'annonce 
 Pour modifier une annonce, cliquez sur le lien **Modifier** **[5]** pour cette annonce. 
 
 ## Supprimer l'annonce 
 Pour supprimer une annonce, cliquez sur le lien **Supprimer** **[6]** pour cette annonce.

<Callout type="info" emoji="ℹ️">
  Announcements are in a rich content format and When users log in, announcements are visible on their landing page.
</Callout>
