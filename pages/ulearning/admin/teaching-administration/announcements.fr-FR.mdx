# Gestion des annonces 

 Les annonces servent de moyen pour communiquer des informations à l’échelle de l’école dans un format de contenu enrichi. Elles peuvent être affichées aux étudiants comme aux enseignants. Lorsque les utilisateurs se connectent, les annonces sont visibles sur leur page d’accueil.
 
 Utilisez cette fonction pour créer et publier une annonce à l’échelle de l’école : 
 - Créer une annonce 
 - Voir une annonce 
 - Modifier une annonce 
 - Supprimer une annonce 

 ![]( /img/ulearning/admin/announcement01.png) 

 ## Créer une annonce 

 1. Cliquez sur l'onglet **Annonces** **[1]**. 
   - Utilisez la fonction de recherche pour trouver des annonces spécifiques **[2]**. 
   - Pour créer une nouvelle annonce, cliquez sur le bouton **Nouvelle annonce** **[3]**. 
 
 ![]( /img/ulearning/admin/announcement02.png) 

 ## Voir une annonce 
 Pour afficher une annonce, cliquez sur le lien **Voir** **[4]** de cette annonce. 
 
 ## Modifier une annonce 
 Pour modifier une annonce, cliquez sur le lien **Modifier** **[5]** de cette annonce. 
 
 ## Supprimer une annonce
 Pour supprimer une annonce, cliquez sur le lien **Supprimer** **[6]** de cette annonce.

<Callout type="info" emoji="ℹ️">
  Les annonces sont présentées dans un format riche en contenu. Lors de la connexion des utilisateurs, elles s’affichent sur leur page d’accueil.
</Callout>
