# Gestion des classes administratives

Une classe administrative est différente d'une classe de cours. Il s'agit d'une cohorte d'étudiants qui sont entrés dans le même programme au cours du même trimestre d'admission. À des fins administratives, une école peut organiser ce groupe d'élèves en une classe/cohorte administrative. Ils peuvent s'inscrire à la même classe de cours ou à des cours différents. 
 
Si votre école utilise ce type de groupe académique, vous pouvez utiliser cette fonction pour créer, modifier et supprimer les classes administratives : 
- Ajouter une classe 
- Classe d'importation 
- Modifier la classe 
- Supprimer la classe 
- Voir la liste des classes


![](/img/ulearning/admin/classes01.png)

## Ajouter une classe administrative

1. Cliquez sur l'onglet Classe administrative [1]. 
2. Vous pouvez filtrer/rechercher des classes administratives [2]. 
3. Pour ajouter une classe manuellement, cliquez sur le bouton Ajouter une classe [3].

![](/img/ulearning/admin/classes02.png)
## Importer des classes administratives 

Importer des classes administratives en vrac **[4]**. Assurez-vous de ce qui suit lorsque vous remplissez le formulaire : 
   - Le nom de la classe est obligatoire (chiffres et lettres uniquement). 
   - Si vous spécifiez des informations sur l'école (collège), les noms doivent correspondre exactement à ceux du système. 
 
## Modifier la classe administrative 
1. Pour modifier une classe administrative, cliquez sur le lien **Modifier** **[5]** pour cette classe. 
 
## Supprimer la classe administrative 
1. Pour supprimer une classe administrative, cliquez sur le lien **Supprimer** **[6]** pour cette classe. 
 
## Voir la liste des classes 
 1. Pour afficher la liste des étudiants d'une classe d'administration, cliquez sur le lien **Liste** **[7]** pour cette classe.

<Callout type="info" emoji="ℹ️">
  Assurez des informations précises lors de l'importation de classes et respectez les conventions de dénomination pour plus de cohérence.
</Callout>
