# Gestion des classes administratives

Une classe administrative est différente d’une classe de cours. Il s’agit d’une cohorte d’étudiants ayant intégré le même programme durant le même trimestre d’admission. À des fins administratives, une école peut regrouper ces étudiants dans une classe ou cohorte administrative. Ils peuvent être inscrits aux mêmes cours ou à des cours différents. 
Si votre établissement utilise ce type de groupe académique, vous pouvez utiliser cette fonctionnalité pour créer, modifier et supprimer les classes administratives :
- Ajouter une classe 
- Importer une classe 
- Modifier une classe
- Supprimer une classe 
- Voir la liste des classes


![](/img/ulearning/admin/classes01.png)

## Ajouter une classe administrative

1. Cliquez sur l'onglet Classe administrative [1]. 
2. Vous pouvez filtrer ou rechercher des classes administratives [2]. 
3. Pour ajouter une classe manuellement, cliquez sur le bouton Ajouter une classe [3].

![](/img/ulearning/admin/classes02.png)
## Importer des classes administratives 

Importez des classes administratives en lot **[4]**. Assurez-vous que les éléments suivants sont respectés lors du remplissage du formulaire : 
   - Le nom de la classe est obligatoire (uniquement lettres et chiffres). 
   - Si vous indiquez des informations sur l’établissement (collège), les noms doivent correspondre exactement à ceux présents dans le système.
 
## Modifier la classe administrative 
1. Pour modifier une classe administrative, cliquez sur le lien **Modifier** **[5]** correspondant à cette classe. 
 
## Supprimer la classe administrative 
1. Pour supprimer une classe administrative, cliquez sur le lien **Supprimer** **[6]**  correspondant à cette classe.
 
## Voir la liste des étudiants 
 1. Pour afficher la liste des étudiants d’une classe administrative, cliquez sur le lien **Liste** **[7]** pour cette classe.

<Callout type="info" emoji="ℹ️">
  Assurez des informations précises lors de l'importation de classes et respectez les conventions de dénomination pour plus de cohérence.
</Callout>
