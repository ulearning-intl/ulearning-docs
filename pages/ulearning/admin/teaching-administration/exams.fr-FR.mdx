# Guide de gestion des examens

Gérer efficacement les examens avec les fonctionnalités suivantes.
![](/img/ulearning/admin/exam01.png)

 **Accès à la console** 
 1. Cliquez sur l'onglet **Exams** **[1]**. 
     - Utilisez la fonction de filtre/recherche pour localiser les examens **[2]** (la recherche n'est pas sensible à la casse). 
     - Ouvrez la console de gestion des examens en cliquant sur le lien **Voir les résultats** **[3]** pour un examen spécifique. 
 
 ## Console de gestion des examens

![](/img/ulearning/admin/exam02.png)

 ### Modification des instructions pour l'examen 
 1. Modifier les instructions de l'examen **[1]** pour un examen. 
     - Ces instructions sont visibles par les candidats à l'examen sur l'écran d'accueil de l'examen avant de commencer l'examen. 
 
 ### Autres actions 
 1. Explorez une gamme d'actions dans la console de gestion des examens : 
     - Reportez-vous au Guide de l'enseignant ULearning **[2]** pour une liste complète des actions disponibles pour le titulaire de l'examen.

<Callout type="info" emoji="ℹ️">
    Pour les étapes détaillées de chaque action, consultez le Guide de l'enseignant ULearning.
</Callout>
