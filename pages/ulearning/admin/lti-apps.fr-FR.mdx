# Applications LTI

L'application LTI est une fonctionnalité de la plate-forme LMS qui permet aux administrateurs d'ajouter et de modifier des applications externes pour étendre les fonctionnalités et les ressources de la plate-forme. 
 
## Ajouter une application LTI

![](/img/ulearning/admin/lti01.png)

1. Cliquez sur l'onglet Applications **[1]** pour accéder à l'écran Applications LTI. 
2. Pour afficher les détails d'une application, cliquez sur le lien Détails >> **[2]** pour cette application. 
3. Pour ajouter une application externe à l'ULearning de votre école, cliquez sur le lien +Ajouter une application **[3]**.

![](/img/ulearning/admin/lti02.png)
1. Entrez un nom pour l'application [1]. 
2. Entrez la clé du consommateur fournie par le fournisseur de l'application [2]. 
3. Entrez le secret partagé fourni par le fournisseur de l'application [3]. 
4. Cliquez sur le bouton Confirmer [4]. 

Remarque : cette fonction est une version bêta à tester. Actuellement, seules deux applications externes sont disponibles dans notre environnement de test. Si votre école veut ajouter vos outils externes (compatibles avec LTI) à la plate-forme ULearning de votre école. Veuillez nous contacter pour poursuivre la discussion.
