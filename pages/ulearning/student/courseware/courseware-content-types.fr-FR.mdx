# Types de contenu de didacticiel

ULearning permet aux enseignants de créer et d'inclure divers types de contenu dans les didacticiels. Voici les différents types de contenu que vous pouvez rencontrer : 
 
### 1. Texte enrichi avec des images, des liens, etc. 
 
 - Ce type de contenu peut contenir du texte, des images et des hyperliens. 
 - Si la page ne contient que ce type de contenu, le système enregistrera votre apprentissage de la page comme terminé au fur et à mesure que vous chargerez cette page.

![](/img/ulearning/student/courseware09.png)

 ### 2. Contenu avec des questions d'exercice 
 
 - Les enseignants peuvent insérer 13 types de questions dans les pages de didacticiels. 
 - Suivez les instructions à l'écran pour compléter les questions d'exercice.


![](/img/ulearning/student/courseware10.png)
![](/img/ulearning/student/courseware11.png)

 - Après avoir répondu aux questions, cliquez sur le bouton Soumettre. 
 - Vos résultats d'exercice seront affichés. 
 
 Notes : L'enseignant peut déterminer : 
 - Combien de tentatives autorisées pour faire l'accise ? 
 - Si la nouvelle réessai est autorisée ou non. 
 - Si la bonne réponse sera affichée après avoir été soumise ou non.


![](/img/ulearning/student/courseware12.png)

### 3. Contenu avec vidéo intégrée des médias sociaux (par exemple, YouTube) 
 - Ce contenu est similaire au texte enrichi avec des images et des liens. 
 - Utilisez le lecteur vidéo pour gérer et naviguer dans le contenu.
![](/img/ulearning/student/courseware13.png)


### 4. Contenu avec documents intégrés (Word, PDF, PPT, etc.) 
 - Cliquez sur le lien "Cliquez pour afficher le document" pour ouvrir le fichier dans la visionneuse de cours.
![](/img/ulearning/student/courseware14.png)


### 5. Pour afficher le document sur un écran plus grand, cliquez sur le bouton "Plein écran"

![](/img/ulearning/student/courseware15.png)

 - Utilisez les outils de visualisation de documents pour effectuer un zoom avant/dézoomer et naviguer dans les pages. 
 - Cliquez sur le bouton "Plein écran" pour revenir à l'écran du didacticiel.

![](/img/ulearning/student/courseware16.png)

### 6. Contenu avec vidéo insérée par votre professeur 
 
 - Lorsque vous regardez cette vidéo pour la première fois, un message indique que le fait de faire glisser l'en-tête de lecture est désactivé. 
 - La page ne sera marquée comme "Terminée" qu'après avoir regardé toute la vidéo. 
 - Cliquez sur le bouton "J'ai compris".


![](/img/ulearning/student/courseware17.png)

  - Le statut de progression de la vidéo sera marqué comme "non visualisé" dans le coin inférieur droit du lecteur vidéo. 
  - Cliquez sur le bouton "Lire" pour commencer à regarder la vidéo.

![](/img/ulearning/student/courseware18.png)

 - Au fur et à mesure que vous regardez, l'état de la progression de la vidéo passera à "Partiellement terminé" et sera mis à jour avec le pourcentage terminé. 
 - Pour regarder la vidéo en mode plein écran, cliquez sur le bouton "Plein écran".

![](/img/ulearning/student/courseware19.png)
 - Après avoir terminé la vidéo, l'état de progression de la vidéo passera à "Finished". Si la page n'a que du texte enrichi et cette vidéo, le statut de progression de la page sera marqué d'un signe de coche.

![](/img/ulearning/student/courseware20.png)

### 7. Contenu avec le composant Spoken English Test 
 - Lorsque vous entrez dans une page avec un test d'anglais parlé, un message vous invitera à vous assurer que Flash est activé dans votre navigateur Web pour l'enregistrement vocal. 
 - Cliquez sur le bouton "X" pour fermer le message.

![](/img/ulearning/student/courseware21.png)

 - L'enseignant peut fournir un échantillon audio pour la prononciation/la pratique de l'anglais parlé. Vous pouvez cliquer sur le bouton "Play" pour l'écouter (facultatif).

![](/img/ulearning/student/courseware22.png)

  - L'enseignant peut tester votre anglais parlé pour les mots, les phrases et les paragraphes. Lorsque vous êtes prêt, cliquez sur le bouton "Enregistrer" et lisez à haute voix le contenu du test. 
  > **Remarque :** Assurez-vous que vous êtes dans une pièce calme et que votre bouche est proche du microphone de votre PC ou de votre appareil mobile.

![](/img/ulearning/student/courseware23.png)

  - Cliquez sur la vague d'enregistrement rouge pour terminer l'enregistrement.

![](/img/ulearning/student/courseware24.png)

  - Le système notera automatiquement votre enregistrement et affichera votre score au test d'anglais parlé sur le côté droit, ainsi que le nombre total de tentatives données par l'enseignant et le nombre de tentatives que vous avez utilisées. 
  - Vous pouvez cliquer sur le bouton "Audio" à côté du bouton Enregistrer pour écouter votre enregistrement.

> **Remarque :** Votre professeur peut vous donner jusqu'à 3 tentatives pour le test d'anglais parlé.

![](/img/ulearning/student/courseware25.png)

### 8. Contenu avec audio 
 - Votre professeur peut ou non ajouter des sous-titres à l'audio. 
 - Pour écouter l'audio, cliquez sur le bouton "Lire".

![](/img/ulearning/student/courseware26.png)

### 9. Contenu avec audio annoté 
 - C'est similaire au type de contenu audio précédent. 
 - La différence est que l'audio peut être annoté avec du texte riche par votre professeur. 
 - Pour écouter l'audio annoté, cliquez sur le bouton "Lire".

![](/img/ulearning/student/courseware27.png)
