# Types de contenu du matériel de cours

ULearning permet aux enseignants de créer et d'inclure divers types de contenu dans le matériel de cours. Voici les différents types de contenu que vous pouvez rencontrer :
 
### 1. Texte enrichi avec images, liens, etc. 
 
 - Ce type de contenu peut contenir du texte, des images et des hyperliens.
 - Si la page contient uniquement ce type de contenu, le système enregistrera votre apprentissage de la page comme terminé dès que vous la chargerez.

![](/img/ulearning/student/courseware09.png)

 ### 2. Contenu avec questions d’exercice 
 
 - Les enseignants peuvent insérer 13 types de questions dans les pages du matériel de cours.
 - Suivez les instructions à l'écran pour compléter les questions.


![](/img/ulearning/student/courseware10.png)
![](/img/ulearning/student/courseware11.png)

 - Après avoir répondu aux questions, cliquez sur le bouton Soumettre.
 - Vos résultats seront affichés.
 
 Notes : L’enseignant peut déterminer :
 - Combien de tentatives sont autorisées ?
 - Si une nouvelle tentative est autorisée ou non.
 - Si la bonne réponse sera affichée après soumission ou non.


![](/img/ulearning/student/courseware12.png)

### 3. Contenu avec vidéo intégrée depuis les réseaux sociaux (par exemple YouTube)
 - Ce contenu est similaire au texte enrichi avec images et liens.
 - Utilisez le lecteur vidéo pour gérer et naviguer dans le contenu.
![](/img/ulearning/student/courseware13.png)


### 4. Contenu avec documents intégrés (Word, PDF, PPT, etc.)
 - Cliquez sur le lien « Cliquez pour afficher le document » pour ouvrir le fichier dans le visualiseur du matériel de cours.
![](/img/ulearning/student/courseware14.png)


### 5. Pour afficher le document sur un écran plus grand, cliquez sur le bouton "Plein écran"

![](/img/ulearning/student/courseware15.png)

 - Utilisez les outils de visualisation de documents pour effectuer un zoom avant/arrière et naviguer dans les pages.
 - Cliquez sur le bouton "Plein écran" pour revenir à l’écran du matériel de cours.

![](/img/ulearning/student/courseware16.png)

### 6. Contenu avec vidéo insérée par votre enseignant 
 
 - Lorsque vous regardez cette vidéo pour la première fois, un message indiquera que le déplacement de la tête de lecture est désactivé.
 - La page sera marquée comme « Terminée » uniquement après avoir regardé l’intégralité de la vidéo.
 - Cliquez sur le bouton "J’ai compris".


![](/img/ulearning/student/courseware17.png)

  - Le statut de progression de la vidéo sera affiché comme « Non visualisé » dans le coin inférieur droit du lecteur vidéo.
  - Cliquez sur le bouton « Lecture » pour commencer à regarder la vidéo.

![](/img/ulearning/student/courseware18.png)

 - Pendant le visionnage, le statut passera à « Partiellement terminé » et sera mis à jour avec le pourcentage complété.
 - Pour regarder la vidéo en plein écran, cliquez sur le bouton "Plein écran".

![](/img/ulearning/student/courseware19.png)
 - Après avoir terminé la vidéo, le statut de progression passera à « Terminé ».
Si la page contient uniquement du texte enrichi et cette vidéo, le statut de progression de la page sera marqué par une coche.

![](/img/ulearning/student/courseware20.png)

### 7. Contenu avec le composant Spoken English Test
 - Lorsque vous accédez à une page contenant un test d’anglais parlé, un message vous invitera à vérifier que Flash est activé dans votre navigateur pour l’enregistrement vocal.
 - Cliquez sur le bouton "X" pour fermer le message.

![](/img/ulearning/student/courseware21.png)

 - L’enseignant peut fournir un échantillon audio pour la pratique de la prononciation. Vous pouvez cliquer sur le bouton « Lecture » pour l’écouter (optionnel).

![](/img/ulearning/student/courseware22.png)

  - L’enseignant peut tester votre anglais oral à travers des mots, phrases ou paragraphes. Lorsque vous êtes prêt, cliquez sur le bouton "Enregistrer" et lisez à voix haute le contenu.
  > **Remarque :** Assurez-vous d’être dans un environnement calme et que votre bouche soit proche du micro de votre PC ou de votre appareil mobile..

![](/img/ulearning/student/courseware23.png)

  - Cliquez sur l’onde rouge d’enregistrement pour terminer.

![](/img/ulearning/student/courseware24.png)

  - Le système notera automatiquement votre enregistrement et affichera votre score à droite, avec le nombre total de tentatives autorisées et celles que vous avez utilisées.
  - Vous pouvez cliquer sur le bouton "Audio" à côté du bouton "Enregistrer" pour écouter votre enregistrement.

> **Remarque :** Votre enseignant peut vous accorder jusqu’à 3 tentatives pour ce test.

![](/img/ulearning/student/courseware25.png)

### 8. Contenu avec audio 
 - Votre enseignant peut ou non ajouter des sous-titres à l'audio.
 - Pour écouter l'audio, cliquez sur le bouton « Lire ».

![](/img/ulearning/student/courseware26.png)

### 9. Contenu avec audio annoté 
 - C’est similaire au type de contenu audio précédent.
 - La différence est que l’audio peut être annoté avec du texte enrichi par votre professeur.
 - Pour écouter l’audio annoté, cliquez sur le bouton "Lire".

![](/img/ulearning/student/courseware27.png)
