# Visualisation des matériels de cours


 ### Étape 1 
 Cliquez sur l’onglet Matériel de cours dans le volet gauche.
![](/img/ulearning/student/courseware01.png)

 ### Étape 2 

 Vous verrez la liste de tous les matériels de cours de ce cours, accompagnée de leurs données d’apprentissage résumées ci-dessous.
 
 En haut, un résumé global du matériel de cours :
 - Les jours restants pour compléter le matériel de cours. 
 - La progression en pourcentage achevé.
 - Le temps total d’apprentissage que vous avez accumulé jusqu’à présent. 
 - Le score que vous avez obtenu jusqu’à présent. 
 
 En bas, les données d’apprentissage sont affichées module par module, ainsi que les actions possibles à chaque niveau :
 - La progression de l’apprentissage en pourcentage achevé.
 - Le temps total d’apprentissage que vous avez passé.
 - Le score obtenu.
 - Le temps autorisé pour l’apprentissage (si votre professeur l’a défini).
 - L’action ou les actions que vous pouvez effectuer pour chaque module.

 Remarques  : 
 - Votre enseignant peut vous expliquer comment les données d’apprentissage sont calculées ou mesurées.
 - Il peut aussi décider si les résultats seront pris en compte ou non dans votre note finale.

![](/img/ulearning/student/courseware02.png)


 ### Étape 3 
 
 S’agissant d’un nouveau matériel de cours, nous vous recommandons de commencer par le premier module :
 - Cliquez sur le bouton Démarrer pour apprendre.
 
 Remarque : 
 Selon la taille du contenu de la première page et la vitesse de votre connexion Internet, le chargement peut prendre un peu de temps.

![](/img/ulearning/student/courseware03.png)


 ### Étape 4 
 Si c’est votre première fois avec un matériel de cours ULearning, une série de conseils s’affichera pour vous aider à naviguer :
 - Cliquez sur Je sais pour lire chaque conseil.
 - Ou cliquez sur Ignorer tous les conseils pour commencer directement l’apprentissage.
![](/img/ulearning/student/courseware04.png)
![](/img/ulearning/student/courseware05.png)

### Étape 5

Dans la visionneuse du matériel de cours :
 - Par défaut, le volet gauche affiche la Table des matières (TOC). Cliquez sur un titre pour accéder directement à la page correspondante. Pour masquer/afficher la TOC, cliquez sur le bouton Masquer/Afficher la table des matières.
 - La zone de droite affiche le contenu d’apprentissage, page par page, avec le titre en haut.
 - En bas, un pied de page vous permet de naviguer :
     - Cliquez sur Page précédente/Titre pour revenir à la page précédente.
     - Cliquez sur Page suivante/Titre pour aller à la page suivante.
 
 - Une fois que vous avez complété le contenu d’une page, son titre dans la TOC sera marqué d’un signe de coche.

![](/img/ulearning/student/courseware06.png)

- Si vous avez terminé ou souhaitez quitter, cliquez sur Quitter pour revenir à la page du cours.
![](/img/ulearning/student/courseware07.png)

 - Une fois de retour, vous remarquerez que les données d’apprentissage du matériel de cours ont été mises à jour.
 - Pour continuer l’apprentissage d’un module en cours, cliquez sur Continuer l’apprentissage.
 - Pour commencer un nouveau module, cliquez sur Commencer à apprendre.
![](/img/ulearning/student/courseware08.png)
