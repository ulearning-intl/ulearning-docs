# Visualisation des didacticiels


 ### Étape 1 
 Cliquez sur l'onglet Courseware dans le volet gauche.
![](/img/ulearning/student/courseware01.png)

 ### Étape 2 

 Il répertorie tous les didacticiels de ce cours avec les données d'apprentissage des didacticiels résumées ci-dessous. 
 
 En haut, il y a le résumé de l'ensemble du didacticiel : 
 - Les jours restants pour compléter le didacticiel. 
 - La progression en pourcentage terminé. 
 - Le temps d'apprentissage total que vous avez déjà passé à jour. 
 - Le score que vous avez gagné jusqu'à présent. 
 
 En bas, il s'agit des données d'apprentissage module par module + les actions que vous pouvez prendre au niveau du module : 
 - La progression de l'apprentissage en pourcentage terminé. 
 - Le temps d'apprentissage total que vous avez passé à jour. 
 - Le score que vous avez gagné jusqu'à présent. 
 - Le temps autorisé pour l'apprentissage (si votre professeur l'a spécifié). 
 - La ou les actions que vous pouvez effectuer pour chaque module. 
 
 Notes : 
 - Votre enseignant peut expliquer comment les données d'apprentissage sont calculées ou mesurées. 
 - Votre enseignant peut également déterminer si les résultats d'apprentissage des didacticiels seront inclus ou exclus de votre note finale.

![](/img/ulearning/student/courseware02.png)


 ### Étape 3 
 
 En tant que nouveau didacticiel, nous vous recommandons de commencer à partir du premier module : 
 - Cliquez sur le bouton Démarrer pour apprendre. 
 
 Notes : 
 En fonction de la taille du contenu de la première page de votre didacticiel et de votre vitesse Internet, cela peut prendre un certain temps pour charger le didacticiel pour la première fois.

![](/img/ulearning/student/courseware03.png)


 ### Étape 4 
 Si c'est la première fois que vous consultez les didacticiels ULearning, la visionneuse de didacticiels affichera une série de conseils pour vous montrer comment naviguer dans la visionneuse de didacticiels. Vous pouvez : 
 - Soit cliquez sur le bouton Je sais pour passer en revue chaque conseil. 
 - Ou cliquez sur le lien Ignorer tous les conseils pour commencer l'apprentissage.
![](/img/ulearning/student/courseware04.png)
![](/img/ulearning/student/courseware05.png)

### Étape 5

Sur la visionneuse de didacticiels : 
 - Par défaut, le volet de gauche est la table des matières (TOC), vous pouvez cliquer sur n'importe quel titre de page pour accéder directement à cette page. Si vous souhaitez masquer ou afficher la table des matières, cliquez sur le bouton Masquer la table des matières/Afficher la table des matières ci-dessus. 
 - La bonne zone est l'endroit où afficher le contenu de l'apprentissage page par page avec la vignette de page en haut. 
 - Le pied de page sous la zone de contenu principale héberge le bouton de navigation pour naviguer dans le contenu : 
     - Cliquez sur le bouton Page précédente/Titre pour revenir à la page de contenu précédente. 
     - Cliquez sur le bouton Page suivante/Titre pour revenir à la page de contenu suivante. 
 
 - Si vous avez terminé tout le contenu de la page, le titre de la page TOC sera marqué d'un signe de coche.

![](/img/ulearning/student/courseware06.png)

- Si vous avez terminé ou souhaitez quitter votre apprentissage, cliquez sur le bouton Quitter pour revenir à la page de votre cours.
![](/img/ulearning/student/courseware07.png)

 - Vous êtes de retour sur la page du cours et vous remarquerez que les données d'apprentissage du logiciel de cours sont mises à jour. 
 - Pour poursuivre l'apprentissage du même module, cliquez sur le bouton Continuer l'apprentissage de ce module. 
 - Pour commencer l'apprentissage d'un nouveau module, cliquez sur le bouton Commencer à apprendre pour ce module.
![](/img/ulearning/student/courseware08.png)
