# Passer un examen

 ### Étape 1 
 Depuis la page d’accueil du cours, cliquez sur l’onglet Examens.
![](/img/ulearning/student/exam01.png)

 ### Étape 2 
 Sur l'écran Examens : 
 - Tous les examens qui vous sont assignés y sont répertoriés.
 - Portez une attention particulière aux dates de début et de fin pour ne pas manquer la date limite.
 - Un statut de progression est affiché pour chaque examen.
 - Pour commencer un examen, cliquez sur le bouton Démarrer l’examen correspondant.
![](/img/ulearning/student/exam02.png)

 ### Étape 3 
 Cliquez sur le bouton Je sais, maintenant commencer.
![](/img/ulearning/student/exam03.png)

### Étape 4 
 Sur l'écran de l'examen : 
 - En haut, vous trouverez les informations de l’examen : durée, nombre total de points, nombre total de questions, disponibilité, etc.
 - Un compte à rebours est également affiché en haut pour indiquer le temps restant.
 - Vous avez trois options pour un examen déjà commencé : 
     - Le reprendre plus tard
     - Enregistrer 
     - Soumettre 
 - Faites attention au nombre de sections de l’examen pour ne pas en oublier.

![](/img/ulearning/student/exam04.png)

- Suivez les instructions à l’écran pour répondre à chaque question.
- Dans la barre de numéros de questions en bas, les questions avec un fond sombre sont celles déjà répondues, celles avec un fond clair ne le sont pas encore. Cliquez sur un numéro pour accéder directement à une question spécifique.
![](/img/ulearning/student/exam05.png)

 ### Étape 5 
 Pour soumettre l’examen, cliquez sur le bouton Soumettre l’examen.
![](/img/ulearning/student/exam06.png)


### Étape 6 
 Dans la fenêtre contextuelle, cliquez sur le bouton Soumettre l’examen.
 - Remarque : un message peut vous informer qu’il reste des questions sans réponse. Si vous souhaitez continuer, cliquez sur "Continuer à répondre".
![](/img/ulearning/student/exam07.png)
