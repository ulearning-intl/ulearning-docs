# Passer un examen

 ### Étape 1 
 Sur la page d'accueil du cours, cliquez sur l'onglet Examens.
![](/img/ulearning/student/exam01.png)

 ### Étape 2 
 Sur l'écran Examens : 
 - Il répertorie tous les examens qui vous sont assignés. 
 - Portez une attention particulière aux dates de début/fin des examens et assurez-vous de ne pas manquer la date d'échéance. 
 - Un statut de progression est attribué à chaque affectation. 
 - Pour passer un examen, cliquez sur le bouton "Démarrer l'examen" pour cet examen.
![](/img/ulearning/student/exam02.png)

 ### Étape 3 
 Cliquez sur le bouton "Je sais, maintenant commencer".
![](/img/ulearning/student/exam03.png)

### Étape 4 
 Sur l'écran de l'examen : 
 - En haut, il répertorie les informations de l'examen telles que le temps d'examen, le nombre total de points possibles, le nombre total de questions, la disponibilité de l'examen, etc. 
 - Une minuterie s'affiche également en haut, indiquant combien de temps il reste pour votre examen. 
 - Vous avez trois choix pour l'examen que vous avez déjà commencé : 
     - Prenez-le la prochaine fois 
     - Enregistrer 
     - Soumettre 
 - Faites très attention au nombre de sections de l'examen et assurez-vous de ne manquer aucune d'entre elles.

![](/img/ulearning/student/exam04.png)

- Suivez les instructions à l'écran pour répondre à chaque question d'examen. 
- Sur la barre de numéro de question en bas, les questions avec un fond sombre sont celles auxquelles vous avez répondu, tandis que celles avec un fond de couleur claire sont celles auxquelles vous n'avez pas répondu. Vous pouvez cliquer sur le numéro de la question pour accéder directement à cette question.
![](/img/ulearning/student/exam05.png)

 ### Étape 5 
 Pour soumettre l'examen, cliquez sur le bouton "Soumettre l'examen".
![](/img/ulearning/student/exam06.png)


### Étape 6 
 Cliquez sur le bouton "Soumettre l'examen" dans la fenêtre contextuelle. 
 - Remarque : le message peut vous informer que vous avez encore des questions sans réponse. Si vous souhaitez continuer à travailler sur l'examen, cliquez sur le bouton "Continuer à répondre".
![](/img/ulearning/student/exam07.png)
