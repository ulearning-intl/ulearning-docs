# Affichage des ressources

 Pour consulter les ressources de la plate-forme ULearning, procédez comme suit : 
 
 ### Étape 1 
 Cliquez sur l'onglet Ressources dans le volet gauche.
![](/img/ulearning/student/resource01.png)

 ### Étape 2 
 Il répertorie tous les fichiers de ressources de ce cours. Pour afficher/ouvrir un fichier, cliquez sur le bouton Afficher.
![](/img/ulearning/student/resource02.png)

 Il ouvre la ressource dans une nouvelle fenêtre : 
 - Vous pouvez utiliser les boutons de navigation intégrés associés au type de fichier pour naviguer dans le fichier. 
 - Pour télécharger le fichier, cliquez sur le bouton Télécharger. 
 
 Remarques : Cela ne s'applique que si votre professeur a configuré le fichier avec l'option de téléchargement. 
 - Après avoir terminé votre lecture, cliquez sur le bouton Fermer pour fermer le fichier.
![](/img/ulearning/student/resource03.png)
