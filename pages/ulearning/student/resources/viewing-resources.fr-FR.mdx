# Affichage des ressources

 Pour consulter les ressources sur la plateforme ULearning, suivez les étapes ci-dessous :
 
 ### Étape 1 
 Cliquez sur l’onglet Ressources dans le volet de gauche.
![](/img/ulearning/student/resource01.png)

 ### Étape 2 
 Une liste de tous les fichiers de ressources du cours s’affiche. Pour visualiser ou ouvrir un fichier, cliquez sur le bouton Afficher.
![](/img/ulearning/student/resource02.png)

 Le fichier s’ouvre dans une nouvelle fenêtre :
 - Vous pouvez utiliser les boutons de navigation intégrés, selon le type de fichier, pour parcourir le document.
 - Pour télécharger le fichier, cliquez sur le bouton Télécharger.
 
 Remarque : cette option n’est disponible que si votre enseignant a autorisé le téléchargement du fichier.
 - Une fois votre lecture terminée, cliquez sur le bouton Fermer pour quitter le fichier.
![](/img/ulearning/student/resource03.png)
