# Viewing Resources

To view resources in the ULearning platform, follow these steps:

### Step 1
Click the Resources tab in the left pane.
![](/img/ulearning/student/resource01.png)

### Step 2
It lists all the resource files in this course. To view/open a file, click the View button.
![](/img/ulearning/student/resource02.png)

It opens the resource in a new window:
- You can use the built-in navigation buttons associated with the file type to navigate the file.
- To download the file, click the Download button.

Notes: It applies, only if your teacher set up the file with the download option.
- After you've done your reading, click the Close button to close the file.
![](/img/ulearning/student/resource03.png)
