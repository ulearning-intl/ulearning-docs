 # Ressources de visualisation : Micro-cours

 Le micro-cours est un type de contenu d’apprentissage similaire à une présentation PowerPoint, créé à l’aide de l’outil de création exclusif d’ULearning. Trois types de fichiers présents dans la bibliothèque de ressources ULearning peuvent être convertis en micro-cours : ppt, pdf et image. Il s’agit essentiellement d’un contenu basé sur des diapositives avec une narration audio.
 
 ### Étape 1 
 Cliquez sur le bouton Afficher pour le micro-cours que vous souhaitez ouvrir.
 Remarque : les micro-cours ULearning sont identifiés par une petite icône bleue ressemblant à un lecteur.
![](/img/ulearning/student/resource04.png)

 ### Étape 2 
 La visionneuse de micro-cours est très similaire à PowerPoint. L’interface est simple et intuitive :
 - Pour écouter la narration, cliquez sur le bouton Lecteur.
 - Utilisez la flèche vers le haut pour revenir à la diapositive précédente, et la flèche vers le bas pour passer à la suivante.
 - L’état de progression est affiché en bas de l’écran.
 - Pour passer en mode Trieur de diapositives, cliquez sur le bouton représentant quatre petits carrés.
![](/img/ulearning/student/resource05.png)

 Depuis la vue Trieur de diapositives
 - Vous pouvez cliquer sur une diapositive pour y accéder directement.
 - Pour savoir comment votre progression est calculée, cliquez sur le lien Description du progrès.
![](/img/ulearning/student/resource06.png)

 Après avoir consulté le message "Description du progrès", cliquez sur le bouton Confirmer.
![](/img/ulearning/student/resource07.png)

 Après avoir terminé le micro-cours, cliquez sur le bouton Fermer pour revenir à la page de votre cours.
![](/img/ulearning/student/resource08.png)
