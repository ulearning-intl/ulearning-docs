 # Ressources de visualisation : Micro Course

 Le micro cours est un type de contenu d'apprentissage de type PowerPoint créé à l'aide de l'outil de création propriétaire d'ULearning. Il existe trois types de fichiers dans la bibliothèque de ressources ULearning qui peuvent être convertis en micro-cours : ppt, pdf et image. Il s'agit essentiellement d'un type de contenu basé sur des diapositives avec voix off. 
 
 ### Étape 1 
 Cliquez sur le bouton Afficher pour le micro cours que vous souhaitez ouvrir. 
 Notes : Les micro-cours ULearning sont marqués d'une petite icône bleue ressemblant à un joueur.
![](/img/ulearning/student/resource04.png)

 ### Étape 2 
 La visionneuse de micro-cours est très similaire à PowerPoint. L'interface est simple et intuitive : 
 - Pour écouter la voix off, cliquez sur le bouton Lecteur. 
 - Utilisez la flèche vers le haut pour revenir à la diapositive précédente et la flèche vers le bas vers la diapositive suivante. 
 - En bas, l'état de progression est affiché. 
 - Pour passer à la vue Trieur de diapositives, cliquez sur le bouton 4 petits carrés.
![](/img/ulearning/student/resource05.png)

 À partir de la vue du trieur de diapositives : 
 - Vous pouvez cliquer sur une diapositive pour accéder directement à cette diapositive. 
 - Pour savoir comment vos progrès d'apprentissage sont calculés, cliquez sur le lien Description du progrès.
![](/img/ulearning/student/resource06.png)

 Après avoir consulté le message "Description des progrès", cliquez sur le bouton Confirmer.
![](/img/ulearning/student/resource07.png)

 Après avoir terminé l'apprentissage du micro-cours, cliquez sur le bouton Fermer pour revenir à la page de votre cours.
![](/img/ulearning/student/resource08.png)
