# Réalisation d’un devoir de quiz

 ### Étape 1 
 Depuis la page d’accueil du cours, cliquez sur l’onglet Devoirs.
![](/img/ulearning/student/assignment01.png)

 ### Étape 2 
 L’écran des devoirs affiche tous les devoirs que l’enseignant a assignés à la classe :
 - Il existe trois types de devoirs : 
     - Devoir individuel (indiqué par “Individuel”)
     - Devoir de groupe (indiqué par “Groupe”)
     - Devoir de quiz (indiqué par "Quiz") 
 - Portez une attention particulière aux dates de début et de fin des devoirs pour ne pas dépasser la date limite.
 - Un statut de progression est également affiché pour chaque devoir.
 - Pour compléter un quiz, cliquez sur le bouton Écrire les devoirs correspondant à ce quiz.


![](/img/ulearning/student/assignment02.png)

 ### Étape 3 
 Suivez les instructions à l’écran pour répondre à chaque question du quiz.

![](/img/ulearning/student/assignment03.png)

 ### Étape 4 
 Une fois le quiz terminé, cliquez sur le bouton Soumettre le devoir.
![](/img/ulearning/student/assignment04.png)
