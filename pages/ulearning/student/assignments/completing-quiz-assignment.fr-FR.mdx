# Achèvement du devoir de quiz

 ### Étape 1 
 Sur la page d'accueil du cours, cliquez sur l'onglet Affectation.
![](/img/ulearning/student/assignment01.png)

 ### Étape 2 
 Sur l'écran des devoirs, il répertorie tous les devoirs que l'enseignant a assignés à la classe : 
 - Il existe trois types d'affectations : 
     - Affectation individuelle (marquée "Individuel") 
     - Affectation de groupe (marquée "Groupe") 
     - Devoir de quiz (marqué "Quiz") 
 - Faites attention aux dates de début/fin des devoirs et assurez-vous de ne pas manquer la date d'échéance. 
 - Un statut de progression est également attribué à chaque affectation. 
 - Pour compléter un quiz, cliquez sur le bouton "Écrire les devoirs" pour ce quiz.


![](/img/ulearning/student/assignment02.png)

 ### Étape 3 
 Suivez les instructions à l'écran pour répondre à chaque question du quiz.

![](/img/ulearning/student/assignment03.png)

 ### Étape 4 
 Une fois que vous avez terminé, cliquez sur le bouton Soumettre le devoir.
![](/img/ulearning/student/assignment04.png)
