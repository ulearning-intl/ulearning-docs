# Vue d'ensemble

Il s'agit d'un guide de démarrage rapide pour les étudiants. L'objectif de ce guide est de donner aux utilisateurs étudiants d'ULearning les connaissances et les compétences nécessaires pour utiliser ULearning LMS. En suivant les étapes de ce guide, vous pouvez rapidement commencer votre voyage pour effectuer les tâches LMS les plus courantes en tant qu'étudiant. 
 
Ce guide est organisé autour des tâches nécessaires que vous allez effectuer, étape par étape accompagnées de captures d'écran de la plateforme ULearning.
