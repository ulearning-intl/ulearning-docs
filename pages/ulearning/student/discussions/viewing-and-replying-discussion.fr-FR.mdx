# Visualisation et réponse aux discussions

 ### Étape 1 
 Sur la page d'accueil du cours, cliquez sur l'onglet Discussions.
![](/img/ulearning/student/discussion01.png)

 ### Étape 2 
 Sur l'écran Discussions : 
 - Vos données globales de discussions sont affichées en haut de l'écran. 
 - Les données de chaque point de discussion sont énumérées ci-dessous. 
 - Portez une attention particulière à la colonne Heure afin de ne pas manquer la date d'échéance pour les discussions limitées dans le temps. 
 - Pour afficher/répaire à une discussion, cliquez sur le bouton "Afficher" pour cette discussion.
![](/img/ulearning/student/discussion02.png)

 ### Étape 3 
 - Pour répondre à la discussion, entrez votre réponse dans le champ de texte enrichi en bas ou cliquez sur le bouton "Répondre" juste en dessous du message de discussion original publié par l'enseignant. 
 - Pour aimer une publication, cliquez sur l'icône du pouce vers le haut pour cette publication. 
 - Pour ne pas être un message, cliquez sur le lien "Annuler ce que vous aimez". 
 - Pour répondre à un message spécifique, cliquez sur le lien "Répondre" à ce message.
![](/img/ulearning/student/discussion03.png)

 ### Étape 4 
 Une fois que vous avez fini de rédiger votre réponse, cliquez sur le bouton "Soumettre".
![](/img/ulearning/student/discussion04.png)

 - Votre réponse sera affichée en haut de la discussion. 
 - Cliquez sur le bouton "Fermer" pour revenir à la page de votre cours.
![](/img/ulearning/student/discussion05.png)

 ### Étape 5 
 Vous remarquerez que les données de vos discussions sont mises à jour.
![](/img/ulearning/student/discussion06.png)
