# Connexion Ulearning

1. Cliquez sur le bouton de connexion.
![](/img/ulearning/student/login01.png)

2. Entrez votre nom d'utilisateur/mot de passe, puis cliquez sur le bouton Connexion.
![](/img/ulearning/student/login02.png)

3. Il ouvre votre page d'accueil (liste des cours) et vous êtes connecté.
![](/img/ulearning/student/login03.png)
