# Visualisation/lecture d'une annonce

 Les annonces sont utilisées par votre professeur pour communiquer avec vous concernant la logistique de votre cours. Il peut les utiliser pour :
 - Les annonces sont utilisées par votre professeur pour communiquer avec vous concernant la logistique de votre cours. Il peut les utiliser pour :
 - Vous orienter vers des ressources internes et externes qui vous aideront à atteindre les objectifs du cours.
 - Laisser un message pour toute la classe avec des commentaires vidéo ou audio.
 - Célébrer la réussite des étudiants et les événements importants susceptibles de vous intéresser.

 ### Étape 1 
 Pour lire une annonce, cliquez sur le bouton Développer afin d'afficher le message complet.
 - Remarque : La mention "Lire" indique que vous n'avez pas encore lu l'intégralité du message.


![](/img/ulearning/student/announcements01.png)

 ### Étape 2 
 Le message complet s’affiche.
 - Si l’annonce contient un lien, cliquez dessus pour ouvrir la page correspondante.
 - Si l’annonce contient des pièces jointes, cliquez sur Télécharger pour récupérer le fichier.
 - Une fois le message lu, cliquez sur Réduire pour le refermer.
