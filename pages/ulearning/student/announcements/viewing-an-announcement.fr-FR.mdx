# Visualisation/lecture d'une annonce

 Les annonces sont utilisées par votre professeur pour communiquer avec vous au sujet de la logistique de votre cours. Votre professeur peut utiliser les annonces pour : 
 - Vous rappeler ce que vous devez accomplir pour rester sur la bonne voie. 
 - Vous oriente vers des ressources internes et externes qui vous aideront à atteindre les résultats des cours. 
 - Laissez un message pour toute la classe avec des commentaires vidéo ou audio. 
 - Célébrez la réussite des étudiants et les événements importants qui pourraient vous intéresser. 

 ### Étape 1 
 Pour une annonce, cliquez sur le bouton **Élargir** pour lire l'intégralité du message. 
 - Notes : Le signe "Lire" indique que vous n'avez pas lu le message complet de cette annonce.


![](/img/ulearning/student/announcements01.png)

 ### Étape 2 
 Tout le message s'affiche. 
 - Si l'annonce contient un lien, cliquez sur le lien pour ouvrir la page liée. 
 - Si l'annonce contient des pièces jointes, cliquez sur le lien **Télécharger** pour télécharger le fichier. 
 - Après l'avoir lu, cliquez sur le bouton **Rechute** pour réduire le message.
