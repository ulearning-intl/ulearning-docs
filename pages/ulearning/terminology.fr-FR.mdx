 # Terminologie ULearning 

 ## Annonces 
 Annonces est un outil de communication qui permet aux enseignants de publier des annonces pour tous les étudiants d'un cours. Les annonces peuvent être publiées sur un cours ou sur une ou plusieurs classes sélectionnées au sein d'un cours. Les super administrateurs d'ULearning peuvent également faire des annonces pour l'ensemble de la plate-forme ULearning. Les annonces sont accessibles via le lien "Annonces" dans la navigation du cours. 
 
 ## Affectations 
 Un devoir est une activité d'évaluation créée par l'enseignant. Les devoirs peuvent inclure des devoirs individuels, des devoirs de groupe et des quiz. Les devoirs peuvent être attribués à tout le monde dans le cours ou différenciés par classe. Le lien "Devoirs" dans la navigation sur le cours donne accès à la page des devoirs, qui répertorie tous les types de devoirs d'un cours. 
 
 ## Calendrier 
 Le calendrier est un outil de communication qui montre aux utilisateurs tous les événements et les devoirs de leurs cours. Il est accessible via le lien "Calendrier" sur la page d'accueil de l'enseignant. 
 
 ## Cours 
 Les classes sont des divisions d'étudiants au sein d'un cours. Toutes les classes de cours voient le même contenu de cours, mais les plans d'apprentissage peuvent être différenciés. Par exemple, un cours peut avoir plusieurs cours avec des horaires différents. Les classes sont parfois appelées offres ou sections. 
 
 ## Cours 
 Les cours sont des unités d'enseignement dans une matière qui durent généralement un semestre. Les cours peuvent être créés par des administrateurs de cours ou des enseignants. Un cours ne peut avoir qu'une seule classe ou plusieurs classes (sections/offres). 
 
 ## Cours 
 Le didacticiel est un objet d'apprentissage numérique créé par l'enseignant. Il est structuré en cinq couches : didacticiel, module, sujet, page et éléments de page (texte, images, vidéo, audio, plugin, questions de test, etc.). Le didacticiel peut également contenir ses propres ressources et des questions de quiz/test/pratique. Il peut être utilisé de manière autonome ou inclus dans un cours comme contenu d'apprentissage du cours. Le lien "Courseware" dans la navigation du cours donne accès au matériel de cours. 
 
 ## Statut des didacticiels 
 Le statut du didacticiel fait référence à l'état d'un didacticiel ULearning. Tous les nouveaux didacticiels ULearning commencent dans un état non publié. Un état publié signifie que le didacticiel est disponible pour les utilisateurs. 
 
 ## Page d'accueil du cours 
 La page d'accueil du cours est la première page que les enseignants voient dans un cours. Il est défini comme la page d'annonce du cours. 
 
 ## Navigation sur le cours 
 La navigation de cours est un menu sur le côté gauche d'un cours ULearning. Il se compose de liens de navigation dirigés vers toutes les zones de caractéristiques d'un parcours. 
 
 ## Discussions 
 Discussions est un forum conçu pour faciliter la communication informelle entre les étudiants d'un cours. Les discussions peuvent être attribuées à tout le monde dans le cours ou différenciées par classe. Le lien "Discussions" dans la navigation du cours répertorie toutes les discussions d'un cours. 
 
 ## Notes 
 Les notes sont une mesure de la performance des élèves. Les enseignants peuvent définir des règles de notation et consulter les notes des élèves sur la page Notes. Le lien "Grades" dans la navigation du cours donne accès à la page Notes. 
 
 ## Groupes 
 Les groupes sont un outil qui aide les enseignants à séparer les étudiants en unités de cours plus petites. Des groupes peuvent être créés pour les étudiants afin de permettre une collaboration efficace. 
 
 ## Éditeur de mathématiques 
 L'éditeur de mathématiques est une barre d'outils compatible avec le latex dans l'éditeur de contenu enrichi qui permet aux enseignants d'écrire facilement des expressions et des équations mathématiques. Les enseignants et les élèves peuvent utiliser l'éditeur de mathématiques lors de la création et de la prise de devoirs/de quiz/d'examens. 
 
 ## Membres 
 Les membres sont l'endroit où l'enseignant gère les classes, les élèves et les membres de l'équipe d'enseignement. Il est accessible via le lien "Membres" dans la navigation du cours. 
 
 ## Éditeur de contenu enrichi 
 L'éditeur de contenu enrichi (RCE) est un traitement de texte qui aide les enseignants à créer et à formater du contenu dans des domaines spécifiques de l'ULearning. 
 
 ## Rôle 
 Un rôle décrit les autorisations accordées à un utilisateur donné dans un contexte donné (cours et compte). ULearning comprend à la fois des rôles au niveau du cours et au niveau du compte. Les types de rôles de base sont Student, Teacher Assistant (TA), Teacher, Admin et Super Admin. 
 
 ## Vue de l'étudiant 
 Student View est un outil utilisé par les enseignants pour voir à quoi ressemble leur cours aux étudiants. L'activation de Student View génère un étudiant de test qui apparaît dans un cours ULearning. 
 
 ## Agir comme 
 Act As est un outil utilisé par les administrateurs pour agir en tant qu'autre utilisateur (enseignant ou étudiant) pour voir ou effectuer tout ce que l'utilisateur peut voir ou effectuer. Il est principalement utilisé à des fins de dépannage. 
 
 ## Semestre/Terme 
 Un semestre est une période de temps affectée à l'enseignement académique. Dans ULearning, les dates des cours s'alignent généralement sur les dates du trimestre, bien que les dates des cours puissent s'étendre au-delà ou être plus courtes que les dates du trimestre par défaut. 

 ## Examens 
 Examens est un lien dans la navigation de cours qui affiche tous les examens de cours d'un cours et diverses actions liées à l'examen que l'enseignant peut effectuer. Les examens sont comme des conteneurs qui contiennent tout le contenu lié à l'examen.
 
 ## Examen 
 Un examen est un objet d'évaluation créé par un enseignant et prêt à être inséré dans un examen de classe. Il est composé de questions d'examen et peut être réutilisé et partagé. 
 
 ## Question d'examen 
 Une question d'examen est une question de test. ULearning prend en charge 13 types de questions d'examen, y compris le choix multiple, la réponse multiple, le vrai/faux, le remplissage du blanc, la réponse courte, la formule calculée, la séquence, la banque de mots, la liste déroulante multiple, le test Cloze, la langue - lecture à haute voix, la langue - orale et la combinaison. 
 
 ## Banque de questions 
 La banque de questions est une bibliothèque contenant des questions de test qui peuvent être ajoutées aux examens, aux quiz et aux activités pratiques. 
 
 ## Banque d'examens 
 Exam Bank est une bibliothèque contenant des objets d'examen prédéfinis et prêts à l'emploi qui peuvent être ajoutés aux examens de classe. 
 
 ## Examen de classe 
 Un examen de classe utilise soit des examens de la banque d'examens (objets d'examen prédéfinis et prêts à l'emploi), soit des examens générés aléatoirement directement à partir de la banque de questions (objets d'examen aléatoires/instantanés) pour mesurer les connaissances, les compétences, les aptitudes, etc. d'un candidat. Les examens de classe peuvent être attribués à tout le monde dans le cours ou différenciés par classe ou étudiant. 
 
 ## Éditeur de questions 
 Question Editor est l'outil de création propriétaire d'ULearning pour créer des questions d'examen. Il est lancé lorsque le bouton "Ajouter une question" est cliqué dans la plate-forme ULearning. 
 
 ## U-Author (Éditeur de cours) 
 U-Author est l'outil de création propriétaire d'ULearning pour créer des didacticiels. Il est lancé lorsque le bouton "Créer/Modifier des didacticiels" est cliqué dans la plate-forme ULearning. 
 
 ## Éditeur de micro-cours 
 Micro Course Editor est l'outil de création propriétaire d'ULearning pour créer des micro-cours. Il est lancé lorsque le bouton "Record Micro-course" est cliqué dans la plate-forme ULearning. 
 
 ## Éditeur vidéo 
 Video Editor est l'outil d'édition propriétaire d'ULearning pour éditer des vidéos dans les didacticiels. Il est lancé lorsque le bouton "Modifier" est cliqué sur une vidéo insérée dans un didacticiel ULearning.
