## 3. Trae 里克隆项目

1. 随便哪里新建文件夹
2. 打开文件夹在路径栏输入 `cmd` 然后回车
3. 在打开的弹窗输入：

```bash
git clone -b test https://github.com/ulearning-intl/ulearning-docs.git
```

4. 最后显示 `done` 即代表完成了
5. 然后用 Trae 打开这个下载的文件夹
