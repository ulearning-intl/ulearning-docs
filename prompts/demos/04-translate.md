详细版提示词位置：prompts/templates/04-translate.md

# 翻译文档内容

---

## 场景一：翻译单个文件

### 格式1：翻译到所有其他语言（默认不覆盖）
```
使用提示词04：场景：【单文件】 源文件路径：【ulearning/getting-started】 源语言：【zh-CN】
```

### 格式2：翻译到指定语言（默认不覆盖）
```
使用提示词04：场景：【单文件】 源文件路径：【ulearning/getting-started】 源语言：【zh-CN】 目标语言：【en-US】
```

### 格式3：强制覆盖已有内容
```
使用提示词04：场景：【单文件】 源文件路径：【ulearning/getting-started】 源语言：【zh-CN】 是否覆盖：【是】
```

---

## 场景二：翻译文件夹下所有文件

### 格式4：翻译文件夹到所有其他语言（默认不覆盖）
```
使用提示词04：场景：【文件夹】 文件夹路径：【smart-scheduling】 源语言：【zh-CN】
```

### 格式5：翻译文件夹到指定语言（默认不覆盖）
```
使用提示词04：场景：【文件夹】 文件夹路径：【smart-scheduling】 源语言：【zh-CN】 目标语言：【en-US】
```

### 格式6：强制覆盖文件夹所有文件
```
使用提示词04：场景：【文件夹】 文件夹路径：【smart-scheduling】 源语言：【zh-CN】 是否覆盖：【是】
```

---

## 参数说明

| 参数 | 说明 | 必填 | 默认值 |
|---|---|---|---|
| `场景` | `单文件` 或 `文件夹` | 是 | — |
| `源文件路径` | MDX 文件相对路径（不含语言后缀），如 `ulearning/getting-started` | 场景=单文件 | — |
| `文件夹路径` | 文件夹相对路径（不含语言后缀），如 `smart-scheduling` | 场景=文件夹 | — |
| `源语言` | 源文件的语言：`zh-CN`、`en-US`、`fr-FR`、`ar` | 是 | — |
| `目标语言` | 翻译到哪个语言，留空表示翻译到**所有其他语言** | 否 | 所有其他语言 |
| `是否覆盖` | `是`：覆盖已有内容；`否`：跳过正文有内容的文件 | 否 | `否` |

**注意**：`ar` 语言的 MDX 文件和 `_meta.json` 后缀为 `ar`（而非 `ar-AR`），如 `getting-started.ar.mdx`、`_meta.ar.json`
