# 在产品子目录下添加页面/目录


### 格式1：新增目录/单页面
```
使用提示词02：顶部栏路径：【产品->学习管理系统/文档】 父目录【无/管理者/管理者->教学管理/入门】 子项：【安装指南,配置说明(单页面),一级目录->二级目录,一级目录->二级目录(文件夹)】 插入位置：【插在xxx之后/最前面/最后面】
```

### 格式2：批量导入
```
使用提示词02-批量：顶部栏路径：【产品->学习管理系统】
父目录【管理者】 子项：【安全设置,插件管理】 插入位置：【插在系统设置之后】
父目录【教学管理】 子项：【课程管理(单页面),考试管理】 插入位置：【最前面】
父目录【无】 子项：【API参考】 插入位置：【最后面】
```

**说明**：
- 开头指定产品目录路径，后续所有操作都在该产品目录下
- 每行一个条目，依次填写父目录、子项、插入位置
- 子项格式：`名称`（有侧边栏目录）、`名称(单页面)`（单页面）、`一级目录->二级目录`（层级嵌套）
- **ar 语言后缀**：`ar` 语言的 MDX 文件和 `_meta.json` 后缀为 `ar`（而非 `ar-AR`），如 `getting-started.ar.mdx`、`_meta.ar.json`
