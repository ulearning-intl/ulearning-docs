# 写入文档内容

## 格式：写入内容到指定 MDX 文件

### 场景1：新增文件
```
使用提示词03：文件路径：【ulearning/getting-started】 语言：【zh-CN】 内容：【这里填入文档内容，支持 Markdown 和 MDX 语法】 翻译同步：【是/否】
```

### 场景2：修改已有文件（覆盖全部内容）
```
使用提示词03：文件路径：【ulearning/getting-started】 语言：【zh-CN】 内容：【更新后的全部内容...】 翻译同步：【是/否】
```

### 场景3：修改已有文件（在指定位置插入内容）
```
使用提示词03：文件路径：【ulearning/getting-started】 语言：【zh-CN】 插入位置：【## 安装步骤之后】 内容：【### 常见问题\n\nQ: 如何重置密码？\nA: 联系管理员重置。】 翻译同步：【是/否】   
```

### 通用说明

- 文件路径相对于 `pages/` 目录，不包含语言后缀，后缀由语言参数自动拼接
- 提示词03仅负责写入 MDX 文件内容，不涉及目录结构或 `_meta.{locale}.json` 的创建/更新
- **修改场景下**：若原文件已有 `title`，且用户未指定新 `Title`，保留原 `title` 不变

**ar 语言后缀**：`ar` 语言的 MDX 文件后缀为 `ar`（而非 `ar-AR`），如 `getting-started.ar.mdx`
